## 106 調節裝備幫你前刃彎「不累」且穩定

### 問題描述 (What)
後刃轉前刃時，人會有點拉起來的感覺，踮腳尖把腳跟拉起來，膝蓋比較直，腳踝打開。這樣會導致不穩定且很累，尤其是小腿和大腿。

### 改善目標 (Why)
採取正確的站姿，讓人整個沉下去，板子角度相對低一點來支撐，這樣站一整天也不會累，同時更加穩定。

### 改善方法 (How)
**裝備調節方法**：
1. **固定器調整**：
   - 把兩邊的固定器稍微鬆一點（鬆一到兩格）
   - 這樣綁帶不會頂得那麼厲害
   - 膝蓋彎的時候就可以沉下去更多
   - **注意**：不要鬆太多，鬆太多容易崴腳

2. **鞋子調整**（如果鬆固定器還找不到感覺）：
   - 把鞋子稍微繫鬆一點
   - 幫助找到膝蓋彎、腳踝閉合的感覺

3. **適用對象**：
   - 主要針對新手小白
   - 裝備很新的人
   - 喜歡買硬裝備的人

**正確站姿**：
- 人整個沉下去
- 膝蓋彎曲
- 腳踝閉合
- 不要過度（稍微收回來一點）

**錯誤站姿**：
- 踮腳尖
- 膝蓋直
- 腳踝打開
- 人拉起來

**效果**：
- 每次轉完彎後站姿非常棒
- 非常穩定
- 不會累

**原理**：通過調節裝備的方式來幫助找到前刃的感覺，當裝備不會過度支撐時，更容易找到正確的下沉姿態。

## 107 試試這個，幫你感受你有腳踝

### 問題描述 (What)
膝蓋與腳踝不夠緊致，換刃時板子無法穩定地吸在身體下方。

### 改善目標 (Why)
讓膝蓋與腳踝緊致，板子能吸在身體下方，不管跳多高都永遠立在板子上面，對起跳跳台時感受腳踝的緊致有幫助。

### 改善方法 (How)
**第一步：平地練習**
1. 找一個平地
2. 膝蓋腳踝緊致的情況下，直上直下跳
3. **正確感覺**：板子會吸在身體下方（腳沒綁也能做到）
4. **錯誤感覺**：如果不讓膝蓋腳踝緊，板子會很鬆

**第二步：蹬起來往前走**
1. 下了纜車後
2. 在蹬起來往前走的過程中嘗試跳一下
3. 這個比較簡單

**第三步：坡度上練習**
1. 在坡度往下走的時候也可以嘗試
2. 這個對從坡上面往下起跳跳台、感受腳踝緊致有幫助

**核心感受**：
- 膝蓋與腳踝緊的時候，板子可以吸在身體下方
- 能深刻理解這個緊致的感覺

**漸進順序**：
平地跳 → 平地往前走跳 → 坡度上跳

## 108 試試讓你的膝蓋這麼「轉」

### 問題描述 (What)
很多人以為轉彎時需要「轉動膝蓋」，直接去轉動膝蓋會傷害膝蓋且不穩定。

### 改善目標 (Why)
理解膝蓋不是主動轉動，而是隨著髖轉動，這樣不會傷害膝蓋，大腿肌肉會發力，更加穩定。

### 改善方法 (How)
**錯誤概念**：
- 主動去轉動膝蓋（像擰膝蓋一樣）
- 這會傷害膝蓋

**正確概念**：
1. **膝蓋永遠是隨著髖轉彎**
2. **膝蓋只做彎曲動作**：
   - 隨著固定器的角度彎下去
   - 朝著固定器的方向下降
3. **轉動來自髖**：
   - 當髖轉動時，膝蓋會跟著動
   - 但膝蓋本身不主動轉

**正確做法**：
- 把膝蓋關節保持在固定器方向
- 然後去轉髖
- 膝蓋還是朝著固定器的方向下降
- 但從外觀看膝蓋會跟著髖一起動

**效果**：
- 不會傷害膝蓋
- 大腿肌肉會發力
- 更加穩定

**CASI 術語**：這就是 CASI 裡面叫做的「牛仔站姿」

**核心原則**：膝蓋跟隨髖轉動，而非主動轉動膝蓋

## 109 嘗試讓你的髖保持打開，小回轉就小了

### 問題描述 (What)
髖關節與板子的角度不對，導致刃與刃之間的轉換不夠快速，無法轉小彎。

### 改善目標 (Why)
讓髖關節保持打開一點點的角度，使前刃與後刃之間的轉換更加快速，給壓力或軸轉都會更快，從而能轉更小的彎。

### 改善方法 (How)
**角度說明**（用杆子示範）：
1. **板子角度**：正朝著攝像機方向
2. **後刃時的髖角度**：
   - 比較開放
   - 杆子放在髖上，杆子跟板子有一定角度
3. **進階/小彎時的髖角度**：
   - 髖關節應該打開一點點
   - 杆子與板子的角度更明顯

**保持打開**：
- 從後刃換到前刃
- 換回後刃
- 全程保持這個打開的角度

**好處**：
1. 刃與刃之間的轉換更快速
2. 給壓力更快
3. 軸轉更快
4. 能轉更小的彎

**視覺化練習**：
- 可以拿著杆子去滑
- 這樣能清楚看到髖關節是否一直保持打開狀態

**適用場景**：
- 進階滑手
- 想轉小一點的搓雪小彎

**核心**：髖關節打開角度 = 快速換刃 = 小彎

## 110 轉彎時候的「轉」先轉還是後轉？

### 問題描述 (What)
轉彎順序錯誤：從彎上面到滾落線先轉髖轉身體，速度變快後才下蹲用壓力控制速度。這樣會導致抓地力不夠，速度控制忽快忽慢，坡多陡速度就有多快，無法自主控制。

### 改善目標 (Why)
建立正確順序，讓轉的過程有抓地力能抓住雪，不管坡陡與緩都能控制好速度全程下來，在陡坡上有更好的控速能力。

### 改善方法 (How)
**錯誤順序**：
1. 換重心、換刃
2. 先轉髖、轉身體（從彎上面到滾落線）
3. 速度變快後才下蹲
4. 用壓力控制速度

**結果**：
- 上半弧失速
- 抓地力不夠
- 速度忽快忽慢
- 坡多陡速度就有多快（被動）

**正確順序**：
1. **換重心**
2. **換刃**（不需要傾倒，因為坡度已經幫你立刃了）
3. **首先下降**
4. **控制好速度之後再轉髖**
5. **轉下半身關節帶著下來**
6. **持續帶著下來**

**關鍵原因**：
- 當在滾落線前下降之後再轉，轉的過程是有抓地力的
- 能抓住雪
- 不是被坡度控制速度，而是自己主動控制

**漸進練習**：
1. 先在綠道上做
2. 綠道做得好再去藍道
3. 藍道做得好再去黑道

**核心概念**：
- 抓地力是控速的很好元素之一
- 把抓地力建立起來就會越來越棒

**順序口訣**：換重心 → 換刃 → 下降 → 轉髖 → 持續帶下來
