# 47 開放站姿｜第一集：如何保持開放

## 📝 教學內容

### 問題描述 (What)
不知道如何保持開放站姿，不了解開放站姿的好處和應用情境。

### 改善目標 (Why)
- 在很爛的雪裡面更加的穩定
- 同時也更加的靈活
- 可以隨時換重心
- 橫貫雪道的時候比較安全而且比較快速
- 不會從隊伍裡面落下

### 改善方法 (How)

**開放站姿腳壓位置技巧**：

1. **後刃時的腳壓位置**：
   - 前腳：壓在了腳跟的內側
   - 後腳：壓在了整個腳掌上面

2. **前刃時的腳壓位置**：
   - 後腳：壓在了腳尖位置
   - 前腳：壓在了腳掌的位置

**為什麼需要保持開放站姿**：

1. **在道外的好處**：
   - 尤其是在道外，有時候會感覺雪它深一腳淺一腳
   - 或者它是屬於那種像那種淺的小鐵塊在上面，咯啦咯啦咯啦，就會讓不平整
   - 在這種情況下有這種方法，隨時可以保持一隻腳是踩在平板上
   - 需要換重心怎麼可能換得滿呢

2. **重心居中**：
   - 當留著一隻腳在板子正上方的時候，重心其實會被拉到了板子的內部、板子邊緣的內側
   - 所以說整個人的重心是偏居中的

**應用場合**：

**橫貫雪道時**：
- 當橫貫一些雪道的時候，或者橫切雪道想要去想去的地方的時候
- 比如說左腳在前，當往這邊橫切的時候，會讓前腿踩在後刃上面，放鬆一點踩在平板上面
- 如果要是前刃往那邊切，前腿會很放鬆，會踩在平板，但後腿會踩在腳尖上面
- 這就是可以幫助保持一個開放的站姿

# 48 開放站姿｜第二集：換重心可以排在二三位

## 📝 教學內容

### 問題描述 (What)
不了解開放站姿和閉合站姿在換重心順序上的差異，不知道開放站姿如何提高換刃效率。

### 改善目標 (Why)
- 提高效率
- 大大增加換刃之間的效率
- 在刻滑或者說在地形比較複雜、在包裡面的時候，有時候可能重心不過來但已經把下一個彎都轉的差不多了
- 軸轉會大幅度的減小

### 改善方法 (How)

**開放站姿換重心順序技巧**：

1. **閉合站姿的順序**：
   - 從後刃換前刃、前刃換後刃
   - 如果是閉合站姿的情況下，跟板齊平
   - 換重心的話需要：先換重心 → 然後再立刃（通過傾倒或者彎曲關節）→ 然後再施壓 → 然後最後再轉動
   - 這是它的一個順序

2. **開放站姿的順序**：
   - 當保持開放的時候，比如說要從後刃換到前刃的時候
   - 身體可以：先出現立刃 → 甚至有可能可以先出現施壓 → 然後重心再慢慢地跟著過來

**效率的提高**：
- 當如果要是通過先換重心再立刃再施壓，彎相對來說花的時間去換刃需要時間比較長一點
- 當從刃到刃之間的轉換只需要跳過重心的轉換，只需要靠傾倒或者施壓的話，重心隨之慢慢過來的話，這個大大增加了換刃之間的效率
- 也就是為什麼可以在刻滑或者說在地形比較複雜、在包裡面的時候，有時候會發現有時候可能重心不過來，但已經把下一個彎都轉的差不多了，甚至已經轉到滾落線了
- 就是因為保持了一個開放站姿，大大提高了換刃的效率

**動作模型的改變**：
- 從之前的：先換重心 → 再立刃 → 再施壓 → 最後軸轉
- 變成：立刃 → 然後施壓與重心是在過程中去進行一個轉換的 → 最後出現了軸轉
- 而且當去保持開放站姿的時候，軸轉會大幅度的減小

**練習方法**：
- 當在後刃換前刃的時候，讓重心不要去過去
- 讓重心保持在後刃上
- 先通過傾倒把刃換過去之後
- 重心通過施壓慢慢的再去壓過去
- 這個方法可以讓彎與彎之間的轉換變得更加有效率更加的快
- 同時會感覺到施壓也會更加的多一些
- 在施壓的過程中重心也就換過來了
