# 152. 單板第一次後刃換前刃練這個

## 問題描述 (What)

我在新手階段後刃滑得很熟練，但一換到前刃就不敢換或者會摔跤。我不知道如何安全地從後刃過渡到前刃，缺乏一個循序漸進的練習方法。這個換刃的恐懼和不穩定讓我無法完成完整的轉彎，限制了我的滑行進步。

## 改善目標 (Why)

我需要通過兩個步驟來攻克後刃換前刃的難題：第一步是練習前刃的J字型轉彎，讓我熟悉前刃的感覺和控制；第二步是在後刃直線滑行後，運用J字型轉彎的動作完成換刃。這樣我就能建立信心，掌握穩定的換刃技巧，不再害怕或摔跤。

## 改善方法 (How)

### 步驟拆解

1. **學習前刃J字型轉彎**：
   - 以身體中間為軸稍微降低重心
   - 轉動髖部，帶動膝蓋與腳踝（兩隻腳一起）
   - 讓板子趨近滾落線（往下衝）
   - 衝一段後再轉回來控制速度
   - 動作幅度很小，不需要大動作

2. **練習前刃J字型轉彎**：反覆練習直到動作流暢穩定

3. **後刃直線滑行**：讓後刃達到直線往下衝的狀態

4. **完成換刃**：
   - 在後刃直線滑行時
   - 讓髖部轉向滾落線
   - 到滾落線後做前刃的動作（平板放松）
   - 用前刃J字型轉彎的動作把板子拉回來
   - 完成後刃換前刃

### 教練提示

- J字型轉彎的動作幅度非常小，不要用力過猛
- 以身體中間為軸，兩隻腳同時轉動
- 換刃時要經過平板放鬆的階段
- 前刃J字型轉彎熟練後再嘗試換刃

### 常見錯誤

- 前刃J字型轉彎動作幅度太大
- 換刃時沒有經過平板放鬆
- 還沒練熟前刃J字型就急著換刃
- 換刃時身體僵硬，沒有用髖部帶動

### 修正方法

- 先在緩坡反覆練習前刃J字型轉彎，確保動作流暢
- 換刃時刻意放鬆，讓板子經過平板狀態
- 用髖部轉動帶動整個動作，不要只用腳
- 循序漸進，不要急於求成
