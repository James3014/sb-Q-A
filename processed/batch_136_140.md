## 136 陡坡換刃停止下「墜」

### 問題描述 (What)
在藍道或黑道比較陡的道上，一轉彎過滾落線之後板子會往下掉，不會像綠道一樣繼續轉，而是直接往下推然後再停靠推坡去停，尤其在陡坡上時常發生。

### 改善目標 (Why)
讓轉彎不會往下掉，會帶動雪板去轉彎而不是往下滑，彎形在任何情況下都是受到控制的，由細慢慢變粗再慢慢變細。

### 改善方法 (How)
**改錐比喻**：
- 把自己想像成一個改錐（螺絲刀）
- 用改錐去擰螺絲緊螺絲的時候
- 不會輕輕地去擰
- 會先給它按下去，然後再使勁擰

**陡度比喻**：
- 陡度就像一個很難搞的螺絲
- 先要在轉之前按下去
- 先給它上上勁兒
- 然後再擰擰擰

**具體做法**：
1. **換完刃之後**
2. **轉之前先壓下去**
3. **先給它上上勁兒**
4. **然後再轉**
5. **增加抓地力**
6. **就不會往下掉了**

**效果**：
- 轉彎不會往下掉
- 會 steer the board（帶動雪板）去轉彎
- 而不是往下 sliding（滑）往下掉
- 彎形由細慢慢變粗然後再慢慢變細
- 彎形在任何情況下都是受到控制的

**注意事項**：
- 這種滑法稍微累了點
- 但不會讓彎形像一個細線然後啪發散開
- 做熟練之後可以稍稍起來一點點
- 去減小這個範圍
- 就會變得更加輕鬆

**核心**：先壓下去上勁兒 → 再轉 = 增加抓地力不往下掉

## 137 陡坡冰蘑菇小技巧

### 問題描述 (What)
在比較冰的包裡面，不可能像雪好的時候有那麼強的抓地力，不知道如何應對。

### 改善目標 (Why)
在冰包裡面通過策略確保人不要摔倒，雖然無法得到相同的抓地力但可以安全通過。

### 改善方法 (How)
**包子比喻**：
- 像吃包子，包子剛蒸熟從鍋裡拿出來
- 用手去抓不可能一直抓
- 肯定是這樣拿來拿去的（快速碰一下）

**策略應用**：
- **包子 = 雪**
- **手 = 腳**
- 需要去碰一下雪
- 但不能完全碰
- 碰一下雪
- 立刻把重心彈到另外一側
- 再碰一下再彈到另外一側
- 確保人不要摔倒就好

**核心認知**：
- 不可能在雪差的情況下得到相同的抓地力在包裡面
- 所以利用這個策略去嘗試

**動作要點**：
- 快速碰雪
- 立刻彈開
- 彈到另一側
- 重複動作
- 保持不摔倒

**核心**：快速碰雪 → 立刻彈開 → 彈到另一側 = 冰包安全通過

## 138 陡坡前刃進階：彎末小練習

### 問題描述 (What)
在很陡的坡換刃時，人會情不自禁想去轉彎因為覺得陡坡轉彎可以多減速，導致人會大幅度傾倒朝離心方向，尤其後換前會讓人整個倒進去。

### 改善目標 (Why)
保持人在彎的下半段不要倒進去，讓人是直立的，連接後刃時人就不會倒或者失控，可以幫助考體系二級甚至三級。

### 改善方法 (How)
**問題分析**：
- 在很陡的地方（例如純黑道）
- 人會情不自禁想去轉彎
- 覺得陡坡轉彎可以多減速
- 導致人大幅度傾倒朝離心方向
- 尤其後換前基本上會讓人整個倒進去

**小練習**：
1. **保持人在彎的下半段不要倒進去**
2. **彎的下半段是滾落線之後的位置**
3. **在滾落線做練習**：
   - 讓兩個手往身體後面拉
   - 把手頂出去
4. **效果**：能保持人是直立的

**進階要點**：
- 先試試這個練習
- 至少讓人是直立的
- 這樣當連接後刃的時候
- 人就不會倒或者失控了

**為什麼不只是頂髖**：
- 需要讓背肌夾緊
- 這樣背是直的
- 同時換到後刃的時候
- 手還可以保持在那

**動作組合**：
- 不只是髖頂
- 手也往後拉
- 背肌夾緊
- 背是直的

**核心**：手往後拉 + 髖頂出去 + 背肌夾緊 = 人直立不倒進去

## 139 陡坡基礎第二集：出彎技巧

### 問題描述 (What)
在樹林裡面轉或陡坡粉比較深的時候，刃容易卡在雪裡陷進去，卡住就會趴地上很累，不知道如何讓板底多給支撐力。

### 改善目標 (Why)
盡量多讓板底給支撐力，盡量少讓刃卡在雪裡，通過板尾的彈力往前彈幫助前腳壓住雪進入下一個彎，順利進行換刃。

### 改善方法 (How)
**目的**：
- 盡量多讓板底給支撐力
- 盡量少讓刃卡在雪裡
- 雪比較軟很容易卡住
- 卡住就趴地上很累

**出彎技巧**：
1. **彎轉下來到彎的下半段時**
2. **讓後腳去做一個 Ollie 這樣的感覺**
3. **當做完 Ollie 的感覺**
4. **重心不是會往板尾移動**
5. **因為當 Ollie 壓在板尾時**
6. **人會被板尾的彈力往前彈**
7. **往板頭方向**
8. **幫助前腳去壓住雪進入下一個彎**

**靜態練習**：
1. **先在平地感受板尾可以把人彈回中間的感覺**
2. **利用板尾的力量**
3. **要確定彈回前刃或者後刃**
4. **而不是完全在平板**
5. **這個很重要**
6. **在粉雪或地形複雜時需要利用這些換刃**
7. **彈到前刃、彈到後刃**

**動態練習**：
- 在平地熟悉動作後
- 帶到平地的雪道上
- 板尾、後刃、板尾、前刃、板尾、後刃

**適用場景**：
- 樹林裡面比較寬敞一點的
- 雪比較厚一點的
- 可以嘗試用板底多去做 Ollie
- 幫助順利進行換刃

**核心**：後腳 Ollie → 板尾彈力往前彈 → 前腳壓雪進下一彎

## 140 陡坡小彎：你的雙腳還同時發力？

### 問題描述 (What)
小彎走到小回轉時很喜歡兩個腳一起去發力，導致彎形只出現在滾落線之後，上半弧完全缺失。

### 改善目標 (Why)
讓上半弧慢慢出來，不會只在下半弧減速，彎形更加圓潤，不會像推坡一樣下蹲起來下蹲起來。

### 改善方法 (How)
**問題分析**：
- 滑挺穩的，重心換得挺好
- 但兩個腳一起發力
- 彎形只出現在滾落線之後
- 上半弧完全缺失

**現狀**：
- 滾落線之後這一部分做得相當棒
- 速度控制相當好

**改善方向**：
1. **在滾落線之前的位置**
2. **先感覺到前腳壓住了**
3. **感覺到已經跟雪對抗了**
4. **產生對抗了**
5. **後腳再跟上**
6. **先感覺到前腳，再感覺到後腳**
7. **先一下，再感覺到第二下**

**類比**：
- 有可能跟大家所認知的平板或者 pedaling（踩踏板）有點像
- 總是讓前腳先支撐住
- 後腳再跟上

**效果**：
- 上半弧就會慢慢出來了
- 不會只在下半弧減速了
- 不會感覺像推坡一樣下蹲起來下蹲起來
- 讓兩個腳分開
- 慢慢變得更加圓潤
- 尤其上半弧都出來了

**核心**：前腳先壓住對抗 → 後腳再跟上 = 上半弧出來彎形圓潤
