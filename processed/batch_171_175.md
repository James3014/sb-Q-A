# 171. 滑蘑菇小Tip：包裡"下腳"的位置

## 問題描述 (What)

我在滑蘑菇時經常忽略轉彎位置的選擇，可能會在包的下半部分轉彎，那裡的側壁比較冰，導致我直接滑下去無法控制速度。我不知道應該在包的哪個位置轉彎才能獲得更好的雪況和控制。

## 改善目標 (Why)

我需要學會選擇正確的轉彎位置。不要在所在包的下半部分轉彎（那裡側壁較冰），而是要轉到左右兩邊包的上半部分位置（那裡雪軟）。在雪軟的位置轉彎，我的速度能降下來，也不會有刮冰的聲音，轉彎會更穩定可控。

## 改善方法 (How)

### 步驟拆解

1. **識別包的位置**：
   - 認清自己所在包的位置
   - 識別左右兩邊包的位置
   - 區分包的上半部分和下半部分

2. **避開錯誤位置**：不要在所在包的下半部分轉彎（側壁較冰）

3. **選擇正確位置**：
   - 在所在包上平板滑行
   - 轉向左右兩邊包的上半部分位置
   - 在那裡的雪軟位置轉彎

4. **感受差異**：
   - 在冰面轉彎會有刮冰聲音，速度難控制
   - 在雪軟位置轉彎聲音小，速度能降下來

5. **重複練習**：每次轉彎都選擇左右兩邊包的上半部分

### 教練提示

- 轉彎位置的選擇比技巧更重要
- 包的下半部分側壁通常較冰，要避開
- 左右兩邊包的上半部分雪較軟，是理想轉彎位置
- 可以通過聲音判斷雪況：聲音大表示冰面，聲音小表示雪軟

### 常見錯誤

- 在所在包的下半部分轉彎，遇到冰面
- 沒有提前觀察和選擇轉彎位置
- 不知道包的不同位置雪況差異
- 轉彎位置選擇隨機，沒有策略

### 修正方法

- 滑行前先觀察包的分布和位置
- 刻意練習在左右兩邊包的上半部分轉彎
- 注意聽轉彎時的聲音，判斷雪況
- 每次轉彎前都提前規劃好位置

# 172. 滑蘑菇的一個底層邏輯

## 問題描述 (What)

我在滑蘑菇時不理解底層邏輯，可能會站得比較直，導致板子和包的側壁（側臂）形成很大的角度。這個角度就像在平道上刻滑一樣，刃會入得很深，板子會卡在雪裡轉不動，無法完成轉彎和減速。

## 改善目標 (Why)

我需要理解滑蘑菇的底層邏輯：當上包的側壁時，要通過壓縮身體來降低板刃角度，而不是把刃立起來。板刃角度小了，板子就可以轉動來減速。如果板刃角度太大（身體站太直），板子會卡在雪裡轉不了。這個原理適用於所有大包小包的側壁。

## 改善方法 (How)

### 步驟拆解

1. **理解側壁結構**：
   - 每個包都有側面（側壁/側臂）
   - 側壁是斜著的角度
   - 不同包的側壁斜度可能不同

2. **理解角度問題**：
   - 如果身體站得比較直，板子和側壁會形成很大角度
   - 這個角度相當於在平道上刻滑，刃會入很深
   - 板子會卡在雪裡轉不動

3. **壓縮身體降低角度**：
   - 上側壁時壓縮身體（下蹲）
   - 把板刃角度變小
   - 板子就可以轉動了

4. **轉彎減速**：
   - 板刃角度小的狀態下轉動板子
   - 實現減速和控制
   - 避免板子卡在雪裡

5. **在綠道練習**：先在綠道練習壓縮身體降低板刃角度的感覺

### 教練提示

- 滑蘑菇的底層邏輯是控制板刃角度
- 側壁是斜的，如果站太直板子會卡住
- 壓縮身體是為了降低板刃角度，讓板子能轉
- 這個原理適用於所有大小包

### 常見錯誤

- 身體站太直，板刃角度太大
- 把刃立起來，導致板子卡在雪裡
- 不理解側壁的斜度特性
- 沒有通過壓縮身體來調整板刃角度

### 修正方法

- 上側壁時刻意壓縮身體
- 感受板刃角度的變化
- 先在綠道練習，再到蘑菇道應用
- 如果板子轉不動，檢查是否身體太直

# 173. 滑蘑菇第一集：走側壁最簡單

## 問題描述 (What)

我第一次接觸道外或有包地形時，不知道該怎麼選擇路線，可能會隨意選擇轉彎位置，導致身體起伏太大、不夠穩定。我不清楚包的哪個位置最簡單，也不知道如何循序漸進地進入蘑菇道。

## 改善目標 (Why)

我需要理解路線選擇比技巧更重要。包的側壁是最簡單的位置，不要選擇包頂或包底（會讓身體起伏太大）。進入蘑菇田的方法是：先通過橫切蘑菇練習上下吸收包的起伏，熟練後再選擇包的側壁轉彎。側壁就像國內旗門旁邊的位置，在那裡往上滑一下轉彎。通過熟練度逐漸縮短橫切距離，最終達到每次都在包的側壁上轉彎。

## 改善方法 (How)

### 步驟拆解

1. **認識包的結構**：
   - 包頂：包的最高點
   - 包底：包與包的中間
   - 側壁：圍著包的一圈（最簡單的位置）

2. **階段一：橫切蘑菇練習上下**：
   - 橫切過包時，通過膝蓋和腳踝上下吸收包的起伏
   - 上去、下來、上去、下來
   - 感受身體的上下節奏
   - 不要急著轉彎

3. **階段二：選擇側壁轉彎**：
   - 橫切過幾個包後，下定決心找一個包的側壁
   - 在側壁位置轉彎（像在旗門旁邊往上滑轉彎）
   - 轉彎後繼續橫切
   - 再找下一個包的側壁轉彎

4. **階段三：縮短橫切距離**：
   - 通過熟練度逐漸縮短橫切雪道的距離
   - 最終達到每次都在包的側壁上轉彎
   - 不再需要長距離橫切

5. **選擇較緩的包**：找比較緩一點的包來練習，降低難度

### 教練提示

- 路線選擇比技巧更重要
- 側壁是最簡單的位置，避開包頂和包底
- 不要急於求成，先練好橫切上下再轉彎
- 側壁轉彎就像在旗門旁邊轉彎
- 循序漸進縮短橫切距離

### 常見錯誤

- 一開始就想在包頂或包底轉彎
- 沒有先練習橫切上下就急著轉彎
- 不知道側壁是最簡單的位置
- 橫切距離沒有循序漸進地縮短
- 選擇太陡的包來練習

### 修正方法

- 先在小包區域練習橫切上下
- 熟練後再選擇側壁轉彎
- 刻意選擇較緩的包
- 每次練習都有意識地縮短橫切距離
- 把側壁想像成旗門旁邊的位置

# 174. 滑雪"下蹲"這個量如何把控

## 問題描述 (What)

我在滑雪時不知道下蹲應該蹲多少才夠，有時候朋友說我蹲得不夠，但我不清楚這個"量"該如何把控。我可能蹲得太少導致無法控制速度，或者蹲得太多（超過90度）變成無效蹲，腿已經不再朝雪發力。我也不理解在包裡超過90度和主動蹲超過90度的區別。

## 改善目標 (Why)

我需要學會把控下蹲的量。正確的下蹲量是：大腿小腿不要超過90度（六七十度左右），在這個角度下蹲30秒到1分鐘會感覺大腿酸，這就是正確的量。超過90度就是無效蹲，因為已經靠關節支撐而不是肌肉發力。在包裡被雪推導致超過90度是正常的，但主動蹲不應該超過90度。我要通過靜態下蹲找到腿酸的感覺，然後在雪道上轉彎時複製這個感覺，如果能感覺到相同的酸，說明蹲的量就夠了。

## 改善方法 (How)

### 步驟拆解

1. **理解正確的蹲量**：
   - 大腿小腿角度約六七十度
   - 不要超過90度
   - 超過90度是無效蹲（靠關節不靠肌肉）

2. **靜態練習找感覺**：
   - 腳放在固定器裡
   - 下蹲到六七十度
   - Hold住30秒到1分鐘
   - 感覺大腿酸，可能需要腳踝往上勾讓小腿幫助支撐
   - 記住這種酸的感覺

3. **雪道上應用**：
   - 記住靜態蹲的酸感後立即去滑
   - 在陡一點的道或雪比較爛的道轉彎
   - 如果能感覺到腿有相同的酸，說明蹲的量夠了
   - 如果完全感覺不到酸且失去速度控制，需要再多蹲一點

4. **調整蹲量**：
   - 腳踝勾起來一點
   - 膝蓋彎曲一點
   - 稍微找到一點酸的感覺
   - 逐漸接近目標

5. **理解包裡的特殊情況**：
   - 在包裡被雪推可能會超過90度
   - 這是雪推造成的，不是主動蹲造成的
   - 主動蹲不應該超過90度

### 教練提示

- 正確的蹲量是六七十度，不要超過90度
- 通過靜態蹲找到腿酸的感覺，然後在雪道上複製
- 超過90度是無效蹲，靠關節不靠肌肉
- 包裡被雪推超過90度是正常的，但主動蹲不應該
- 如果感覺不到酸且失控，說明蹲得不夠

### 常見錯誤

- 蹲得太少，無法控制速度
- 蹲超過90度變成無效蹲
- 不知道如何判斷蹲的量是否正確
- 沒有通過靜態練習找到參考感覺
- 混淆主動蹲和被雪推的區別

### 修正方法

- 先做靜態蹲30秒到1分鐘，記住腿酸的感覺
- 立即去雪道上滑，嘗試複製這個酸感
- 如果感覺不到酸，逐步增加蹲的量
- 用腳踝和膝蓋的彎曲度來調整
- 在陡坡或爛雪上練習，更容易感受到

# 175. 進階版牛仔站姿：幫助你更居中穩定

## 問題描述 (What)

我知道基礎的牛仔站姿（膝蓋往外朝固定器像扎馬步）可以幫助膝蓋彎曲更往板子中央，但我不清楚牛仔站姿在進階階段還有什麼更深層的作用。我可能無法靈活調節髖部在板子上的位置（偏後刃、中間或偏前刃），也不知道如何在上半弧時利用牛仔站姿來保持居中和穩定。

## 改善目標 (Why)

我需要理解牛仔站姿的進階用法。當我通過牛仔站姿往下壓時，腳踝可以瞬間勾起來，讓我的重心往下沉。這個下沉有三個方向：屁股出板（偏後刃，不穩）、往板子中間沉（相對穩定）、往前刃沉（在上半弧時使用）。牛仔站姿讓腳踝變得非常靈活，可以在上半弧時調節髖部的位置，讓我完全居中，可以隨意起伏給板子施加壓力。這個技巧在包裡、道內或陡坡都很有用，能把全身最重的位置（髖部）放在板子正上方，讓我更平衡和穩定。

## 改善方法 (How)

### 步驟拆解

1. **建立牛仔站姿**：
   - 像扎馬步一樣
   - 兩個膝蓋往外一點點朝向固定器
   - 這是基礎姿勢

2. **理解三個下沉方向**：
   - 屁股出板：偏後刃，不穩定
   - 往板子中間沉：相對穩定
   - 往前刃沉：在上半弧時使用，完全居中

3. **練習腳踝勾起**：
   - 通過牛仔站姿往下壓
   - 兩個腳踝往上一勾
   - 重心瞬間往下沉

4. **在上半弧應用**：
   - 在上半弧時使用牛仔站姿
   - 通過腳踝勾起往前刃沉
   - 雖然在後刃但看起來完全居中
   - 可以隨意起伏給板子施加壓力

5. **調節髖部位置**：
   - 通過牛仔站姿和腳踝靈活度
   - 調節髖部在板子上的位置
   - 可以偏後刃、中間或偏前刃
   - 把全身最重的位置放在板子正上方

6. **應用場景**：
   - 包裡面
   - 道內
   - 陡一點的道
   - 任何需要更居中穩定的地形

### 教練提示

- 牛仔站姿的進階用法是調節髖部位置
- 腳踝勾起是關鍵，讓腳踝變得非常靈活
- 在上半弧使用可以完全居中，隨意起伏
- 把髖部（全身最重位置）放在板子正上方
- 這個技巧在進階和二級教練以上都很有用

### 常見錯誤

- 只知道牛仔站姿的基礎用法（膝蓋彎曲往中央）
- 不會利用腳踝勾起來調節重心
- 不理解三個下沉方向的差異
- 沒有在上半弧時使用牛仔站姿
- 髖部位置固定，無法靈活調節

### 修正方法

- 先練習腳踝勾起的動作，感受重心下沉
- 刻意練習三個下沉方向，體會穩定性差異
- 在上半弧時使用牛仔站姿，感受完全居中的感覺
