# 49 開放站姿｜第三集：過包抽拉板子

## 📝 教學內容

### 問題描述 (What)
在滑包的時候，滑比較大一點的蘑菇的時候，過包頂之後板頭會被揚起來，一旦過了板子中央的時候，人會被往後置，然後板子會送出去。

### 改善目標 (Why)
- 把板子拉回來、抽拉回來
- 讓板子在身體下方保持一個更加穩定的滑行
- 尤其是在包裡面

### 改善方法 (How)

**開放站姿抽拉板子技巧**：

1. **動作原理**：
   - 當一旦過了中線之後會把板子往回拉
   - 這個動作參與了身體裡關節的什麼呀，也就是腳踝

2. **閉合站姿的問題**：
   - 如果完全跟板平行的話，這麼去往回拉的話，會很容易崴到腳踝
   - 因為往這邊側會崴到

3. **開放站姿的優勢**：
   - 當稍微打開一點的時候，會借用到這個站姿下的一個情況
   - 可以讓腳踝稍微這麼斜著一點往回這麼拉
   - 比如說現在把重心放後腿，可以斜著一點把這腳往回拉

4. **腳踝的作用**：
   - 當保持一個開放站姿的時候，身體開放一點的時候
   - 腳踝不只是單純的這種左右這麼移動
   - 而是參與了前後關閉腳踝打開腳踝的移動

5. **應用時機**：
   - 尤其是當上包的時候
   - 也就是快即將要換刃的時候
   - 在這個位置身體保持一個打開一點的狀態的話
   - 可以有效的幫助把板子抽拉回來

# 50 開板：如何不讓後刃卡住摔倒

## 📝 教學內容

### 問題描述 (What)
後刃的時候，滾落線過後，板子會非常的抖動，然後速度會失去。刃卡雪裡的原因是因為後刃站的比較高。

### 改善目標 (Why)
- 讓後刃不要卡在雪裡
- 降低立刃角度
- 從刻滑變成搓雪

### 改善方法 (How)

**膝蓋頂向前刃技巧**：

1. **核心動作**：
   - 把膝蓋在後刃頂向前刃
   - 立刃角度就會降低
   - 就不是刻滑大神了，就開始搓雪大神

2. **具體做法**：
   - 後刃當下彎的時候，兩個膝蓋只能這麼彎曲
   - 當它彎曲的時候，讓兩個膝蓋往前刃去頂
   - 這樣的話，立刃角度就會被壓迫的變小
   - 就不會用刃卡在雪裡了

**關鍵提示**：
- 前刃不會卡裡，是因為前刃天然的關節容易控制
- 但是後刃不行
- 所以說最基礎的一個最關鍵的一個點就是把膝蓋在後刃頂向前刃

# 51 引申：釋放板刃去換刃

## 📝 教學內容

### 問題描述 (What)
換刃的時候，板刃如果卡雪裡，不站起來直接這麼趴過去的話，這個幅度是控制不了的。

### 改善目標 (Why)
- 能夠順利換刃
- 控制換刃的幅度
- 在包裡面或者在雪道裡面都會有一個很好的結果

### 改善方法 (How)

**引申（Flexion & Extension）技巧**：

1. **什麼是引申**：
   - 引申顧名思義，會下降，然後會起來，它來組成引申
   - 就是 flexion 跟 extension

2. **為什麼要站起來**：
   - 比如說在後刃滑的時候，會下壓，板子它會去搓雪
   - 如果不站起來，當下壓的時候板刃，它是多多少少會有一點卡雪裡的
   - 不管是高立刃還是低立刃，它都會卡雪

3. **換刃順序**：
   - 要想換刃的話，需要站起來，再換刃
   - 站起來，再換刃
   - 這就是引申

4. **核心原理**：
   - 需要把板刃釋放出來
   - 然後再換到新刃上面
   - 這也就是引申的統一部分
