# 186. 淺聊團身

## 問題描述 (What)

我對團身滑法感興趣，但不清楚它是不是唯一的滑法、是不是我的終極目標、它的優缺點是什麼、以及在CASI體系中的定位。我可能認為要滑得越來越好就需要越來越團，或者不知道團身只是一個工具而非終點。

## 改善目標 (Why)

我需要從三個維度理解團身：

**1. 不是唯一滑法，也不是終極目標**：團身不是唯一滑法，也不是終極目標。想滑得越來越好，應該是盡量不去團身但也能達到團身的效果。

**2. 優點與缺點**：
- 優點：永遠貼著雪、換刃施壓立刃旋轉都最快、非常穩定、適應所有地形（冰面、窄道、陡坡、蘑菇）、對初學道外的人更安全
- 缺點：累（對身體素質要求高）、沒有持久性（團兩三圈就累了）、表現力不夠（接觸雪面時間長摩擦力大速度起不來）

**3. CASI體系定位**：CASI三級會教團身，但明確告訴你不是唯一滑法。學團身是因為三級考蘑菇，團身是最佳安全方法，且要求雪板不能離開雪面。三級往上不要使用團身去滑，團身只作為工具（速度過快時找回節奏、比賽時先試雪況）。

## 改善方法 (How)

### 步驟拆解

1. **理解團身的定義**：
   - 起點就是蹲著、團著
   - 永遠貼著雪
   - 可以最快進行換刃施壓立刃旋轉

2. **體驗團身的感覺**：
   - 靠牆靜蹲1分鐘
   - 感覺相當於團身滑10個彎的感覺
   - 如果不酸說明團得不夠低

3. **理解優點的應用場景**：
   - 冰面：更穩定
   - 窄道陡坡：需要跳著轉彎
   - 蘑菇：不被包送飛起來
   - 道外初學：更安全

4. **認識缺點的限制**：
   - 累，沒有持久性
   - 速度起不來
   - 表現力不如引身

5. **CASI體系中的正確使用**：
   - 三級學團身是為了考蘑菇
   - 團身是工具不是終點
   - 三級往上不要全程用團身
   - 用團身找回節奏或試雪況

6. **進階理解**：
   - 越滑得好，團身會慢慢離你而去
   - 目標是不團身也能達到團身效果

### 教練提示

- 團身不是唯一滑法，也不是終極目標
- 團身的優點很閃光，但缺點也明顯
- CASI十幾年前就在三級教團身，作為工具使用
- 不要全程用團身，會非常累
- 三級往上，團身會慢慢離你而去

### 常見錯誤

- 認為團身是唯一滑法
- 認為要滑得好就要越來越團
- 不理解團身的缺點
- 全程用團身導致太累
- 不知道團身只是工具

### 修正方法

- 理解團身是工具不是終點
- 在需要時使用團身（蘑菇、找回節奏、試雪況）
- 不要全程團身
- 目標是不團身也能達到效果
- 三級往上逐漸減少團身使用

# 187. 能在平地練習的"豚跳"

## 問題描述 (What)

我想學習臀跳（豚跳），但不知道如何練習。我可能不理解臀跳的本質是什麼，也不知道如何在平地就開始練習這個動作。

## 改善目標 (Why)

我需要理解臀跳就是Ollie的一種，本質是重心在前腳與後腳之間進行轉換。如果只是往後移，Ollie比較小；如果往前再往後，Ollie比較大。在雪道上的臀跳動作模式是一樣的。我可以用一個簡單的方法練習：像騎馬一樣，用板體進行重心前後轉移。轉移可以是髖往前往後，或者髖不動、腳往後腳往前送板。熟練後可以加入板底轉換（前刃後刃前刃後刃），在平地練習到比較安全後，再帶到綠道或藍道嘗試。

## 改善方法 (How)

### 步驟拆解

1. **理解臀跳本質**：
   - 臀跳是Ollie的一種
   - 本質是重心在前後腳之間轉換
   - 往後移：Ollie小
   - 往前再往後：Ollie大

2. **平地基礎練習**：
   - 像騎馬一樣的動作
   - 用板體進行重心前後轉移
   - 這是最基礎的臀跳

3. **兩種轉移方式**：
   - 方式一：髖往前往後移動
   - 方式二：髖不動，腳往後腳往前送板
   - 兩種都可以達成臀跳

4. **加入板底轉換**：
   - 熟練後加入板底轉換
   - 前刃、後刃、前刃、後刃
   - 逐漸變成換刃的臀跳

5. **平地練習到安全**：
   - 在平地反覆練習
   - 達到比較安全的程度
   - 確保動作流暢

6. **帶到雪道應用**：
   - 先在綠道嘗試
   - 再到藍道練習
   - 就比較簡單了

### 教練提示

- 臀跳本質是重心前後轉換
- 像騎馬一樣的感覺
- 可以用髖移動或腳送板
- 平地就可以開始練習
- 加入板底轉換後更接近實際

### 常見錯誤

- 不理解臀跳的本質
- 沒有在平地先練習
- 只用一種方式（髖或腳）
- 沒有加入板底轉換
- 直接在雪道上嘗試

### 修正方法

- 先在平地練習像騎馬的動作
- 嘗試兩種方式：髖移動和腳送板
- 熟練後加入板底轉換
- 確保平地安全後再上雪道
- 從綠道開始逐步進階

# 188. 帶轉角度："消失"的前刃

## 問題描述 (What)

我學會了後刃的Steering Angle（帶轉角度），但不知道前刃如何做出Steering Angle。我可能不理解前刃和後刃的Steering Angle有什麼差異，也不知道為什麼前刃的角度會比後刃小很多。

## 改善目標 (Why)

我需要理解前刃的Steering Angle：當在前刃時，肩已經朝向行進方向，但板子還在原位置。膝蓋會領先於板子，髖朝一邊，肩會更閉合一點。板子會跟隨身體這個角度。前刃的靈活度遠超後刃，一旦做多了板尾會立刻掃出去，所以前刃的Steering Angle會比後刃小很多。後刃可能是這麼多，前刃可能就這麼多（看起來像閉合式站姿）。在閉合情況下其實存在Steering Angle，只是比較小。一旦做太多，身體會失去平衡且板尾會甩出去。原因是前刃發力關節的靈活度比後刃快很多。這樣做的目的是在地形複雜時，讓上半身帶動下半身，讓身體駕馭板子，比板子帶著人走更安全穩定。

## 改善方法 (How)

### 步驟拆解

1. **理解前刃Steering Angle的形態**：
   - 肩已經朝向行進方向
   - 板子還在原位置
   - 膝蓋領先於板子
   - 髖朝一邊
   - 肩更閉合一點

2. **理解前後刃的差異**：
   - 前刃靈活度遠超後刃
   - 前刃做多了板尾會立刻掃出去
   - 前刃Steering Angle比後刃小很多

3. **控制角度大小**：
   - 後刃：角度可以比較大
   - 前刃：角度要小（看起來像閉合式站姿）
   - 前刃角度太大會失去平衡、板尾甩出去

4. **理解原因**：
   - 後刃發力關節 vs 前刃發力關節
   - 前刃靈活度快很多
   - 這是主要原因

5. **理解目的**：
   - 地形複雜時需要
   - 上半身帶動下半身
   - 板子跟著上半身行進
   - 身體駕馭板子比板子帶人更安全穩定

### 教練提示

- 前刃Steering Angle比後刃小很多
- 看起來像閉合式站姿，但其實有Steering Angle
- 前刃靈活度快，容易板尾掃出去
- 不要做太多，會失去平衡
- 目的是讓身體駕馭板子

### 常見錯誤

- 前刃Steering Angle做得跟後刃一樣大
- 不理解前後刃靈活度差異
- 角度太大導致板尾掃出去
- 失去平衡
- 不知道為什麼要做Steering Angle

### 修正方法

- 前刃角度要比後刃小
- 感受前刃的靈活度
- 控制角度，不要太大
- 保持身體平衡
- 在複雜地形應用

# 189. 帶轉角度第一集：後刃邏輯

## 問題描述 (What)

我已經可以在藍道甚至黑道熟練轉彎，尤其是與板齊平的情況下，但發現彎永遠不能轉得更小，或者效率比較低。我不知道什麼是Steering Angle（帶轉角度），也不知道如何應用它來提升滑行。

## 改善目標 (Why)

我需要理解Steering Angle：當我與板子保持一個人領先於板子的狀態，會出現一個夾角，這就是帶轉角度。具體做法是：板子朝一個方向，但我的髖、膝蓋、肩不會與板齊平，而是去打開或朝板頭打開。肩領先於髖，髖領先於膝蓋，膝蓋領先於腳（腳踝在鞋裡看不到）。

好處是：在轉的過程中，板子會永遠跟隨我人打開的角度去match、去找我人的角度。如果人一直能保持這個夾角不變，人會領先於板子。在地形複雜或想把彎轉小的情況下，都可以用到帶轉角度。

## 改善方法 (How)

### 步驟拆解

1. **理解Steering Angle定義**：
   - 人領先於板子
   - 出現一個夾角
   - 這就是帶轉角度

2. **理解身體排列**：
   - 板子朝一個方向
   - 髖、膝蓋、肩不與板齊平
   - 朝板頭打開
   - 肩領先於髖領先於膝蓋領先於腳

3. **理解好處**：
   - 板子會跟隨人打開的角度
   - 板子會去match人的角度
   - 人領先於板子

4. **理解應用場景**：
   - 地形複雜的情況
   - 想把彎轉小的情況
   - 都可以用到

5. **保持夾角不變**：
   - 人一直保持這個角度
   - 夾角不變
   - 這樣走

6. **觀看示範**：
   - 看示範理解動作
   - 感受人領先於板子的感覺

### 教練提示

- 恭喜你從小白走向高階專業
- Steering Angle是人領先於板子的夾角
- 肩領先於髖領先於膝蓋領先於腳
- 板子會跟隨人的角度
- 適用於複雜地形和小彎

### 常見錯誤

- 人與板齊平，沒有夾角
- 不知道如何打開身體
- 身體排列順序錯誤
- 夾角不穩定
- 不知道何時應用

### 修正方法

- 刻意讓人領先於板子
- 練習身體排列：肩→髖→膝蓋→腳
- 保持夾角穩定
- 在複雜地形和小彎中應用
- 觀看示範模仿

# 190. 解決前換後卡刃小練習

## 問題描述 (What)

我經常在前刃彎結束換後刃的時候卡後刃。當前刃彎結束時，由於身體朝山下打開比較多（看向山下），會形成一個稍微低的站姿（因為在壓雪），在這個站姿想換後刃就會卡住，人可能直接翻過去。

## 改善目標 (Why)

我需要學會解決方式：當彎結束時，讓上半身與板齊平，甚至讓前肩超過板子。這樣換刃時就不會卡了。當身子完全閉合甚至超過平行於板子時，卡刃的風險就少了，而且行進軌跡（左腳前朝左邊行進）會更順暢，然後平板換刃就不會卡刃了。

## 改善方法 (How)

### 步驟拆解

1. **理解卡刃原因**：
   - 前刃彎結束時身體朝山下打開比較多
   - 形成稍微低的站姿（壓雪）
   - 在這個站姿換後刃會卡住

2. **解決方式**：
   - 彎結束時讓上半身與板齊平
   - 甚至讓前肩超過板子
   - 這樣換刃時就不會卡

3. **理解效果**：
   - 身子完全閉合甚至超過平行於板子
   - 卡刃風險少了
   - 行進軌跡更順暢（左腳前朝左邊）

4. **平板換刃**：
   - 在閉合狀態下平板
   - 換刃就不會卡了

5. **場地選擇**：
   - 到稍微陡一點的地方示範
   - 更容易看出效果

### 教練提示

- 彎結束時上半身要與板齊平或超過
- 身子完全閉合是關鍵
- 行進軌跡要順暢
- 平板換刃不會卡
- 在陡坡上更明顯

### 常見錯誤

- 彎結束時身體還是打開的
- 上半身沒有與板齊平
- 沒有閉合就換刃
- 行進軌跡不順暢
- 直接換刃導致卡刃

### 修正方法

- 彎結束時刻意閉合上半身
- 讓前肩超過板子
- 確保身子完全閉合
- 平板後再換刃
- 在陡坡練習效果更好
