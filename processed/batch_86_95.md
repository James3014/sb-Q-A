## 86 改變你後刃「蹲坑」

### 問題描述 (What)
在陡坡後刃滑行時，出現「蹲坑拉屎」的姿態，下降時無法獲得更多抓地力和減速力量。

### 改善目標 (Why)
下降的同時獲得更多抓地力和減速力量，讓板子逐漸橫切雪道幫助減速，動作更美觀。

### 改善方法 (How)
1. **髖關節打開下降**：當下降時，讓屁股越出板子，髖關節蹲得越低時加入打開狀態
2. **效果**：
   - 不只是減速
   - 讓板子逐漸橫切雪道（橫切也幫助減速）
   - 速度控制更好
   - 動作更美觀
3. **位置關係**：板子 - 髖（打開）- 肩膀（三者關係）
4. **雙重減速**：下降減速 + 轉動打開帶動板子轉動減速
5. **適用場景**：陡坡或冰面

**關鍵**：髖關節打開下降 = 橫切雪道 + 轉動帶板 = 雙重減速

## 87 刻滑：先傾倒再折疊（發生的位置）

### 問題描述 (What)
刻滑時不清楚傾倒和折疊的時機與順序，不知道什麼時候用哪個，導致無法全程刻滑或需要搓雪減速。

### 改善目標 (Why)
實現全程刻滑的彎形，通過正確的傾倒和折疊順序保持速度不變但增加穩定性。

### 改善方法 (How)
1. **立刃兩種方式**：
   - 折疊：調節刃的角度
   - 傾倒：往前往後傾倒
2. **順序與時機**：
   - **先傾倒**：立刃最快的方式，入刃清晰
   - **再折疊**：進入滾落線後，通過折疊增加穩定性而非減速
3. **位置說明**：
   - 彎的上半段：傾倒入刃
   - 即將進入滾落線：開始折疊，越來越低保持穩定
   - 過了滾落線：開始站起來
   - 橫切雪道：幫助降速（而非搓雪減速）
4. **折疊差異**：
   - 後刃折疊：向下折疊
   - 前刃折疊：壓壓壓，壓到一定程度人開始往外走
5. **雪況調整**：雪軟時增加折疊減小立刃角度，避免刃卡進雪裡

**核心順序**：傾倒入刃 → 滾落線前折疊穩定 → 過滾落線站起 → 橫切降速

## 88 刻滑小練習：Rail to Rail

### 問題描述 (What)
開放式刻滑時，刃到刃的轉換不夠流暢，缺乏style。

### 改善目標 (Why)
增加蓄力傳導，幫助出刃，出來後感覺輕鬆，滑行中增加style。

### 改善方法 (How)
1. **Rail to Rail 練習**：
   - 前刃：刃卡在雪裡朝滾落線方向時，像做Ollie一樣
   - 後刃：刃卡在雪裡時，像做Ollie一樣
2. **加入髖扭轉**：
   - 前刃：加入髖的扭轉
   - 後刃：往另一邊扭轉
3. **效果**：
   - 出現蓄力傳導
   - 幫助出刃
   - 出來後很輕鬆
   - 增加style

**核心**：刃卡雪裡 + Ollie動作 + 髖扭轉 = 蓄力傳導

## 89 刻滑後刃：如何入彎不下墜或者掃出去

### 問題描述 (What)
刻滑後刃入彎時，彎拐得太往下太迅速，上半圓不是圓的而是瞬間拐下來，或者從開始就掃出去，彎的線寬度加寬，無法保持細線。

### 改善目標 (Why)
讓後刃跟前刃一樣保持一條細線，上半圓走出來。

### 改善方法 (How)
1. **第一點：先平板**
   - 前刃過渡到後刃時，先平板
   - 感受刃慢慢入到雪裡
   - 讓板子帶著走一點，把上半圓走出來
   - 不要直接從前刃立刻入後刃（會導致板頭朝下）
2. **第二點：前腿受力**
   - 過渡到平板後，感覺前腿受力（非常關鍵）
   - 如果只有後腿受力，板頭會飄
   - 板頭飄會導致彎無法保持細線
3. **耐心（Patient）**：
   - 從前刃非常耐心地換到平板
   - 感覺前腿受力
   - 刃扎到雪裡
   - 讓板子帶走一段
4. **效果**：前刃與後刃的彎對齊

**核心**：先平板 + 前腿受力 + 耐心 = 細線刻滑

## 90 單板開肩：小回轉

### 問題描述 (What)
不清楚開肩站姿應該開多少，肩膀擺在什麼位置比較友好。

### 改善目標 (Why)
找到適合不同水平的開肩角度，幫助板子轉動但不過度。

### 改善方法 (How)
**新手/小白**：
1. **肩膀位置**：
   - 後肩：保持在前刃裡側
   - 前肩：保持在後刃裡側
   - 在板子寬度範圍內，不要超出太多
2. **原因**：肩開一點點幫助帶動板子轉一點點，需要這個力量但不能太多

**進階/小彎**：
1. **後刃入彎**：
   - 肩已經打開
   - 打開基本在後刃上面或甚至超一點點
2. **前刃入彎**：
   - 更加齊平於板子
3. **前刃結束**：
   - 肩是打開一點的

**思考題**：為什麼前刃入彎時身體更齊平，後刃入彎時肩打開？（點贊過500揭曉）

## 91 單板開肩：底層邏輯

### 問題描述 (What)
不理解開肩的底層邏輯，不清楚開肩是站姿還是軸轉，以及為什麼需要開肩開髖的站姿。

### 改善目標 (Why)
理解開肩的底層邏輯，知道何時用站姿開肩、何時用軸轉，以及開肩開髖站姿的優勢。

### 改善方法 (How)
**底層邏輯一：肩隨髖開**
1. 肩永遠是隨著髖而開的
2. 滑雪時上半身稍微朝前一點，肩打開的同時髖也打開
3. 肩開會比髖稍稍多一點點
4. 髖在動時，肩相當於也跟著打開

**底層邏輯二：站姿 vs 軸轉**
1. **站姿開肩**：
   - 一開始就站成肩打開的樣子
   - 滑行過程中保持這個開肩，不會讓肩閉合與板平行
2. **軸轉**：
   - 一開始與板平行
   - 滑行過程中肩做轉動動作
   - 以肩為軸進行轉動

**為什麼不能通過肩帶動轉彎**：
1. 由上往下帶有延迟，行動很慢
2. 傳遞到板子上效率低
3. 適合場景：平花、跳台公園（需要增加旋轉力和圈數）

**為什麼需要開肩開髖站姿**：
1. **刃到刃轉換更快**：稍微往前往後傾斜，刃與刃轉換變快
2. **立刃更快**：立刃變快
3. **施壓更快**：可以利用上下起伏幫助給板施壓更快

**核心**：開肩開髖站姿 = 快速換刃立刃施壓

## 92 單板後刃「突突突」解決方法之一

### 問題描述 (What)
後刃在陡坡或冰面滑行時，後腿出現chattering（哒哒哒不停抖動）。

### 改善目標 (Why)
消除後腿抖動，保持穩定的推雪動作。

### 改善方法 (How)
1. **正確發力**：
   - 腳踝往上勾，膝蓋往下彎
   - 大腿與小腿肌肉發力
   - 確認方法：舌頭貼近小腿感覺很緊，大腿累酸
2. **推雪訣竅**：
   - 推的時候不要把腳踝與膝蓋完全蹬直
   - 永遠保留一定餘地
   - 不要完全蹬直（會出現chattering往下抖）
3. **練習方法**：
   - 在彎裡面推兩到三次
   - 如果推了一次推不了第二次 = 已經完全蹬直（錯誤）
   - 如果能推兩次三次甚至四五次 = 沒有完全蹬直（正確）
4. **重點**：尤其是後腿（重心腿）

**關鍵**：保留餘地不完全蹬直 = 消除抖動

## 93 單板如何滑「爛雪」

### 問題描述 (What)
在爛雪（slushy、heavy、重）中滑行，不知道如何調整技術。

### 改善目標 (Why)
在爛雪中穩定滑行，避免板子滑出去，通過壓力而非軸轉來轉彎。

### 改善方法 (How)
1. **CASI轉彎模式**：換重心 → 立刃 → 施壓 → 軸轉
2. **爛雪調整**：
   - 不加入軸轉（最後這個轉彎不做）
   - 原因：加入軸轉時板子會滑出去
3. **爛雪策略**：
   - 少立刃
   - 多利用壓力
   - 推雪時立刃角度小一點，雪會被推走
   - 推的時候人會往另一個方向去（這就是轉彎）
   - 通過壓力把自己彈到另一個方向
4. **具體做法**：
   - 換重心
   - 少立刃（但也在刃上）
   - 下降
   - 推一下
   - 去另一個地方
   - 幾乎不做軸轉
5. **例外**：雪道非常陡時，會加一點轉動髖（Pivot軸轉），也是在最後加

**核心**：換重心 + 少立刃 + 多壓力 + 不軸轉 = 爛雪控制

## 94 單板重心轉換第一課：前刃後刃的重心位置

### 問題描述 (What)
不清楚CASI所謂的重心是什麼，不知道前刃後刃的正確重心位置。

### 改善目標 (Why)
理解重心是轉彎的前提，掌握前刃後刃的正確重心位置，讓滑行更穩定居中。

### 改善方法 (How)
1. **完全穩定時的重心**：壓在足弓內側
2. **前刃重心位置**：
   - Ball of your foot（腳上的球）
   - 足弓向下延伸的位置
   - 腳踝略微勾起來一點點
   - 膝蓋稍微彎一點點
   - 確保感覺清晰
3. **後刃重心位置**：
   - 腳的肉墊內側，足弓的盡頭
   - 髖稍稍打開一點，讓肩膀在後刃正上方
   - 腳踝略微往上彎曲一點點
   - 膝蓋略微彎曲一點點
4. **重心轉換**：從後刃位置換到前刃位置（不說的話看不太出來）
5. **危險區域**：
   - 超過前刃位置或後刃位置就危險
   - 身體會大幅度傾倒
   - 導致彎腰撅屁股或後刃把板子推出去摔跤
   - 原因：重心遠離板子，板子無法給足夠支撐

**核心**：正確重心位置 = 穩定居中 = 板子有效支撐

## 95 單板減少刃與刃之間「卡頓」現象

### 問題描述 (What)
刃與刃之間出現卡頓現象，特別是在冰面蘑菇或兩個包之間，會變成side slipping推坡狀態卡住，無法順利銜接到下一個包。

### 改善目標 (Why)
減少刃與刃之間的卡頓，順利銜接到下一個包，甚至可以加速。

### 改善方法 (How)
1. **延續上次軸轉內容**：
   - 上次：幫助減速，把圓畫得更圓潤
   - 這次：更深層 - 增加creating energy，甚至加速，釋放板刃進入下一個彎
2. **主要體現**：
   - 小回轉
   - 蘑菇地形
   - 包地形
3. **運動模式**（參考上一個視頻）：
   - 後腳往下，前腳往上
   - 這次在最後時候做得稍微大一點
4. **效果**：
   - 拧板子讓板子彈起來
   - 通過彈起來順便帶著速度進入下一個彎
   - 特別適合包裡面
5. **應用場景**：冰面蘑菇、兩包之間的銜接

**核心**：後腳下前腳上（做大一點）= 拧板彈起 = 帶速度進下一彎
