# 158. 滑蘑菇第一集：走側壁最簡單

## 問題描述 (What)

我第一次接觸道外或有包地形時，不知道該怎麼選擇路線，可能會隨意選擇轉彎位置，導致身體起伏太大、不夠穩定。我不清楚包的哪個位置最簡單，也不知道如何循序漸進地進入蘑菇道。

## 改善目標 (Why)

我需要理解路線選擇比技巧更重要。包的側壁是最簡單的位置，不要選擇包頂或包底（會讓身體起伏太大）。進入蘑菇田的方法是：先通過橫切蘑菇練習上下吸收包的起伏，熟練後再選擇包的側壁轉彎。側壁就像國內旗門旁邊的位置，在那裡往上滑一下轉彎。通過熟練度逐漸縮短橫切距離，最終達到每次都在包的側壁上轉彎。

## 改善方法 (How)

### 步驟拆解

1. **認識包的結構**：
   - 包頂：包的最高點
   - 包底：包與包的中間
   - 側壁：圍著包的一圈（最簡單的位置）

2. **階段一：橫切蘑菇練習上下**：
   - 橫切過包時，通過膝蓋和腳踝上下吸收包的起伏
   - 上去、下來、上去、下來
   - 感受身體的上下節奏
   - 不要急著轉彎

3. **階段二：選擇側壁轉彎**：
   - 橫切過幾個包後，下定決心找一個包的側壁
   - 在側壁位置轉彎（像在旗門旁邊往上滑轉彎）
   - 轉彎後繼續橫切
   - 再找下一個包的側壁轉彎

4. **階段三：縮短橫切距離**：
   - 通過熟練度逐漸縮短橫切雪道的距離
   - 最終達到每次都在包的側壁上轉彎
   - 不再需要長距離橫切

5. **選擇較緩的包**：找比較緩一點的包來練習，降低難度

### 教練提示

- 路線選擇比技巧更重要
- 側壁是最簡單的位置，避開包頂和包底
- 不要急於求成，先練好橫切上下再轉彎
- 側壁轉彎就像在旗門旁邊轉彎
- 循序漸進縮短橫切距離

### 常見錯誤

- 一開始就想在包頂或包底轉彎
- 沒有先練習橫切上下就急著轉彎
- 不知道側壁是最簡單的位置
- 橫切距離沒有循序漸進地縮短
- 選擇太陡的包來練習

### 修正方法

- 先在小包區域練習橫切上下
- 熟練後再選擇側壁轉彎
- 刻意選擇較緩的包
- 每次練習都有意識地縮短橫切距離
- 把側壁想像成旗門旁邊的位置
