## 141 陡坡換刃第三集：雙腿承重Alignment

### 問題描述 (What)
在很陡的雪道（雙黑陡度）且雪比較爛時，身體打開太多會間接導致壓到一條腿上，無法充分利用雙腳支撐，在陡坡上不夠穩定。

### 改善目標 (Why)
盡可能多利用雙腳給支撐，讓身體與板齊平，雙腳承重達到最穩定狀態，在陡坡上滑行越來越穩定。

### 改善方法 (How)
**核心原則**：
- 身體打開換刃效率高，但不一定適用於很陡的地方
- 需要盡量讓身體與板齊平

**何時齊平**：
1. **換刃時可以稍微打開一點**（因為陡坡換刃機會少，打開增加效率）
2. **換過來之後要追求身體與板齊平**
3. **尤其是往下衝這一塊**（從滾落線往下衝，速度很快）
4. **這樣可以把速度降下來**

**後刃雙腳承重感受**：
- 由於坡度陡會害怕，身體會情不自禁往後倒
- **讓兩個脛骨貼著舌頭**
- 身體在後刃時甚至有一種要往下撲的感覺（不是真的往下撲）
- 脛骨貼著舌頭，就會直立於陡坡之上

**前刃雙腳承重感受**：
- 不能往後仰，一仰會感覺要折下去
- **需要下降**
- 下降時會感覺背板這個位置或鞋子頂住了小腿
- 有點像小時候玩的：後面人鎖喉一蹬壓後膝蓋窩會跪下那種感覺
- 感覺像被人頂了一下往下這麼一壓
- 會感覺後面這兩個位置貼緊了
- **這就是雙腳承重最穩定的狀態**

**關鍵要點**：
- 必須兩個腳都感覺到
- 尤其是越來越往下拐的時候
- 一開始可以一個彎一個彎地拐
- 之後就可以慢慢把它連起來

**核心**：脛骨貼舌頭（後刃）+ 背板頂小腿（前刃）= 雙腳承重最穩定

## 142 陡坡第五集：直板入

### 問題描述 (What)
遇到懸崖或樹時，不知道如何安全落地，如果板子橫過來會非常危險，受到衝擊時人會整個往前翻滾著往下走停不下來。

### 改善目標 (Why)
讓板子直著往下走，落地後板子逐漸打橫而非立刻打橫，安全通過懸崖或樹的障礙。

### 改善方法 (How)
**核心原則**：
- **絕對不能讓板子橫過來**
- 要讓板子直著往下走
- 落地之後板子去逐漸打橫
- 不能說立刻打橫

**原因**：
- 往下的衝擊力會非常大
- 如果橫著板子下去會非常危險
- 一旦受到衝擊，人會整個往前翻滾著往下走
- 會停不下來

**彎形特點**：
- 基本上會遇到的彎形就是從滾落線往下、滾落線往下、滾落線往下
- 有點像初學者轉彎一樣

**適用場景**：
- 接觸跳崖或樹的第一步
- 懸崖
- 樹之類的障礙

**核心**：直板入 → 落地後逐漸打橫 ≠ 橫著下去

## 144 陡坡換刃第四集：上帶下

### 問題描述 (What)
在陡坡上推著推著停不住了，想用下半身帶動上半身去轉彎但做不出來，不知道何時用上帶下、何時用下帶上。

### 改善目標 (Why)
理解上帶下和下帶上的使用時機，在陡坡上能夠根據情況選擇正確的轉彎方式。

### 改善方法 (How)
**上帶下的使用時機**：
1. **當停住的時候**
2. **想轉下一個彎**
3. **用上半身去帶下半身**
4. **原因**：上帶下的力量會更大，需要這麼大的力量去瞬間讓人轉過來

**下帶上的使用時機**：
1. **沒有停住**
2. **在上一個彎帶著速度**
3. **壓一下**
4. **當壓這麼一下的時候就可以給空間**
5. **讓髖去進行下帶上**
6. **這麼把彎轉過來**

**實際應用**：
- **第一開始滑的時候（啟動）**：第一個彎希望是上帶下
- **之後如果做對了**：可以一直下帶上
- **如果有一個彎速度太快有點停住**：再重新開始再 reset，就是上帶下了

**兩種方法都是對的**：
- 只不過是看用到什麼時候

**核心**：停住時 = 上帶下；帶速度時 = 壓一下 + 下帶上

## 145 陡坡黑道基礎換刃第一集：跳換（黑鑽）

### 問題描述 (What)
在爛雪陡坡裡面一看到陡坡會害怕身體僵住，壓在前刃的刃上會卡著，不知道如何換刃。

### 改善目標 (Why)
讓上下起伏出來，用整個板底來支撐，壓在掌中寶位置，通過跳起來讓板刃從爛雪裡得到釋放，能夠轉過來。

### 改善方法 (How)
**核心原理**：
- 要讓上下起伏出來
- 有點像泡泡一樣

**關鍵位置**：
- **要感知到掌中寶的位置**
- **壓在掌中寶**
- 這樣用整個板底來支撐了
- 如果壓在前刃的刃上會卡著

**運動模式**：
1. **壓在掌中寶**
2. **然後嘗試下壓**
3. **然後跳起來**
4. **當起來之後，板刃從爛雪裡面會得到釋放**
5. **這樣就能轉過來**

**重要認知**：
- 練習的目的不是讓跳起來
- 而是為了讓習慣這種運動模式
- 要不然不釋放板刃，這麼一轉就卡住了

**練習順序**：
- 壓住
- 推
- 等壓穩了
- 下降
- 習慣了運動模式
- 然後再滑

**核心**：壓掌中寶 → 下壓 → 跳起來 = 板刃釋放能轉過來
