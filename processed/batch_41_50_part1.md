# 41 什麼是立刃？如何立刃？需要哪種立刃方式

## 📝 教學內容

### 問題描述 (What)
不清楚什麼時候用什麼樣的立刃方式，不了解傾倒和折疊的差異和適用情境。

### 改善目標 (Why)
- 了解立刃的兩種方式：傾倒和折疊
- 知道什麼時候用什麼樣的順序
- 根據雪況選擇合適的立刃方式
- 讓板子不會被蹬飛出去，不會蹬離開

### 改善方法 (How)

**立刃兩種方式**：

1. **傾倒立刃**：
   - 靠板子的傾倒
   - 身體一倒，刃就立在雪裡面，邊刃就扎在雪裡面
   - 速度快，但沒有辦法做出細微的調節
   - 一旦傾倒了，傾斜角度是變大的，或者說是很難控制的

2. **折疊立刃**：
   - 靠折疊
   - 關節可以調節這個刃、雪板立刃的高低
   - 相對來說得慢一點，因為需要至少三個關節的配合：髖、膝、還有腳踝
   - 有時候甚至可能需要加大一些旋轉之類的東西
   - 可以做出細微的調節

**適用情境**：

**傾倒適用**：
- 雪非常好的，極佳的面條雪
- 速度越快，傾倒的角度也可以越大

**折疊適用**：
- 當沒有速度的時候，要靠折疊
- 當雪不太好的時候，要靠折疊去進行立刃
- 在爛雪裡、粉雪裡、或者像沙子一樣的雪裡面
- 因為那個時候相當於在細微的調節

**為什麼立刃這麼關鍵**：
- 當把刃立住之後，在雪板上給的壓力、蹬雪板的時候，板子不會被蹬飛出去，不會蹬離開

**順序建議**：
- 先傾倒再折疊：在比較硬一點的雪、抓地力比較強的雪的情況下可以做，立刃是非常快的，帶的比較快的速度，壓根沒有想去減速
- 先折疊再傾倒：立刃是比較慢一點的，折疊增加了很多，甚至一開始沒有想去傾倒，永遠立於板子上面，但是搓的力量就會大了很多

**控速建議**：
- 如果想控速控得更好的話，少立刃用板底去搓，速度的控制會更加的好，可以稍微慢一點
- 如果想保持一定的速度的話，需要建立更多的立刃

# 42 Saddle 小彎

## 📝 教學內容

### 問題描述 (What)
做小彎或中彎的時候，因為雪況發生了改變（地形不變、雪況改變），比如說比較冰一點、比較硬一點，或者雪比較好一點、比較軟一點。當積壓過，滑出來的小彎動作雖然一樣，但是滑出來的感覺感受、甚至彎形、或者能不能搓住雪都不一樣。

### 改善目標 (Why)
- 根據雪況調整立刃、施壓、軸轉的比例
- 增加整個的表現力 performance
- 在不同雪況下都能有效控制

### 改善方法 (How)

**根據雪況調整三要素**：

1. **比較冰一點的時候**：
   - 把軸轉增加
   - 把立刃跟施壓都減少一點，或者說稍微緩慢一點、可控一點

2. **雪比較好的時候**：
   - 可以把立刃、施壓增加
   - 把軸轉減少到最少
   - 這樣可以增加整個的表現力 performance

**三要素**：
- 立刃
- 施壓
- 軸轉
