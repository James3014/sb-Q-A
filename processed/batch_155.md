# 155. 深聊上半弧

## 問題描述 (What)

我對上半弧的理解不夠深入，可能只知道要做上半弧，但不清楚什麼是上半弧、做對了會有什麼感覺、為什麼要做上半弧、以及在什麼情況下必須要有上半弧。這導致我在進階學習和考級時無法展現出完整的弧形對稱性，也無法充分利用上半弧來控速和蓄能。

## 改善目標 (Why)

我需要從四個層面深入理解上半弧：

1. **定義**：上半弧是從橫切滾落線到滾落線的這一段，做對時上半弧和下半弧應該能折疊重合（上下對稱）

2. **身體感覺**：
   - 前腿和板子前半部分壓力較大
   - 力量從前腿慢慢往後腿傳遞
   - 不應該出現壓住前腿但板尾直接掃出去的情況

3. **優點**：
   - 實現上下弧對稱
   - 可以從滾落線之前就開始控速（早立刃早施壓）
   - 可以在上半弧蓄力（Loading Deflection），在任何位置釋放能量

4. **體系重要性**：過了初級階段（CASI一級二級）往上走，尤其是小彎、中彎、大彎，在壓雪道上必須展現上半弧

## 改善方法 (How)

### 步驟拆解

1. **理解上半弧定義**：
   - 從橫切滾落線位置到滾落線的這一段
   - 上半弧和下半弧應該能折疊重合
   - 地上留下的痕跡應該是上下對稱的月牙形

2. **感受正確的身體感覺**：
   - 換刃後在橫切滾落線位置，前腿和板子前半部分壓力較大
   - 在上半弧過程中，力量從前腿慢慢往後腿傳遞
   - 避免壓住前腿但板尾直接掃出去

3. **利用上半弧控速**：
   - 在滾落線之前就換重心和立刃
   - 可以在滾落線之前就開始施壓推雪
   - 實現勻速過彎，不會在下半弧累積過多速度

4. **利用上半弧蓄能**：
   - 在上半弧位置施壓讓板子形變（Loading Deflection）
   - 可以選擇在上半弧、滾落線或下半弧任何位置釋放能量
   - 讓力量運用更多元化（例如臀跳）

5. **考級應用**：
   - CASI三級四級以上，小彎中彎大彎必須展現上半弧
   - 在壓雪道上把弧畫圓
   - 蘑菇地形相對靈活，但壓雪道必須有上半弧

### 教練提示

- 上半弧的核心是上下對稱，不只是左右對稱
- 可以通過地上的痕跡檢查上下弧是否對稱
- 不要過度追求上半弧而把板尾掃出去
- 力量傳遞的速度取決於彎形大小和地形
- 如果地形不允許，不用強制做上半弧，但壓雪道必須有

### 常見錯誤

- 只知道要做上半弧，不理解為什麼
- 過度追求上半弧，壓住前腿把板尾掃出去
- 沒有感受到力量從前腿往後腿的傳遞
- 不會利用上半弧來控速和蓄能
- 考級時在壓雪道上沒有展現上半弧

### 修正方法

- 觀察地上的痕跡，檢查上下弧是否對稱
- 練習時專注於前腿壓力和力量傳遞的感覺
- 在緩坡練習早立刃早施壓，感受上半弧控速
- 練習在上半弧蓄力，在不同位置釋放能量
- 在壓雪道上刻意練習完整的上半弧
