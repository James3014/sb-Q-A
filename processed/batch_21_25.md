# 21 一個練習：改肩帶轉

## 📝 教學內容

### 問題描述 (What)
在滑行時會出現肩帶轉的問題，尤其是在小回轉時。在 Apex（頂點）起跳時，會不自覺地轉一下肩膀，這是身體的本能反應。

### 改善目標 (Why)
- 避免肩帶轉
- 讓下半身（髖、膝、踝）去轉
- 通過下半身的下降來穩定肩膀不要轉
- 通過髖慢慢讓彎轉過來

### 改善方法 (How)

**Apex 頂點起跳練習**：

1. **在 Apex 起跳**：
   - Apex 是彎的頂點，下一個圓發起的地方
   - 很多人在這裡會進行肩帶轉

2. **避免肩膀轉動**：
   - 在頂點跳的時候，盡量避免去轉一下肩
   - 通過下半身的下降來穩定住肩不要轉
   - 而是通過髖慢慢慢慢去讓彎慢慢轉過來

3. **練習方法**：
   - 換到新刃之後，在 Apex 跳跳跳跳一到兩下
   - 然後直接跳到滾落線
   - 這個可以幫助感知問題，同時訓練不要在這個位置出現肩帶轉

# 22 刃的角度：該高？該低？

## 📝 教學內容

### 問題描述 (What)
不清楚應該高立刃還是低立刃，不了解兩者的適用情境。

### 改善目標 (Why)
- 理解高立刃和低立刃的差異
- 根據雪道坡度調整立刃角度
- 追求最強的抓地力
- 讓板子能夠形變

### 改善方法 (How)

**根據坡度調整立刃角度**：

1. **低立刃的特點**：
   - 板底會多支撐，在板子上更加穩定
   - 板底接觸面積大，板子不容易形變
   - 適用於較陡的雪道

2. **高立刃的特點**：
   - 可以卡住雪，給板子最大的壓力去施壓
   - 讓板子形變
   - 適用於較平的雪道（如綠道）

3. **坡度與立刃的關係**：
   - 在平地上，板子跟坡度的角度很小
   - 在陡坡上，刃與坡度之間的角度巨大
   - 刃的角度會根據雪道的坡度而改變

4. **應用原則**：
   - 在陡的道上：盡量把刃的角度壓低（低立刃），多讓雪板貼著雪
   - 在平的道上：盡量把角度提起來（高立刃），可以使勁去頂板給板壓力讓板子形變
   - 取決於雪道的坡度，而不是想高立刃還是低立刃

**關鍵提示**：
- 所有的目的都是為了對板子施壓，讓板子發生形變
- 這樣抓地力就是最強的
- 追求的是抓地力怎麼增強，而不是刃到底該高還是該低
- 立刃跟膝蓋與腳踝息息相關

# 23 小回轉刃立多了，容易被板帶著走

## 📝 教學內容

### 問題描述 (What)
轉小彎時，由於立刃過高，導致板刃卡在雪裡。彎的上半段基本上都是刃帶著走，等到下半段刃稍微維持住了或稍微低一點了，然後施壓把彎轉過來。

### 改善目標 (Why)
- 防止刃卡雪裡
- 避免彎的上半段被板子帶著走
- 讓彎的上半段也能有抓地力
- 提升小彎的控制能力

### 改善方法 (How)

**推板搓雪技巧**：

1. **小彎搓雪**：
   - 如果是小彎的話，盡量去讓板子變形去搓雪
   - 這樣刃就不會卡雪裡了

2. **推板動作**：
   - 在彎的上半段的時候，對板子進行一個推板的動作
   - 在上半段開始就推板
   - 後刃推板：可以看到明顯的推板動作
   - 前刃推板：基本上會壓鞋舌，不會看到大的推板動作

3. **從換刃開始**：
   - 每次在換刃的這個位置開始
   - 就已經沒有讓刃感覺那麼強了
   - 從上面就開始去搓旋了

**關鍵提示**：
- 隨著加入推板動作，彎的上半段不會被板子帶著走
- 從換刃位置就開始搓雪，不要等到下半段

# 24 平窄粘的連接道：髖關節打開一些

## 📝 教學內容

### 問題描述 (What)
在平、窄、粘的連接道上滑行，雪況已經被壓硬且比較顛簸比較窄，容易卡刃摔倒。

### 改善目標 (Why)
- 更容易刃到刃之間切換
- 防止卡刃
- 在顛簸窄道上更穩定
- 換刃速度更快

### 改善方法 (How)

**髖關節打開技巧**：

1. **基礎要點**（前兩天講過）：
   - 稍微蹲低一點
   - 靠踢後腿
   - 盡量在刃上

2. **髖關節打開**（第三步）：
   - 把身體尤其是髖，稍微往板頭方向打開一點點
   - 更容易刃到刃之間切換

3. **打開的效果**：
   - 完全站在後刃上時，打開一點身體後，後腳會把板尾漸漸給踩平
   - 前腳還是保持在後刃
   - 從這個位置一旦換到前刃時，後腳就直接踩到雪裡了
   - 前腳就會變成平一點

4. **為什麼有效**：
   - 身體打開一點時，在這種道上更有助於刃與刃之間的切換，會更快
   - 盡量防止卡刃
   - 兩個腳都在刃上時，很容易有一個腳由於道比較顛簸不平，兩邊一晃就容易卡刃
   - 一卡刃就摔個大的

**關鍵提示**：
- 讓髖稍微朝板頭打開一點，保持都打開就好

# 25 冰蘑菇：處理"冰棱"

## 📝 教學內容

### 問題描述 (What)
在蘑菇裡面，雙板或單板會切出一棱一棱的冰棱。包頂是粉雪，過了粉包頂之後下面全是冰，非常硬。遇到這種棱時會被顛得很慘，而且速度還降不下來。

### 改善目標 (Why)
- 能夠穩定地通過冰棱
- 避免被顛得很慘
- 保持居中在板子上
- 能夠把速度刹住

### 改善方法 (How)

**居中站姿 + 腳踝膝蓋控制技巧**：

1. **前刃技巧**：
   - 盡量把人在板子上面居中一點
   - 讓腳踝往上勾
   - 雖然壓的是前刃，由於腳踝可以往上勾，會更吸在板子中央多一些
   - 遇到這種棱時，可以很簡單地去挺過去
   - 讓腳踝緊一點，盡量讓板子在下面去顛，而不是顛人

2. **後刃技巧**：
   - 盡量讓脛骨小腿貼著鞋舌
   - 脛骨貼著鞋舌
   - 一個彎轉過來如果要刹不住的話，可以比較穩定一點

3. **推坡練習**：
   - 在包裡面練習推坡
   - 不只是單純的往下推
   - 要確保人是居中在板子上面的
   - 前刃需要讓鞋舌更緊（腳踝這個地方）
   - 後刃需要脛骨貼著鞋舌
   - 這樣在上面滑的時候，人是比較居中的
   - 在這種又冰還有棱，一道一道的還陡的情況下，就能把速度刹住

**關鍵提示**：
- 這是稍微有點高級的東西，如果剛接觸蘑菇不建議嘗試
- 用"挺"這個字是無奈，因為沒辦法，這個棱就會把人顛得很慘
