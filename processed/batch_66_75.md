## 66 如何冰面搓雪轉彎

### 問題描述 (What)
在冰面且坡度較陡的情況下，小彎難以像平常一樣完成，容易失控或缺乏抓地力。

### 改善目標 (Why)
在冰面陡坡上既能控制速度，又能保持一定的抓地力，完成穩定的轉彎。

### 改善方法 (How)
1. **降低力刃（傾倒角度）**：減少身體的傾倒幅度
2. **減少壓力**：不要過度施壓板刃
3. **增加軸轉**：加大旋轉動作來完成轉彎
4. **保持居中站姿**：站姿永遠處於板子上方，這是完成以上三個變量的關鍵
5. **調整預期**：冰面上無法像好雪況那樣留下深槽印，不要試圖用力量對抗冰面

**核心原則**：大量軸轉 + 少量力刃 + 少量壓力 = 冰面控制

## 67 如果過度開肩開胯：試試這個動作

### 問題描述 (What)
中等水平滑手在換刃後保持開胯開肩站姿，但過度開胯導致過度依賴後腳，視線無法有效引導轉彎。

### 改善目標 (Why)
讓胯部與板齊平，實現雙腳同時承重，在陡坡或蘑菇地形中更加穩定。

### 改善方法 (How)
1. **前肩視線法則**：在前刃快結束時，讓視線永遠用前肩來看
2. **檢查方法**：如果開胯開肩時視線看前面什麼都沒有，說明胯部過度打開
3. **正確位置**：當肩膀在正確位置時，胯部會自然閉合與板齊平
4. **效果**：胯部閉合可以幫助雙腳承重，讓彎更加閉合
5. **適用場景**：特別適合剛接觸陡坡或蘑菇地形時使用

**對比練習**：先做過度開胯的錯誤示範，再做胯部閉合的正確示範，感受雙腳承重的差異

## 68 團身過蘑菇：解決忽快忽慢

### 問題描述 (What)
團身過包時，如果身體太直，到包頂時來不及反應，導致速度忽快忽慢，無法流暢通過。

### 改善目標 (Why)
實現快速高效的團身動作，越走包頂效果越好，保持流暢穩定的速度。

### 改善方法 (How)
1. **預先彎曲**：不是站直了再團身，而是本來就保持彎曲姿態
2. **從彎曲到團身**：已經彎著的狀態下再去團身，反應會非常快且高效
3. **時機掌握**：如果身體太直，當包頂到你時已經沒有時間反應
4. **漸進練習**：先從比較小的包開始，再到比較大的包
5. **核心原理**：預先彎曲 → 快速團身 → 高效吸收壓力

**關鍵**：團身的效率取決於起始姿態，越接近包頂越需要預先彎曲

## 69 如果彎轉得急：試試這個順序

### 問題描述 (What)
轉彎過於急促，導致轉彎順序被打亂或某些元素缺失，雖然能轉過來但不夠流暢圓潤。

### 改善目標 (Why)
讓轉彎更加流暢圓潤，不會顯得一蹴而就或過於急促。

### 改善方法 (How)
1. **正確順序**：換重心 → 立刃 → 施壓 → 軸轉（四個步驟）
2. **先放慢練習**：先轉大彎，充分感受這四個步驟
3. **逐步加快**：通過熟練度慢慢加強，不要一開始就求快
4. **拆解練習**：把四個過程先拆開做，再通過熟練度組合起來

**關鍵認知**：急促轉彎會導致順序混亂或元素缺失，必須先建立正確順序再追求速度

## 70 如果害怕蘑菇裡後刃換前刃

### 問題描述 (What)
在蘑菇地形中，從後刃重心轉移到前刃時感到恐懼，不敢把重心放在前刃上。

### 改善目標 (Why)
建立前刃重心的信心，在蘑菇地形中如魚得水地完成後刃換前刃。

### 改善方法 (How)
1. **專項練習**：在蘑菇地形練習前，先做5趟前刃直推練習
2. **前刃直推**：重心放在前刃上，一直往下推，不換刃
3. **重心位置**：把重心放在掌中寶位置（前腳掌）
4. **克服障礙**：不管前面有落差、冰面或任何情況，都持續推
5. **感受訓練**：會感覺到大腿燃燒、膝蓋彎曲，這是正常的
6. **實戰應用**：推5趟後再去做後換前，會發現容易很多

**原理**：當習慣了重心在前面的感覺，後刃換前刃時就不會害怕，反而會相信能壓在前刃位置

## 71 如果陡坡後刃彎腰撅屁股：試試這個

### 問題描述 (What)
在陡坡（黑道或更陡）後刃下蹲時，髖關節與板子平行，導致下蹲後屁股撅出去，身體無法居中。

### 改善目標 (Why)
在陡坡上下蹲後仍能保持居中，板子不往下推，同時保持良好的彎形。

### 改善方法 (How)
1. **髖關節打開**：下蹲時髖關節稍微打開，不要與板子完全平行
2. **下蹲方向調整**：朝著稍微打開的方向下蹲，而非垂直向下
3. **幅度增加**：陡坡需要的下蹲幅度比綠道大得多
4. **對比認知**：
   - 錯誤：髖關節與板平行 → 下蹲後屁股撅出去 → 身體不居中
   - 正確：髖關節打開 → 下蹲後身體居中 → 彎形正確
5. **視覺效果**：正確姿態看起來更舒服（proper）

**關鍵**：陡坡下蹲不是單純往下蹲，而是髖關節打開後朝特定方向下蹲

## 72 安全通過連接道：只用刃，別平板

### 問題描述 (What)
在又窄又平的連接道上，使用平板直衝容易在棉硬雪面上失去平衡，不經意就卡刃摔倒。

### 改善目標 (Why)
在窄平連接道上安全通過，避免卡刃摔倒（平地摔最疼）。

### 改善方法 (How)
1. **只用刃原則**：直衝時不要用平板，要麼在前刃要麼在後刃
2. **快速換刃技巧**：
   - **前刃換後刃**：腳踝邊躺邊勾起來，直接換到後刃，不經歷平板
   - **後刃換前刃**：稍微勾腳踝，膝蓋彎一點貼著，直接壓到前刃
3. **結合踢後腿**：在刃上更容易做踢後腿動作
   - 前刃時：用前刃踢後腿
   - 後刃時：用後刃推後腿
4. **保持專注**：千萬不要放鬆，一旦放鬆容易卡刃
5. **累了就休息**：如果累了坐下休息，不要勉強

**核心**：刃到刃的轉換 + 受控的換刃 = 安全通過

## 73 盡早施壓小練習

### 問題描述 (What)
在陡坡或複雜雪況下，施壓時機太晚，無法有效控制速度或在需要時減速。

### 改善目標 (Why)
盡早施壓，在過滾落線後有更多機會不減速或在需要時還能減速，打出提前的負力量。

### 改善方法 (How)
1. **噴濺練習**：用板子推雪時，把雪往雪道兩邊噴濺
2. **目標物練習**：找樹或杆子，嘗試把雪推到樹上或杆子上
3. **時機提前認知**：
   - 想在滾落線位置噴濺 → 需要在滾落線前就開始蹬雪
   - 想在滾落線前蹬雪 → 需要在更早位置就下降
   - 下降 → 腿才有力臂 → 才能蹬腿噴濺
4. **順序**：提前下降 → 獲得力臂 → 蹬腿施壓 → 雪往外噴濺
5. **效果**：間接培養盡早施壓的感覺

**核心概念**：不要光想著在滾落線噴，要想著在滾落線前就要下去準備

## 74 節奏變快後：動作變形怎麼處理

### 問題描述 (What)
在陡坡或平面快速轉彎時，雖然轉得快也順，但重心沒有換過來，是換刃後才踩重心，而非用重心換刃。

### 改善目標 (Why)
用重心換刃而非換刃後踩重心，讓施壓能從彎的早期就開始，而非只集中在彎的後半段。

### 改善方法 (How)
1. **順序糾正**：應該是「用重心換刃」而非「換刃後踩重心」
2. **問題分析**：
   - 錯誤：換刃 → 重心踩過去 → 只能在彎後半段施壓
   - 正確：重心換過來 → 帶動換刃 → 整個彎都能施壓
3. **練習方法**：先轉大彎，充分感受「重心換刃」的過程
4. **漸進提速**：感受正確後再慢慢變形加快
5. **效果**：
   - 可以利用下半身關節的彎曲與伸展造成壓力
   - 可以蹬雪減速
   - 速度控制更好

**核心差異**：重心先行 vs 刃先行，決定了施壓的時機和範圍

## 75 入彎如果是這樣：你得看看這個視頻

### 問題描述 (What)
彎結束時重心在後腳（正確），但進入下一個彎後還是在後腳，導致前腿失去抓地力，上半弧幾乎為零。

### 改善目標 (Why)
讓重心在彎結束後回到中間或往前，使上半弧存在，前腿有抓地力。

### 改善方法 (How)
1. **板尾回彈原理**：利用板尾的彈力把重心彈回中間
2. **中間位置（Neutral Position）**：
   - 前刃換後刃、後刃換前刃都要回到中間位置
   - 板頭板尾也一樣，結束時回到中間
3. **凹力練習**：
   - 先在原地練習凹力（Ollie）
   - 膝蓋與腳踝稍微彎曲
   - 當板尾出現彈力時，主動朝板頭方向移動
4. **實戰應用**：
   - 彎結束時，板尾給一個簡單回彈
   - 幫助重心回到中間
   - 不會像練習時那麼誇張（人不會真正跳起來）
5. **感受培養**：每次彎結束時，人整體有點像往板頭方向哭一下

**關鍵**：利用板尾彈力 + 主動移動 = 重心回中間
