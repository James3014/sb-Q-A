# 166. 板頭板尾（前中後）重心轉移

## 問題描述 (What)

我在滑行時對板頭板尾的重心移動感到困惑，雖然知道這能讓滑行更流暢、更Dynamic、更靈動，但不清楚為什麼會有這種效果，也不知道該移動到什麼位置、用什麼方式移動。我可能只是在模仿動作外形，卻沒有真正理解其中的原理，導致動作徒有其表，甚至可能因為重心移動過度而失去板尾的抓地力。

## 改善目標 (Why)

我需要理解板頭板尾重心移動的底層邏輯——它是通過改變抓地力分布來實現動態滑行的。當我往板頭移動時，板頭抓地力增強，但板尾抓地力會減弱，所以後腳必須及時跟上補回抓地力。我要學會控制移動的範圍（保持在固定器內側的安全範圍），並掌握兩種移動方式：髖部主動移動（慢但力量足）和腳部拉板（快但力量小）。這樣才能真正感受到dynamic的滑行，而不只是照貓畫虎。

## 改善方法 (How)

### 步驟拆解

1. **理解抓地力原理**：
   - 站在板上，感受往板頭移動時板頭抓地力增強、板尾抓地力減弱的變化
   - 這是dynamic滑行的來源

2. **確定安全範圍**：
   - 重心移動時保持在固定器內側
   - 不要超過固定器外側（避免滯留在板頭的感覺）
   - 超出會像做極限的nose press或tail press

3. **練習髖部移動**：
   - 用髖部（核心）主動移動重心
   - 這種方式較慢但力量充足
   - 因為這是核心

4. **練習腳部拉板**：
   - 通過腳把板子往回拉
   - 這種方式較快但力量較小
   - 因為腳離板子連接更近

5. **入彎時應用**：
   - 在橫切滾落線入彎時，往板頭移動增加板頭抓地力
   - 然後後腳立即跟上補回板尾抓地力
   - 這個過程讓滑行更dynamic

6. **整合動作**：
   - 熟練後，重心的前後移動會自然產生臀跳的效果

### 教練提示

- 重心移動的目的是改變抓地力分布，不是單純的前後晃動
- 移動範圍要控制在固定器內側，超出會有極限感
- 後腳要及時跟上，不能只移動前腳而丟失板尾抓地力
- 兩種移動方式各有優劣：髖部移動慢但穩，腳部拉板快但輕
- 不要只做外形動作，要理解抓地力變化的原理

### 常見錯誤

- 只做板頭板尾的外形動作，沒有理解抓地力變化的原理
- 重心移動超過固定器外側，導致滯留在板頭或板尾
- 只往板頭移動而後腳沒跟上，丟失板尾抓地力
- 不知道有兩種移動方式，只用一種方法
- 徒有其表，照貓畫虎

### 修正方法

- 先在平地練習，感受重心移動時板頭板尾抓地力的變化
- 用固定器位置作為參考點，確保重心不超出安全範圍
- 刻意練習後腳跟上的動作，保持板尾抓地力
- 分別練習髖部移動和腳部拉板，體會兩種方式的差異
- 理解原理後再做動作
