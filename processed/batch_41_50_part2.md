# 43 Elongate 的彎

## 📝 教學內容

### 問題描述 (What)
滑起來有點像個機器人，動作比較標準但是熟練度上面還可以加強一些。遇到下午的雪、被滑爛了、有點像春雪死辣式那種感覺、崎嶇不平的雪的時候，會被顛起來。

### 改善目標 (Why)
- 滑的不像一個機器人
- 遇到崎嶇不平的雪的時候不會被顛起來
- 可以找到一種回彈的感覺，幫助進入下一個彎比較輕鬆
- 讓重心在板子上面板頭板尾移動

### 改善方法 (How)

**拉長的 Ollie 技巧**：

1. **核心概念**：
   - 在滑的過程中加入一個很長被拉長的 Ollie
   - 不是快速的 Ollie，而是把這個過程拉長

2. **加入到彎形**：
   - 可以把這個東西加入到彎形裡面
   - 當遇到這種崎嶇不平的雪的時候，可以快速做一個 Ollie 把雪吹出去

3. **效果**：
   - 第一可以找到一種回彈的感覺，幫助進入下一個彎比較輕鬆
   - 第二不會被這種雪給把顛起來
   - 如果滑的有點緊有點緊的話，會被顛起來

4. **重心移動**：
   - 當滑的時候，可以時不時的讓人在板子上面板頭板尾的讓這個重心移動一動
   - 最簡單的一個方式就是把 Ollie 慢慢加進去

5. **時機**：
   - 比如說在後刃，感覺入刃之後加一個 Ollie
   - 前刃入刃之後加一個 Ollie
   - 當做 Ollie 的時候，在刃裡面做 Ollie 的時候，一定要入刃之後去做
   - 不要往下推坡的時候去做，這樣的話是沒有任何意義的
   - 板子在彎形裡面改變不了它的形狀，也就產生不了往這邊送的那股力量

# 44 Deflection 傳導（回彈）簡單邏輯

## 📝 教學內容

### 問題描述 (What)
在粉雪裡或者雪軟一點的時候，不知道怎麼去借助這個彈力去幫助滑行。

### 改善目標 (Why)
- 借助彈力幫助滑行
- 讓滑的更加的有表現力
- 同時也能讓滑的更加省力
- 讓腿比較輕鬆
- 可以借助這個反彈力幫助去進入下一個刃

### 改善方法 (How)

**Deflection 傳導回彈技巧**：

1. **推的方向**：
   - 當去推雪板的時候，讓雪板往兩側去推，往兩側滾落線
   - 不要在滾落線之後去推

2. **滾落線之後推的問題**：
   - 一旦在滾落線之後去推，往下推
   - 這個反彈力板子給的反彈力，頂多是讓人站起來
   - 但是不會讓往山上方向去走

3. **往兩側推的好處**：
   - 如果往兩側去推的話，這個反彈力是可以往兩側去走的
   - 這樣當換刃的時候，腿就是比較輕鬆的
   - 可以借助這個反彈力幫助去進入下一個刃
   - 這也就是 KC 裡面說的 deflection

4. **速度要求**：
   - 當做這個的時候，記著速度是一定要有的
   - 不能沒有速度
   - 如果沒有速度的情況下，這個力量傳導不過去
   - 就會感覺不到這個彈力
   - 從而 deflection 這個反彈力也出現不了

# 45 髖關節這樣做｜第一集：後刃彎末不卡

## 📝 教學內容

### 問題描述 (What)
前刃轉後刃，已經轉到後刃之後，上下半身由於髖關節沒有持續地保持一個打開一點點的狀態，或者說沒有持續讓它稍微轉動一點，會在彎的下半段阻擋後刃繼續往這邊收彎。在這個位置的時候，有時候由於手或者肩或者髖沒有繼續往左邊打開，導致會出現這個情況，沒有辦法把這個彎劃完。

### 改善目標 (Why)
- 讓後刃能夠繼續往這邊收彎
- 能夠把彎劃完
- 讓板子可以繼續往山上方向行進
- 當發力的時候也會發的更舒服

### 改善方法 (How)

**髖關節保持打開技巧**：

1. **時機**：
   - 尤其是在滾落線之後

2. **動作要領**：
   - 需要把髖關節保持打開
   - 或者甚至可以加一點點髖關節的軸轉
   - 來幫助讓前手還有肩放在後刃上面
   - 一直保持放在後刃上面
   - 這樣就可以轉彎了

3. **適用情境**：
   - 不管有沒有包，髖關節保持一個打開，讓板子可以繼續往山上方向行進，這個都是非常重要的

**效果**：
- 第一可以幫助後刃結束
- 第二當發力的時候也會發的更舒服
- 在後刃板子繼續行進的過程中

# 46 髖關節這樣做｜第二集：防止後腿掃

## 📝 教學內容

### 問題描述 (What)
轉的時候，想嘗試去打開髖關節，會把重量全部壓在前腳，這麼去打開。如果這樣去打開的話，髖到這兒是沒有辦法再往後轉的，會卡住。

### 改善目標 (Why)
- 讓髖關節能夠順利打開
- 避免髖關節卡住
- 讓重心居中或者稍微在後腳
- 讓髖關節保持旋轉能得到一個很好的效果
- 讓彎都轉完了

### 改善方法 (How)

**後髖往前推技巧**：

1. **核心概念**：
   - 不只是轉動前髖
   - 而是讓後髖去往前面方向去推
   - 這樣前髖就可以順勢而打開
   - 因為它是組合到一起的

2. **動作要領**：
   - 不只是可以轉這邊（前髖）
   - 還可以通過轉這邊（後髖）把這個髖這麼轉過來
   - 當推後髖的時候，把膝蓋朝著前面方向跪下去的時候
   - 髖自然而然就打開了

3. **重心變化**：
   - 當想著去推後髖的時候，重心就會往後腳這邊拉扯一點
   - 這樣當一推，髖關節就打開了
   - 就不會讓前髖在轉動的時候有一種鎖住一樣的感覺

4. **時機**：
   - 這個動作在哪做出來呢，盡量是在滾落線、沿著滾落線或者滾落線之後慢慢去形成的

**關鍵提示**：
- 當再去做這個轉動的時候，重心不是在前腳的，是居中的
- 甚至有時候一開始做，會感覺有點在後腳
- 那就做對了
- 之後可以從慢慢從後腳移到中間就好了
