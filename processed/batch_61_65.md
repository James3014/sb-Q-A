# 61 雙黑前刃站姿：決定了是否能繼續轉

## 📝 教學內容

### 問題描述 (What)
在黑道或者雙黑的道上，後換前時會把頭跟肩探過去，髖跨不過去。所以說會靠髖關節在減震，會顫顫巍巍的。

### 改善目標 (Why)
- 有一個很好的站姿、一個穩定的站姿進入下一個彎
- 能夠用上力，不只是被動去吸收
- 有辦法去通過鞋舌傳遞壓力去把速度降下來
- 在前刃上面更加的穩定

### 改善方法 (How)

**髖壓過來技巧**：

1. **核心動作**：
   - 在比較陡一點的地方，需要讓髖壓過來
   - 需要這樣，而不是頭跟肩探過來
   - 這樣可以幫助更好的吸收

2. **問題分析**：
   - 如果頭跟肩探過來的話，是沒有辦法有一個很好的站姿、一個穩定的站姿進入下一個彎的
   - 因為已經用不上力了，只能被動去吸收了

3. **正確方式**：
   - 在不管多陡的坡上，人盡量髖是壓到前刃上的
   - 這樣的話，是有辦法去通過鞋舌傳遞壓力去把速度降下來了

4. **心理建設**：
   - 最大的一個難點，至少髖頂不過去
   - 最大一個難點是什麼呀？是要下定決心去做這個事情，去 commit
   - 這樣的話，才能夠在前刃上面更加的穩定

**關鍵提示**：
- 不管這個雪況是如何的，髖都是處於一個大概這種狀態頂過來的，而不是頭跟肩
- 越陡的坡上面，但凡有一點點這種感覺，就會往前趴了，就會往雪上面趴了
- 所以說不要出現一點點這種感覺

# 62 平花粉雪通用小技巧：碾轉360

## 📝 教學內容

### 問題描述 (What)
想學會碾轉的 360，在雪道上面或者在粉雪裡做這個動作。

### 改善目標 (Why)
- 學會碾轉 360
- 先跳大概不到 90，然後借助旋轉力幫助轉回到 360 轉回到正腳
- 在雪道上面也可以在粉雪裡去做這個

### 改善方法 (How)

**碾轉 360 步驟**：

**第一步：反腳直線滑行**
- 首先需要學會反腳的直線滑行
- 左腳在前，所以說反腳就是右腳在前
- 重心稍微低一點，這樣的話會感覺到板底多貼著去

**第二步：壓板尾**
- 當在反腳滑行的時候，可以嘗試壓板尾
- 這個主要是幫助當在碾轉的時候，可以借助板尾的彈力
- 把板尾弄到很大

**第三步：身體提前內轉 90 度**
- 這一步稍微比較難
- 當在壓著反腳的後腳的時候，需要讓身體提前先內轉大概 90 度
- 然後伸展後腳壓著那條腿，然後帶回到正腳

**第四步：正腳練習**
- 當在正腳滑行的時候，滑行角度的 45 度
- 在後刃的時候 45 度，然後跳個 180
- 跳過來之後呢，是斜著朝 45 度方向去走的

**第五步：落地壓正腳前腳**
- 落地之後，去嘗試壓正腳的前腳
- 也就是反腳的後腳

# 63 後刃彎不完整怎麼辦

## 📝 教學內容

### 問題描述 (What)
前後腳的重心是對了，但是每次在彎結束的時候，由於身體尤其是前刃過度的反擰，導致前刃轉後刃的時候，入後刃那個彎的時候，板頭立刻朝下。

### 改善目標 (Why)
- 讓身體完全與板平行，甚至超出去
- 當進入下一個彎的時候，進入後刃的時候，可以讓前腳產生更強的抓地力
- 在後刃時更穩定
- 能夠把彎轉完整

### 改善方法 (How)

**身體完全閉合技巧**：

1. **核心動作**：
   - 當在前刃結束的時候，需要把前腳的那個屁股蛋子夾緊
   - 讓身體完全與板平行，甚至超出去
   - 身體超出去板子一點也可以

2. **好處**：
   - 當進入下一個彎的時候，進入後刃的時候
   - 可以讓前腳產生更強的抓地力
   - 在後刃時更穩定

3. **檢查方法**：
   - 當做對的時候，當把板子完全橫，就是說髖完全轉回來的時候
   - 甚至可以看到前腳背板
   - 這是一個很好的信號

4. **動作順序**：
   - 閉合之後，移入後刃
   - 閉合之後一入後刃，可以踩在這個位置進入控制，也就是上半弧

# 64 團身 Down-Unweighting

## 📝 教學內容

### 問題描述 (What)
不知道什麼是團身，以及如何使用團身技巧。

### 改善目標 (Why)
- 讓雪板一直貼著雪面
- 對施壓來說會更加的快速
- 維持穩定性
- 適用於地形裡面，比如說雪包樹林

### 改善方法 (How)

**團身換刃技巧**：

1. **什麼是團身**：
   - 團身就是處於一個低站姿
   - 然後保持低站姿的情況下進行一個換刃

2. **動作原理**：
   - 假如頭上有一個牆，它會一直壓著，不能站起來
   - 所以說換刃只能是這樣，然後回來，然後這樣
   - 這就是屬於團身換刃

3. **力量運用**：
   - 在蹬出去的那一刻的時候，由於力量比較大
   - 利用這個力量吸收回來用身體
   - 然後再蹬出去，再吸收回來
   - 雪板貼著雪面的一個狀態

4. **適用情境**：
   - 這個方法是適用於地形裡面的
   - 比如說雪包樹林
   - 還有那種有一些要吸收掉的雪道給的重心力的時候
   - 來維持穩定性
   - 穩定性是團身一個好處之一

**練習建議**：
- 可以在室內練習，拿到道外去用
- 但是道內滑雪，這不是滑雪方式，這只是一個練習

**關鍵提示**：
- 團身的滑法會比正常滑行的滑法會累很多
- 因為保持一個低站姿勢，控制力、穩定性

# 65 如何滑下午的冰面

## 📝 教學內容

### 問題描述 (What)
到下午了，上面的好雪已經被刮掉了，面條雪已經被刮掉了，下面剩比較硬或者比較冰的一層雪了。不知道怎麼來控制這個。

### 改善目標 (Why)
- 多讓板底去接觸雪，增加接觸面積
- 幫助更平衡在冰面上
- 保持一個穩定的狀態
- 不管是冰面還是好雪都能應對

### 改善方法 (How)

**冰面站姿技巧**：

1. **後刃站姿**：
   - 人站直的時候，板子跟坡面會產生一個角度
   - 當產生一個角度之後呢，如果往後轉，角度會拉大
   - 為了多讓板底去接觸雪，增加接觸面積
   - 在後轉的時候，會讓兩個膝蓋往前刃方向跪下去，往下壓
   - 當往下跪的時候，讓膝蓋往前壓，會感覺到小腿貼到了鞋舌

2. **前刃站姿**：
   - 同樣一個道理，本身就會產生一個角度
   - 一旦坡變陡或者冰面變冰，本能反應是要往前趴
   - 其實應該做的呢，其實不是往後仰
   - 如果想到了後刃，可能說那是不是應該往後仰
   - 不應該往後仰，只是更加讓重心壓在足弓這個位置
   - 如果壓不到足弓呢，也可以壓掌中寶

3. **壓力控制**：
   - 在這個站姿下，上下去壓縮去給壓力減速
   - 就可以保持一個穩定的狀態
   - 不管是冰面還是好雪，都是一樣的道理

**關鍵提示**：
- 反復強調這個問題很多次
- 尤其是在冰面上
