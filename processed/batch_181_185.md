# 181. 蘑菇第三集：手的位置很重要

## 問題描述 (What)

我在滑蘑菇時，上半身的轉動位置出現問題，導致入彎時要不然是上半身去追下半身，要不然是下半身去追上半身。我不知道如何讓上半身的擺放位置比較OK，導致轉彎不順暢，甚至出現反擰或需要上下追趕的情況。

## 改善目標 (Why)

我需要學會通過手的位置來帶動肩膀，讓上下半身完美結合。關鍵是：換刃之前，先把前手稍微放到要換的刃上面（不是外面）。例如從前刃換後刃時，先把前手放到後刃上面（不是後刃外面），這樣上半身不會打開太多。前刃也是同理，從後刃換前刃時，先讓前手放在前刃上面。就這麼一點點提前量，就可以讓下半身跟上半身完美結合，不會出現反擰或追趕的情況。

## 改善方法 (How)

### 步驟拆解

1. **理解問題**：
   - 上半身位置出現在不應該出現的地方
   - 入彎時上半身追下半身或下半身追上半身
   - 影響轉彎順暢度

2. **理解手帶肩膀的原理**：
   - 手會帶著肩膀
   - 手的位置決定上半身的位置

3. **從前刃換後刃**：
   - 換刃之前，先把前手放到後刃上面
   - 注意：是後刃上面，不是後刃外面
   - 如果放到後刃外面，上半身會大量打開（上帶下）
   - 放到後刃上面，肩膀和上半身不會打太多

4. **從後刃換前刃**：
   - 換刃之前，先把前手放到前刃上面
   - 不要這樣放（外面）
   - 一旦放外面，板尾會甩出去

5. **提前量的重要性**：
   - 就這麼一點點提前量
   - 可以讓下半身跟上半身完美結合
   - 不會出現反擰
   - 不會出現需要上下追趕的情況

### 教練提示

- 手會帶著肩膀，手的位置很重要
- 前手要放在刃的上面，不是外面
- 這是一個很小的提前量，但效果很大
- 上下半身完美結合是關鍵
- 避免反擰和追趕

### 常見錯誤

- 前手放到刃的外面，導致上半身過度打開
- 沒有提前量，換刃時才動手
- 上半身追下半身或下半身追上半身
- 出現反擰
- 板尾甩出去

### 修正方法

- 換刃之前就先把前手放到要換的刃上面
- 確保是刃的上面，不是外面
- 感受這個小提前量的效果
- 觀察上下半身是否完美結合
- 檢查是否還有反擰或追趕

# 182. 蘑菇第二集：過包頂之後"追身"

## 問題描述 (What)

我在滑蘑菇時，根據第一集學會了過包後找側壁轉彎，但當我過包的時候，包會把我頂起來，雖然做了吸收，但身體在包上還是會半頭半尾地晃一下。我不知道過包頂之後應該如何調整重心，導致下一個彎轉不過來。

## 改善目標 (Why)

我需要學會在過包頂之後"追身"。當過了包頂後，包會把我往後掀，我需要人往前追這麼一下。這樣重心就會往板頭方向移動，幫助我進入下一個彎。如果重心壓在後面不追一下，下一個彎是轉不過來的。當我追這麼一下的時候會加速，如果加速了說明做對了。不要太害怕加速，要感受追這一下的感覺。

## 改善方法 (How)

### 步驟拆解

1. **理解包的影響**：
   - 過包時包會把我頂起來
   - 雖然做了吸收，身體還是會晃
   - 包會把我往後掀

2. **過包頂後追身**：
   - 過了包頂之後
   - 人往前追這麼一下
   - 這是關鍵動作

3. **練習方法**：
   - 可以先在橫貫雪道練習
   - 起來追一下，起來追一下
   - 熟悉這個動作

4. **理解效果**：
   - 當追這麼一下時會加速
   - 加速說明做對了
   - 不要太害怕加速
   - 要感受追這一下的感覺

5. **重心移動的重要性**：
   - 重心會往板頭方向移動
   - 幫助進入下一個彎
   - 如果重心壓在後面不追，下一個彎轉不過來

6. **轉彎時機**：
   - 當往前追這麼一下的時候
   - 瞬間去轉彎的上側（側壁）

7. **選擇合適的包**：
   - 不要找很大的包
   - 找小一點的包
   - 道理是一樣的

### 教練提示

- 過包頂後往前追是關鍵
- 追身會加速，這是正常的
- 不要害怕加速，要感受這個動作
- 重心往板頭移動才能進入下一個彎
- 先找小包練習

### 常見錯誤

- 過包頂後沒有往前追
- 重心壓在後面
- 害怕加速而不敢追
- 下一個彎轉不過來
- 一開始就找大包練習

### 修正方法

- 先在橫貫雪道練習追身動作
- 刻意往前追，感受加速
- 克服對加速的恐懼
- 確保重心往板頭移動
- 從小包開始練習

# 183. 粉雪裡的順序：換重心施壓立刃軸轉

## 問題描述 (What)

我在粉雪裡轉彎時，按照CASI的標準順序（先換重心、再立刃、再施壓、最後軸轉），但發現這個順序在深粉雪裡可能不是最流暢的。我不知道如何調整轉彎的順序來適應粉雪的特性。

## 改善目標 (Why)

我需要理解在粉雪裡可以變換轉彎順序。CASI標準是：換重心→立刃→施壓→軸轉。但在深粉雪裡，我可以改成：換重心→直接施壓（不倾倒立刃）→通過壓力被動傾倒→用髖轉彎。這樣做的好處是：彎形更加流暢，不會把速度減到非常低，相當於用壓力去轉彎。由於沒有加入主動傾倒，立刃不明顯，但轉彎更順暢。

## 改善方法 (How)

### 步驟拆解

1. **理解CASI標準順序**：
   - 換重心
   - 立刃（傾倒）
   - 施壓
   - 軸轉

2. **粉雪裡的調整順序**：
   - 換重心
   - 不去傾倒立刃
   - 直接下壓給壓力
   - 通過壓力被動傾倒
   - 用髖去轉

3. **理解調整的好處**：
   - 彎形更加流暢
   - 不會把速度減到非常低
   - 相當於用壓力去轉彎
   - 由於沒有主動傾倒，立刃不明顯

4. **實踐應用**：
   - 在深粉雪裡嘗試這個順序
   - 感受彎形的流暢度
   - 注意速度的保持

### 教練提示

- 粉雪裡可以調整轉彎順序
- 用壓力去轉彎，不是主動立刃
- 被動傾倒比主動傾倒更流暢
- 立刃不明顯是正常的
- 這樣可以保持速度和流暢度

### 常見錯誤

- 在粉雪裡還是用標準順序（主動立刃）
- 把速度減得太低
- 彎形不流暢
- 過度立刃
- 不會用壓力去轉彎

### 修正方法

- 嘗試新順序：換重心→直接施壓→被動傾倒
- 感受用壓力轉彎的感覺
- 不要主動傾倒立刃
- 保持速度和流暢度
- 在深粉雪裡反覆練習

# 184. 淺聊：CASI二級滑行考試

## 問題描述 (What)

我想考CASI二級，知道有四個必考動作（搓雪小彎、刻滑、Ollie、反腳新手轉彎），但不清楚考試的具體要求和評分標準。我可能滑得不錯，視頻也挺好，但考試沒過，不理解原因。我可能對consolidation（穩定性）的要求不夠了解，或者對各個動作的具體標準有誤解。

## 改善目標 (Why)

我需要理解CASI二級考試的關鍵要求：

1. **Consolidation（穩定性）**：在10次裡至少能成功9次，次次基本都能成功。如果發揮不穩定，有時好時壞，考試可能就掛了。

2. **搓雪小彎**：不是彎越小越好，有一定寬度（讀課程指南）。主要考居中靈活站姿、下半身帶轉、在刃上平衡這三點。注意寬度，不要為了轉小而節奏亂。

3. **刻滑**：二級要求與板齊平略微打開一點點（視線看板頭方向），不是高階的完全打開（摸雪摸地）。重心該在前刃時壓前刃，該在後刃時壓後刃，不是上下半身大分離大折疊（那是三級四級）。

4. **Ollie**：關鍵是落地吸收。起跳多高就蹲多低，不要每跳一次被震一下。

5. **反腳新手轉彎**：需要練習。在滾落線之後加入起伏。保持consistent穩定發揮。

## 改善方法 (How)

### 步驟拆解

1. **理解Consolidation**：
   - 10次裡至少成功9次
   - 次次基本都能成功
   - 不能時好時壞
   - 現場穩定發揮很重要

2. **搓雪小彎要點**：
   - 不是彎越小越好，有一定寬度
   - 讀課程指南了解寬度標準
   - 主要考三點：居中靈活站姿、下半身帶轉、在刃上平衡
   - 不要為了轉小而節奏亂
   - 可能會考反角、上下起伏能力

3. **刻滑要點**：
   - 與板齊平略微打開一點點
   - 視線看板頭方向會引領打開一點
   - 不要完全打開（摸雪摸地）
   - 重心該前刃時壓前刃，該後刃時壓後刃
   - 不是上下半身大分離大折疊
   - 規規矩矩做出二級要求

4. **Ollie要點**：
   - 落地吸收是關鍵
   - 起跳多高就蹲多低
   - 不要每跳一次被震一下

5. **反腳新手轉彎要點**：
   - 需要練習
   - 在滾落線之後加入起伏
   - 保持consistent穩定發揮
   - 每10個都能做出來9-10次

### 教練提示

- Consolidation（穩定性）是關鍵
- 不要追求高階動作，做好二級要求
- 搓雪小彎注意寬度，不要太小
- 刻滑不要過度打開或折疊
- Ollie重點是落地吸收
- 反腳需要多練習
- 現場穩定發揮比視頻好看更重要

### 常見錯誤

- 發揮不穩定，時好時壞
- 搓雪小彎為了轉小而節奏亂
- 刻滑過度打開或折疊（做成三四級動作）
- Ollie落地不吸收，每次被震
- 反腳練習不夠
- 追求高階動作而忽略二級要求

### 修正方法

- 反覆練習提高穩定性（10次9次成功）
- 讀課程指南了解具體標準
- 搓雪小彎保持合適寬度和節奏
- 刻滑規規矩矩做二級要求
- Ollie練習落地吸收（起多高蹲多低）
- 反腳多練習，加入起伏
- 上CASI官網看詳細資料和視頻

# 185. 淺聊：你其實轉的不是你的膝蓋，而是…

## 問題描述 (What)

我有個學生認為可以用膝蓋去轉動帶動轉彎，甚至覺得用膝蓋帶動轉彎會更快。我可能也有類似的誤解，認為膝蓋可以扭轉，或者不理解真正的轉動來自哪裡。我可能會感覺膝蓋疼，或者站直轉髖時板子不動，不知道問題出在哪裡。

## 改善目標 (Why)

我需要理解：**不要用膝蓋轉彎**。膝關節沒有辦法出現扭轉，不存在這種運動模式。如果用膝蓋內扣轉動，是靠磨損膝關節、韌帶、軟骨組織來穩定，會導致膝蓋疼。

正確的方式是：**用髖關節去轉動**。如果站直轉髖板子不動或卡刃，是因為需要降低一點、讓膝蓋彎一點，人用板子蹬住雪，再去轉，力量就會大很多，而且比用膝蓋發力更穩定。

有些人講的"膝關節擰轉"其實是**股骨頭的旋轉**（大腿骨的旋轉），不是膝蓋本身。但在單板上（兩腳固定），股骨頭的轉動範圍非常小，可以幫助收緊彎，但用得比較少。主要還是用髖關節轉動，因為膝蓋壓住時髖的轉動力量遠遠大於股骨頭。

## 改善方法 (How)

### 步驟拆解

1. **理解膝蓋不能轉**：
   - 膝關節沒有辦法出現扭轉
   - 不存在這種運動模式
   - 用膝蓋內扣是靠磨損關節、韌帶、軟骨
   - 會導致膝蓋疼

2. **理解人體力學**：
   - 做深蹲時，腿部肌肉想完全發力
   - 內扣沒有辦法讓它發力
   - 內扣的穩定是靠磨損膝關節

3. **用髖關節轉動**：
   - 不要站直轉髖（板子不動或卡刃）
   - 需要降低一點，讓膝蓋彎一點
   - 人用板子蹬住雪
   - 再去轉髖
   - 力量就會大很多
   - 比用膝蓋發力更穩定

4. **理解股骨頭旋轉**：
   - 有些人講的"膝關節擰轉"其實是股骨頭旋轉
   - 不是膝蓋本身在轉
   - 站直時股骨頭轉不了
   - 蹲下來一點可以轉一點，但範圍很小
   - 單板上（兩腳固定）轉動範圍更小
   - 可以幫助收緊彎，但用得比較少

5. **選擇主要方式**：
   - 主要用髖關節轉動
   - 膝蓋壓住時，髖的轉動力量遠大於股骨頭
   - 股骨頭旋轉因人而異，可以作為輔助

### 教練提示

- 膝蓋不能轉，會磨損關節
- 用髖關節轉動是主要方式
- 需要降低一點、膝蓋彎一點、蹬住雪
- 股骨頭旋轉範圍很小，是輔助
- 髖的轉動力量遠大於股骨頭

### 常見錯誤

- 認為可以用膝蓋轉動
- 用膝蓋內扣導致膝蓋疼
- 站直轉髖導致板子不動或卡刃
- 不知道需要降低和彎膝蓋
- 混淆膝蓋轉動和股骨頭旋轉

### 修正方法

- 停止用膝蓋內扣轉動
- 降低重心，膝蓋彎一點
- 用板子蹬住雪
- 用髖關節轉動
- 如果想用股骨頭旋轉，理解它的範圍很小
- 主要還是用髖關節
