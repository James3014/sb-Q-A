## 96 Backside 180（內轉180）落地小技巧

### 問題描述 (What)
內轉180時會忙碌，看向山上讓人害怕，落地後容易繼續旋轉無法停止。

### 改善目標 (Why)
熟悉反腳落地看向山上的感覺，防止落地後繼續旋轉。

### 改善方法 (How)
1. **Riding Fakie 練習**：
   - 反腳在前（例如左腳帶前的人改右腳帶前）
   - 視線往山上看
   - 保持4-6秒
2. **效果**：
   - 熟悉反腳且看向山上的感覺
   - 防止落地後繼續旋轉
3. **漸進練習**：先在雪道上練習Riding Fakie看山上，再應用到內轉180

**核心**：Riding Fakie + 看山上 = 熟悉落地感覺

## 97 進階：CASI三級後如何增加表現力

### 問題描述 (What)
考CASI二級或三級時，在爛雪裡面缺乏表現力。

### 改善目標 (Why)
增加表現力，包括雪的噴濺、彎形的刻度等，通過速度提升來應對更大的衝擊力和離心力。

### 改善方法 (How)
1. **表現力與速度掛鉤**：
   - 速度快 → 衝擊力大 → 離心力大 → 雪噴濺強 → 彎形刻度深
2. **練習方法**：
   - 在原有技術基礎上
   - 極限小彎情況下：迅速三個小彎 → 立刻轉三個更小的彎
   - 重複：三個小的 → 三個更小的 → 三個小的 → 三個更小的
3. **緊急情況下的表現力**：
   - 在很緊急的情況下，表現力會慢慢出來
4. **開放彎加速**：
   - 更小的彎可以是開放彎，讓速度更快
   - 加速後回到正常小彎，需要應對更大阻力
   - 慢慢適應把速度加起來

**核心**：小彎 → 更小彎 → 開放彎加速 → 應對大阻力 = 表現力提升

## 98 談談CASI三級滑行

### 問題描述 (What)
不清楚CASI三級滑行的評判標準、為什麼不同考場考官標準似乎不一樣、如何朝這個方向努力練習。

### 改善目標 (Why)
理解CASI三級滑行的正確認知，包括四項考核內容、四個高階核心能力評判標準、以及針對性練習建議。

### 改善方法 (How)
**四項考核**：
1. **大彎**（7-10米）：藍道，雪況好要刻滑走線，雪況不好走不出線也正常
2. **小彎**（3-4米）：黑道，不要轉太小
3. **中彎**（4-6米）：與小彎同一道，中等速度/表現力（60-70%），不要太多表現力
4. **蘑菇**（3-6米）：黑道（雪況差可能藍道），看穩定發揮、地形控制力、使用地形多樣性、隨機應變能力

**四個高階核心能力**（評判標準）：
1. **流暢度與力量感**：不能像推坡一樣滑
2. **弧到弧**：左右圓對稱，早立刃，立刃時間一樣
3. **蓄能與傳導**（Loading Deflection）：蓄力後利用板子和身體力量進入下一個彎
4. **轉向多功能性**（Steering Versatility）：對雪板控制力，各種地形隨機應變

**為什麼標準看似不一樣**：
- 考場雪況坡度不同，但評判標準（四個高階核心能力）相同
- 考官通過這四個標準審視滑行，不會因雪道不同提升難度

**練習建議**：
1. **所有項目必須有上半弧**：大彎、中彎、小彎、蘑菇都要有，上半弧有時間減速，彎形圓潤對稱
2. **刻滑閉合彎形**：不能開放，坡變緩可稍微開放但速度不減
3. **蘑菇策略**：
   - 看雪道長度調整策略
   - 離考官遠：過3個包再轉，求穩
   - 離考官近：一個一個轉，展現表現力
4. **中彎關鍵**：與小彎同道但速度慢、力量少、發力不猛，展現正反腳控制能力
5. **個人技巧**：中彎時固定器鬆一到兩格，讓板子反應慢一點

**考官建議**：
- 盡量滑得偏向考官風格
- 當場問考官示範中看到的東西是否考試需要

## 99 這三步：幫你由平到陡控速和方向

### 問題描述 (What)
在綠道轉得很順，但到藍道或稍陡的坡無法控制速度，轉得不順，無法控制方向躲人。

### 改善目標 (Why)
從綠道到藍道幫助控速和方向控制，讓彎變小，提高對速度、方向、彎形的控制。

### 改善方法 (How)
**第一步：原地練習**
1. 前刃後刃上下蹲
2. 蹲到大腿小腿接近90度，感覺腿酸
3. 前刃：膝蓋朝兩個固定器下去，朝板子中央下降（不建議屁股出板）
4. 後刃：同樣下蹲

**第二步：綠道滾落線後練習**
1. 找平時滑得順的綠道（不要去藍道陡坡）
2. 在滾落線之後蹲
3. 原因：這個位置蹲了加減速，速度很慢，就算摔也不疼（蹲得越低離地越近）

**第三步：綠道滾落線前練習**
1. 在滾落線之前一點點做
2. 可以把動作延長到下面
3. **關鍵**：在滾落線或之後要起來
4. 原因：不起來轉下一個彎時沒有range無法再蹲

**效果**：
- 蹬雪產生摩擦力和抓地力
- 幫助控速
- 起伏變大時彎變小
- 方向、彎形、速度控制都提高

**注意**：平的道不需要那麼大起伏

## 100 這個上下反擰得解決

### 問題描述 (What)
速度變快或坡變陡時，前刃會出現上下半身反擰，坡越陡速度越快越明顯。

### 改善目標 (Why)
讓人與板齊平，對速度和彎形控制越來越好。

### 改善方法 (How)
1. **兩個動作同時做**：
   - **動作一：後手往後拉**
     - 以髖關節為軸
     - 後手往後拉時，髖會盡量與板齊平
   - **動作二：屁股夾起**
     - 兩個屁股蛋子夾起
     - 身體與板齊平
2. **時機**：後刃換前刃的時候
3. **視線**：還是看著前面，不用管其他事情
4. **避免錯誤**：不要只拉手導致身體扭曲

**核心**：後手往後拉 + 屁股夾起 = 人與板齊平

## 101 換刃之後：按住後腿

### 問題描述 (What)
後腳介入時間太慢，在滾落線之後才介入，在包裡面上包頂後一轉，後腳板尾直接甩到滾落線之後。

### 改善目標 (Why)
讓後腳盡早介入彎形，在滾落線之前就介入。

### 改善方法 (How)
1. **練習方法**：
   - 從開始滑，換刃後一直壓著後腳滑
   - 主要感受後腳介入的區域和感覺
   - 之後再恢復原狀，讓後腳在正確位置介入
2. **位置對比**：
   - 錯誤：後腳在滾落線之後才介入
   - 正確：後腳在滾落線之前就介入
3. **感受**：
   - 壓後腳時後腿會感覺比較累（這是對的感覺）
   - 感受到後腿後再回到正常滑行

**核心**：換刃後壓後腳練習 → 感受後腿介入 → 應用到正常滑行

## 102 換刃之後重心壓的板位置不一樣

### 問題描述 (What)
前刃非常自如，重心轉換和後腳運用收彎做得好，但換到後刃時會先壓前腿，後腿完全不介入，導致後腿很直。

### 改善目標 (Why)
讓後刃時也能利用整段板子滑行，而非只壓前面，實現前後刃平衡。

### 改善方法 (How)
1. **腰子被捅一拳的感覺**：
   - 從前刃換到後刃時
   - 感覺腰子被捅了一拳
2. **兩種情況同時發生**：
   - **第一**：屁股往後一點點
   - **第二**：肩膀往板頭探一點
3. **效果**：
   - 出現「不自然的平衡」
   - 在後刃時感覺利用整段板子在滑
   - 而非壓在前面
4. **站姿**：
   - 人會稍微有點開放
   - 但不要通過刻意開放去找這種站姿
   - 通過「腰子被捅」的感覺來找

**核心**：腰子被捅感覺 = 屁股後 + 肩探前 = 後刃用整段板
