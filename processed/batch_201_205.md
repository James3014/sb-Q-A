# 201. 蘑菇裡先換重心？還是先換刃

## 問題描述 (What)

我在滑包時，過完包頂或過包最高點之後才換重心，相當於從那裡出溜下去，沒有給到任何壓力，速度無法控制。一旦過了包頂，從那裡到下一個包的位置感覺在加速。

## 改善目標 (Why)

我需要理解：在**上包的過程中**，還沒有過包頂時，重心就要逐漸已經換過來了。這樣當過下半邊時，可以通過重心在該在的位置去對板施壓、去推雪，這樣就可以減速。

關鍵點是時機：在上包時強制要求自己站起來，把重心換過來，然後下半段就可以下降去推雪，通過把速度控制好進入下一個包，繼續再轉。

## 改善方法 (How)

### 步驟拆解

1. **識別錯誤模式**：
   - 過完包頂之後才換重心
   - 從那裡出溜下去
   - 沒有給到任何壓力
   - 速度無法控制、在加速

2. **理解正確時機**：
   - 在上包的過程中
   - 還沒有過包頂
   - 重心就要逐漸已經換過來了

3. **理解原因**：
   - 當過下半邊時
   - 可以通過重心在該在的位置
   - 去對板施壓、去推雪
   - 這樣就可以減速

4. **關鍵動作**：
   - 在上包時強制要求自己站起來
   - 把重心換過來
   - 這是關鍵點

5. **下半段動作**：
   - 重心換過來後
   - 下半段可以下降去推雪
   - 通過把速度控制好

6. **進入下一個包**：
   - 速度控制好後
   - 進入下一個包
   - 繼續再轉

### 教練提示

- 上包時換重心，不是過包頂後
- 還沒過包頂時重心就要換過來
- 這樣下半段才能施壓推雪減速
- 在上包時強制站起來換重心
- 速度控制好才能順利進入下一個包

### 常見錯誤

- 過包頂後才換重心
- 出溜下去沒有壓力
- 速度無法控制、一直加速
- 不知道正確時機
- 沒有在上包時站起來

### 修正方法

- 刻意在上包時換重心
- 還沒過包頂就換
- 強制自己站起來
- 下半段下降推雪
- 感受速度控制的效果

# 202. 蘑菇裡後換前：三步腳踝練習

## 問題描述 (What)

我在滑包蘑菇時，後刃換前刃是大部分人的痛點。當從包的高點往下換刃，會墜落到低點，腳踝跟膝蓋會受到很大的落差和衝擊力。我不知道如何訓練腳踝來應對這種情況。

## 改善目標 (Why)

我需要通過三步腳踝練習來訓練。可以在戶外找樓梯練習，訓練腳踝去感受從包的高點到低點的感覺。

**三步練習**：
1. **雙腳扶著跳**：背朝下扶著樓梯，一腳抬起，另一腳蹬然後跳到下一階，讓腳踝充分閉合（不是打開）
2. **單腳扶著跳**：單腳往下跳，還是得拉著，讓腳踝得到充分關閉
3. **雙腳一起跳**：兩腳同時往下跳，稍微拉著一點

關鍵是落地時前腳掌著地（三分之一或一半），不是整個腳掌。這種感覺跟從包上下來落到前刃的感覺一模一樣。做的時候盡量讓髖關節不要撅屁股撅太多，髖關節正常壓在整條膝蓋跟腳尖的位置，垂直下去。

## 改善方法 (How)

### 步驟拆解

1. **找到練習場地**：
   - 找樓梯
   - 可以扶著的樓梯
   - 背朝下面

2. **第一步：雙腳扶著跳**：
   - 扶著樓梯
   - 一腳抬起來
   - 另一腳蹬然後跳到下一階
   - 讓腳踝得到充分閉合
   - 閉合是腳踝往前，打開是腳踝往後

3. **第二步：單腳扶著跳**：
   - 單腳去做
   - 還是得拉著點，不然會摔下去
   - 除非樓梯只有一截可以不拉
   - 單腳往下跳
   - 讓腳踝得到充分關閉

4. **第三步：雙腳一起跳**：
   - 兩個腳一起
   - 稍微拉著一點
   - 兩腳同時往下跳

5. **落地位置**：
   - 不是整個腳掌落地
   - 前腳掌著地
   - 大概三分之一或一半
   - 一半比較簡單，三分之一稍微難一點
   - 這種感覺跟從包上下來落到前刃一模一樣

6. **髖關節位置**：
   - 盡量不要撅屁股撅太多
   - 髖關節正常壓在整條膝蓋跟腳尖的位置
   - 垂直下去

### 教練提示

- 後換前是大部分人的痛點
- 可以在樓梯上練習腳踝
- 三步練習：雙腳扶著→單腳扶著→雙腳一起
- 腳踝要閉合不是打開
- 前腳掌著地，不是整個腳掌
- 髖關節不要撅屁股太多

### 常見錯誤

- 不知道如何訓練腳踝
- 整個腳掌落地
- 腳踝打開而不是閉合
- 髖關節撅屁股太多
- 沒有循序漸進（直接雙腳一起）

### 修正方法

- 找樓梯練習三步
- 確保腳踝閉合
- 前腳掌著地（三分之一或一半）
- 控制髖關節位置
- 循序漸進：雙腳扶著→單腳扶著→雙腳一起

# 203. 蘑菇裡：後刃換前刃

## 問題描述 (What)

我在包裡面後刃換前刃時不太敢，感覺要直接往下衝的那種感覺，特別恐懼。我不知道如何安全地完成後換前。

## 改善目標 (Why)

我需要練習一個關鍵動作：後刃彎結束時，後腿、重心壓在後腿上，壓一下後腿把板子送一下（像Ollie一樣把板頭拉起來）。然後當進入前刃時，**不是壓在刃上，而是壓在板底上面**。這是進入包能穩定的關鍵點之一。

如果在包裡後刃彎結束時產生回彈，它會送你進入前刃。但不能想著進入前刃，因為一旦身體想著去進入前刃，身體會往前刃傾倒，就直接拍過去了。所以最安全的是進入板底。這需要很大的勇氣去壓住板底，因為壓的方向還是朝前刃一點點，只不過思想變了，朝的是前刃的板底而不是前刃。

當會做這個時，會感覺重心換前刃時重心到這就戛然而止了，而不是太往前往前滑，這樣就不會失去重心，能順利換到前刃。這需要commitment，不能怂、不能害怕。當後刃產生回彈時，這就是最佳換前刃時機，不要猶豫直接換。

## 改善方法 (How)

### 步驟拆解

1. **理解恐懼**：
   - 後換前感覺要直接往下衝
   - 特別恐懼
   - 這是可以通過訓練克服的

2. **後刃彎結束動作**：
   - 重心壓在後腿上
   - 壓一下後腿
   - 把板子送一下
   - 像Ollie一樣把板頭拉起來

3. **進入前刃的關鍵**：
   - 不是壓在刃上
   - 而是壓在板底上面
   - 這是關鍵點之一

4. **理解回彈**：
   - 後刃彎結束產生回彈
   - 會送你進入前刃
   - 但不能想著進入前刃

5. **避免傾倒**：
   - 一旦身體想著去進入前刃
   - 身體會往前刃傾倒
   - 就直接拍過去了

6. **最安全方式**：
   - 進入板底
   - 需要很大勇氣
   - 壓的方向還是朝前刃一點點
   - 但思想是朝前刃的板底

7. **效果**：
   - 重心換前刃時到這就戛然而止
   - 不是太往前往前滑
   - 不會失去重心
   - 能順利換到前刃

8. **需要commitment**：
   - 不能怂、不能害怕
   - 後刃產生回彈時是最佳時機
   - 不要猶豫直接換

9. **熟練後的感覺**：
   - 一旦會了會覺得非常簡單

### 教練提示

- 後換前需要勇氣和commitment
- 關鍵是壓板底不是壓刃
- 後刃回彈是最佳換刃時機
- 不要猶豫，直接換
- 思想要朝板底不是朝刃
- 會了之後會覺得很簡單

### 常見錯誤

- 太害怕不敢換
- 想著進入前刃導致傾倒
- 壓在刃上而不是板底
- 猶豫錯過最佳時機
- 沒有commitment

### 修正方法

- 先在簡單的包裡練習
- 刻意壓板底不是刃
- 後刃回彈時不要猶豫
- 建立commitment和勇氣
- 反覆練習直到不害怕

# 204. 彎裡的板頭板尾移動

## 問題描述 (What)

我已經學會了壓前腳和壓後腳，但壓前腳會感覺很鑽，壓後腳有時會感覺有點突突。我不知道如何在彎裡進行重心的板頭板尾移動來改善這個問題。

## 改善目標 (Why)

我需要學習在彎裡面進行重心轉移。在入彎時重心稍微多往板頭一點，讓前腳感覺重一點（大概在滾落線之前的位置）。滾落線的時候可以讓重心稍微往板尾移動，在板尾多一點。這樣可以幫助把彎閉合起來。

練習方法：
1. **原地練習**：板頭平衡與板尾平衡
2. **壓板頭轉彎**：一直壓著板頭轉彎，感覺前腿比較酸，全程不能回到中間或板尾（尤其是滾落線跟滾落線之後）
3. **壓板尾轉彎**：一直壓著板尾轉彎，感覺後腿酸
4. **整合到彎裡**：入彎時感覺前腿酸一點，出彎或過滾落線時感覺後腿酸一點

重心移動方式：一種是主動移過去，第二種是通過彎一條腿的膝蓋與腳踝把重心拉過去。這可以讓滑行更放鬆、更vibrant（生動）、更完美。

## 改善方法 (How)

### 步驟拆解

1. **原地練習平衡**：
   - 板頭平衡
   - 板尾平衡
   - 熟悉兩種感覺

2. **練習壓板頭轉彎**：
   - 一直壓著板頭轉彎
   - 感覺前腿比較酸
   - 全程不能回到中間或板尾
   - 尤其是滾落線跟滾落線之後
   - 否則練習毫無意義

3. **練習壓板尾轉彎**：
   - 一直壓著板尾轉彎
   - 感覺後腿酸
   - 全程保持

4. **理解彎裡的應用**：
   - 入彎：重心往板頭、前腳重
   - 滾落線：重心往板尾
   - 這樣可以閉合彎

5. **兩種移動方式**：
   - 方式一：主動移重心過去
   - 方式二：通過彎一條腿的膝蓋與腳踝把重心拉過去
   - 都可以

6. **整合到彎裡**：
   - 入彎時感覺前腿酸一點
   - 出彎或過滾落線時感覺後腿酸一點
   - 把練習的感覺帶入

7. **效果**：
   - 讓滑行更放鬆
   - 更vibrant（生動）
   - 更完美

### 教練提示

- 已經會壓前後腳是基礎
- 原地練習板頭板尾平衡
- 壓板頭全程不能回中間或板尾
- 壓板尾全程保持
- 入彎板頭、出彎板尾
- 兩種移動方式都可以

### 常見錯誤

- 壓板頭時中途回到中間或板尾
- 壓板尾時沒有全程保持
- 不知道在彎裡如何應用
- 只會一種移動方式
- 沒有感覺到前後腿的酸

### 修正方法

- 原地先練習平衡
- 壓板頭板尾時全程保持
- 感受前後腿的酸
- 兩種移動方式都嘗試
- 把感覺整合到彎裡

我在滑包蘑菇時，不知道如何在包頂進行換刃。包頂換刃是比較簡單的方式，但我不清楚具體怎麼做。

當送這麼一下時，好處是：它能在包頂的時候釋放重心、釋放板刃，讓我在包頂時可以把重心進行換過來，也就是換刃。

如果會Ollie的話最好，可以在彎結束的地方（從上面往下轉的這個位置）去做一個Ollie。這其實就是往包頂去送的一種感覺。當一做Ollie時會感覺人比較輕，這就是在包頂這個位置會出現的感覺。

## 改善方法 (How)

   - 是滑包的基礎

   - 快上包頂之前
   - 把板子往包上面去送
   - 由板尾到板頭這個方向
   - 送這麼一下

   - 從上面往下轉的這個位置
   - 這就是往包頂去送的感覺

5. **感受效果**：
   - 一做Ollie會感覺人比較輕
   - 這就是在包頂會出現的感覺

6. **觀看示範**：
   - 理解送板的動作
   - 感受包頂換刃的流暢

### 教練提示

- 包頂換刃是比較簡單的方式
- 送板動作：由板尾到板頭
- 可以釋放重心和板刃
- 會Ollie的話可以在彎結束做
- 感覺人比較輕是對的

### 常見錯誤

- 不知道如何在包頂換刃
- 沒有送板動作
- 送板方向錯誤
- 不會利用Ollie
- 沒有感覺到人變輕

### 修正方法

- 練習送板動作：板尾到板頭
- 在快上包頂前送板
- 如果會Ollie在彎結束做
- 感受重心和板刃的釋放
- 感受人變輕的感覺
