# 16 以前腿為軸轉彎：得改

## 📝 教學內容

### 問題描述 (What)
前腿很直，後腿會以前腿為軸，去兩邊掃雪刮雪。板尾沒有跟著板頭，而是以板頭為軸轉動。

### 改善目標 (Why)
- 讓板尾跟隨板頭
- 讓彎轉得更順
- 避免以前腿為軸轉動

### 改善方法 (How)

**先彎前腿技巧**：
- 當想發起轉彎時，先讓前腿彎一點
- 板子就不會以前腿為軸了
- 板子會順著轉，板尾也就更容易跟隨板頭

**關鍵提示**：
- 換刃的時候，先彎前腿
- 彎就會越來越好

# 17 動作幅度過大會造成不穩

## 📝 教學內容

### 問題描述 (What)
前刃後刃時，髖關節（核心部位）移動的幅度非常大。當髖關節移動特別大時，會讓刃卡在雪裡特別明顯，板子的立刃角度會被適當拉起來，髖關節出板太多會造成不穩定，尤其在較硬或較冰的雪面上。

### 改善目標 (Why)
- 減少髖關節出板幅度，提升穩定性
- 讓髖關節維持在板寬之內
- 多利用板底搓雪，而不是過度立刃
- 在冰面或硬雪上更穩定

### 改善方法 (How)

**髖關節控制在板寬內技巧**：

1. **減少前後移動**：
   - 讓髖關節少在板子外面動
   - 讓髖關節在板子的板寬之內去做動作

2. **增加上下，減少前後**：
   - 多增加上下動作
   - 不是多增加前刃後刃方向的移動

3. **壓力在足弓之內**：
   - 盡量感覺力量的傳遞在足弓之內
   - 通過膝蓋與腳踝去彎曲調節
   - 訓練這種能力

**效果**：
- 搓雪會特別穩定
- 不會感覺幅度那麼大特別費力
- 髖關節基本維持在板腰寬度之內

**關鍵提示**：
- 理想雪況只占 10%，剩下 90% 的雪不一定理想
- 利用板底多去搓，可以更加穩定

# 18 後刃的"King"

## 📝 教學內容

### 問題描述 (What)
重心永遠留在後刃，尤其是換刃的時候。髖部沒有領出來，核心沒有緊。

### 改善目標 (Why)
- 讓髖部能夠領出來
- 讓核心緊起來
- 避免重心永遠留在後刃

### 改善方法 (How)

**屁股夾緊頂髖技巧**：

1. **跪下練習**：
   - 如果找不到髖領出來的感覺，先跪下
   - 把兩個屁股蛋子夾緊
   - 髖就頂出來了，核心也就緊了

2. **站起來保持**：
   - 屁股夾緊後，保持這種感覺站起來
   - 再找這個感覺

3. **推坡練習**：
   - 站起來找到感覺後，推坡找這個感覺
   - 可以直著推或斜著推

4. **換刃應用**：
   - 推完坡後，換刃時找這種感覺
   - 在彎開始的地方，從後刃換到前刃
   - 換到平板過渡到前刃的一瞬間，就得做出這個姿勢

**關鍵提示**：
- 屁股夾緊是關鍵
- 在換刃的一瞬間就要做出髖領出來的姿勢

# 19 "樓梯"包：別使勁給壓力

## 📝 教學內容

### 問題描述 (What)
在一道一道樓梯狀的蘑菇包裡，如果每個挫疼都使勁給壓力，身體會被彈起來，因為吸收線太多不讓它移動。

### 改善目標 (Why)
- 避免身體被彈起來
- 讓滑行更流暢穩定
- 在樓梯包裡保持控制

### 改善方法 (How)

**放鬆 + 選擇性施壓技巧**：

1. **關節放鬆**：
   - 滑的時候不會使勁給那個位置壓力
   - 讓關節放鬆
   - 像落葉飄一樣，但這次的落葉飄會盡量放鬆

2. **選擇施壓位置**：
   - 不要在一道一道樓梯的位置使勁推
   - 等到下面比較大一點的包，或者比較細膩堆多一點的地方再給壓力
   - 一旦在樓梯位置使勁推，身體會被彈起來

3. **放鬆通過小包**：
   - 所有的挫疼直接就放鬆了，不去使勁給壓力
   - 等到一個比較大的包再給壓力

**關鍵提示**：
- 如果每次往下推都使勁，肯定有一個會把你彈起來
- 放鬆通過小包，在大包處施壓

# 20 5步教會你：前刃摸地刻滑

## 📝 教學內容

### 問題描述 (What)
想學會前刃貼地刻滑（Euro Carve），但不知道如何練習。

### 改善目標 (Why)
- 學會前刃貼地刻滑
- 能夠在滾落線做出動作並維持速度
- 掌握完整的刻滑技巧

### 改善方法 (How)

**前刃貼地刻滑五步驟**：

**第一步：學會傾倒**
- 單純往前刃傾倒，帶著速度
- 不用傾倒太多，一點點就夠
- 當刃咬在雪裡咬住時，進入下一步

**第二步：傾倒後下降**
- 當傾倒壓住刃後，朝前刃下降下去
- 下降方向是板子的正中央（不是往前頂，否則會酥板）
- 通過膝蓋、腳踝、髖的彎曲下降
- 可以彎腰撅屁股，這是刻滑需要做到的一部分

**第三步：用手摸雪**
- 下降之後，用兩個手嘗試去摸雪
- 摸的位置會離前刃非常近
- 確保刃還是咬在雪裡，不能掃雪
- 如果掃了，退回第二步繼續練

**第四步：延伸腿推出去**
- 在滾落線之後摸到雪後，延伸腿把腿蹬出去
- 摸著雪的同時，當蹬出去時腿不會去掃雪，手會被推出去
- 在滾落線之後做是為了安全（蹬出去後會橫貫雪道減速）
- 記得回頭看後面有沒有人

**第五步：拉回到滾落線位置**
- 把剛才所做的動作拉回到滾落線位置
- 追下去的時候就在滾落線蹲了，然後在這就開始拉出去
- 滾落線是能維持速度最快的地方
- 在這做出去後，在這就可以站起來了
- 否則在滾落線之後拉出去，速度減少太多起不來了

**關鍵提示**：
- 這不是 3 天 5 天就能練熟的，需要時間練習
- 按照五個步驟加以實踐，至少能達到基本水平
