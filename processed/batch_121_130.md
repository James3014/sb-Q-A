## 121 換前刃小技巧：別朝著前刃發力

### 問題描述 (What)
陡坡上後刃換前刃時會比較害怕，感覺身體在往山下去趴、往山下去撲，有的人因為害怕真的就往山下撲了，容易摔倒。

### 改善目標 (Why)
理解後刃換前刃的正確發力方向，換到前刃後會比較穩定，不會往山下撲摔倒。

### 改善方法 (How)
**錯誤認知**：
- 後換前時人往山下撲
- 因為害怕真的就往山下撲了
- 容易摔倒

**正確認知**：
- 後換前時跟想像的可能有點不太一樣
- **人不是往山下撲**
- **人在後刃起來之後是往行進方向去壓的**

**具體做法**：
1. **後刃先回到一個平板**
2. **朝著板頭行進方向去壓過去**
3. **慢慢借助往板頭壓的壓力**
4. **慢慢再往腳尖的方向傳遞**
5. **這樣就不會一下子撲出去**

**側面觀察**：
- 從側面能看得清楚
- 是往行進方向壓
- 而不是往山下撲

**關鍵時機**：
- 最關鍵的部分就是換刃的一下
- 從後刃換前刃的一下
- 換到前刃之後就會比較穩定

**適用場景**：
- 比較基礎的技術
- 在比較陡一點的位置
- 想進階去後刃換前刃

**核心**：往行進方向壓 ≠ 往山下撲

## 122 前刃過度頂胯：用膝蓋解決

### 問題描述 (What)
前刃轉彎之後會過度頂胯，大部分都是頂胯過度導致前腿頂到前隊，前腿很酸，對膝蓋造成很大壓力，有種要撅出去或壓出去的感覺。

### 改善目標 (Why)
讓髖盡量居中一點，可以雙腳承重，不會有撅出去或壓出去的感覺，前腿不會那麼酸，不會對膝蓋造成很大壓力。

### 改善方法 (How)
**問題分析**：
- 過度頂胯
- 頂不到的是前隊
- 大部分都是頂胯過度前腿頂到前隊

**解決方法**：
1. **後膝蓋往板尾方向打開**
2. **重心就會拉扯到中間**
3. **不要去想著前膝蓋**
4. **從後刃轉前刃的時候**
5. **讓後膝蓋往彎圍打開**
6. **身體重心就會居中**

**效果**：
- 髖居中
- 雙腳承重
- 不會撅出去或壓出去
- 前腿不酸
- 膝蓋壓力小

**核心動作**：
後膝蓋往板尾/彎圍打開 = 重心拉到中間 = 雙腳承重

## 123 前手讓後刃帶轉更順

### 問題描述 (What)
後刃時上半身可能會阻擋下半身的行進，手還是放在前面與板保持齊平，轉的越多會出現反擰，無法再增加或保持立刃，立刃會慢慢減少，板子越來越往下掉，無法有效施壓。

### 改善目標 (Why)
讓髖跟肩主動保持打開狀態，板子可以一直同向旋轉，效率更高，彎形更加圓潤，不會感覺人鎖死。

### 改善方法 (How)
**問題分析**：
- 換到後刃時，如果手還是放在前面
- 與板保持完全齊平
- 轉的越多會出現反擰

**反擰的壞處**：
1. 無法再增加或保持立刃
2. 立刃會慢慢減少
3. 板子越來越往下掉
4. 無法有效施壓

**解決方法**：
1. **前轉後之後**
2. **髖跟肩應該主動保持打開狀態**
3. **衡量方法：把前手放在後刃的正上方**
4. **肩跟髖保持打開**
5. **轉的時候板子可以一直同向旋轉**

**效果**：
- 效率更高
- 彎形更加圓潤
- 不會感覺人鎖死的狀態

**核心**：
前手放後刃正上方 = 肩髖打開 = 板子同向旋轉 = 效率高彎形圓潤

## 124 換刃時前手別「烙餅」🥞

### 問題描述 (What)
轉後刃轉前刃的時候會「烙餅」，因為後刃還沒有釋放出來就太想去轉前刃了，從後刃直接想往前刃換，尤其在道外、粉雪或包裡面會出現這種烙餅行為。

### 改善目標 (Why)
停止烙餅行為，讓換刃更加流暢耐心，不會出現烙餅動作。

### 改善方法 (How)
**問題原因**：
- 後刃還沒有釋放出來
- 太想去轉前刃了
- 從後刃直接想往前刃換
- 就會出現烙餅行為

**解決方法**：
1. **多耐心一點**
2. **後刃換前刃的時候**
3. **先讓髖回到板子的正中央**
4. **再把髖過渡過去**
5. **由後刃到前刃**
6. **這樣就不會烙餅**

**核心順序**：
後刃 → 髖回到板子正中央 → 髖過渡過去 → 前刃

**關鍵**：
耐心 + 髖先回中央再過渡 = 不烙餅

## 125 前刃：彎末增加抓地力

### 問題描述 (What)
前刃彎結束的時候身體處於彎打開、視線看向山下的狀態，前腳很鬆，轉到後刃時板子直接會往下掉，抓地力會減少很多。

### 改善目標 (Why)
在彎結束的時候雙腳支撐，回到比較基礎的站姿，舌頭這邊產生壓力，人跟板齊平，雙腳推旋，在彎末時雙腳發力，進入下一個彎時有足夠力量和時間準備。

### 改善方法 (How)
**問題狀態**：
- 前刃彎結束時彎是打開的
- 視線看向山下
- 前腳很鬆
- 轉到後刃板子直接往下掉
- 抓地力減少很多

**解決方法**：
1. **彎結束的時候加一點髖關節的旋轉**
2. **讓前髖或肩或手帶到前刃這個位置**
3. **當髖一合上跟板齊平的時候**
4. **在彎結束的時候是兩個腳去支撐**
5. **回到比較基礎的站姿**

**具體做法**：
- 讓手放到前刃
- 後手放到後刃
- 會感覺舌頭這邊產生了壓力
- 人跟板齊平
- 雙腳推旋

**練習建議**：
- 特意讓彎多轉一點
- 確保在彎末的時候雙腳發力
- 進入下一個彎時有足夠力量和時間準備

**核心**：
彎末加髖旋轉 + 髖合上與板齊平 = 雙腳支撐 + 抓地力

## 126 進階：前腳的蓄力與傳導

### 問題描述 (What)
沒有上半弧，或者蹬地推雪這一下沒有辦法傳遞出來能量進入下一個彎，原因是前腳沒有去首先推住雪、壓住雪。

### 改善目標 (Why)
讓前腳在上半段產生抓地力，出現蓄力，在下半段有能量把人送過去進入下一個彎，可以讓滑行越來越輕鬆。

### 改善方法 (How)
**問題原因**：
- 前腳沒有去首先推住雪壓住雪
- 導致沒有上半弧
- 蹬地推雪無法傳遞能量進入下一個彎

**解決方法**：
1. **在彎形上半段的時候**
2. **需要前腳感覺到外側**
3. **通過膝蓋的彎曲感覺到外側**
4. **說明前腳在那個位置已經產生抓地力**

**具體做法**：
- **讓膝蓋引向板頭方向**
- 每當開始入彎的時候這樣做
- 蓄力就出現了

**效果**：
- 在下半段進入下一個彎的時候
- 會有一種能量把人送過去
- 可以讓滑行越來越輕鬆

**核心**：
膝蓋引向板頭 → 前腳外側感覺 → 上半段抓地力 → 蓄力 → 下半段能量送過去

## 127 前刃進階過程：100%會出現的問題

### 問題描述 (What)
前刃結束時肩與板平行，但髖還朝山下打開，只是照貓畫虎用肩帶，起不到很快反過來的作用比較慢，所有力量都承重在後腿上面，前腿舌頭沒有任何壓力。

### 改善目標 (Why)
讓髖閉合與板齊平，前腿舌頭受到跟後腿舌頭等同的壓力，視線裡有前肩膀，雙腿均勻承重。

### 改善方法 (How)
**問題分析**：
- 看起來肩與板平行
- 但髖還朝山下打開
- 只是照貓畫虎
- 用肩帶起不到很快反過來的作用

**為什麼不好**：
- 當把前髖打開的時候
- 板子前面部位沒有壓在前腿舌頭上
- 注定沒有任何壓力
- 所有力量都承重在後腿上

**正確做法**：
1. **後換前的時候把髖壓過來**
2. **當這樣做的時候才是與板齊平**
3. **手是自然下垂的**
4. **不需要去做多餘動作**

**檢查方法**：
- 一旦把髖的位置擺正
- 會感覺前腿舌頭這地方受到跟後腿舌頭等同的壓力
- 視線裡面是有前肩膀的
- 而不是沒有前肩膀

**核心**：
髖壓過來與板齊平 ≠ 只是肩與板齊平

## 128 前腿膝蓋疼，看看是不是這個原因

### 問題描述 (What)
滑行流暢但非常喜歡把重量壓在前腳居多，不管在彎行的任何位置都比較在前腳，導致前腿累、前膝蓋非常疼甚至酸痛，長期可能前膝蓋或腳踝（主要是膝蓋）會受傷。

### 改善目標 (Why)
讓重心在彎行過程中由前面到後面兩腿之間進行轉換，雙腿均勻承重，前腿不累不疼，滑行更加鮮活。

### 改善方法 (How)
**問題分析**：
- 每次把重心多壓前腿居多
- 前腿會累
- 前膝蓋非常疼甚至酸痛
- 長期可能膝蓋或腳踝受傷

**正確認知**：
- 重心在彎行過程中是由前面到後面
- 兩腿之間進行轉換
- 雙腿均勻承重

**重心轉移時機**：
1. **換完刃之後**：重心可以偏前一點點（沒問題）
2. **在滾落線或之前**：重心從前腿開始慢慢往後移動
3. **為什麼不是滾落線之後**：那個位置屬於補救，來不及了，速度太快

**後腿介入的重要性**：
- 讓後腿稍微介入
- 重心往後面一點點
- 彎在這就可以 close up（轉彎）
- 要不然彎順著就下去了（推坡）

**壓後腿練習**：
1. **全程從開始壓著後腿**
2. **前刃壓後腳感覺**：
   - 鞋的外側小拇指下面被擠壓到
   - 下降
   - 膝蓋稍微往外側拉一點
   - 感覺特別明顯
3. **後刃壓後腳感覺**：
   - 大腿朝固定器方向下壓
   - 比較居中
   - 不用太靠後面

**效果**：
- 前腿非常輕鬆一點也不累
- 後腿相對比較酸
- 腳的外側被壓得有點酸（應該有的感覺）
- 彎形在滑的過程中身體在前後之間重心轉移
- 滑得更加鮮活（livery）

**漸進練習**：
1. 先練全程壓後腿
2. 再加入先感覺前腳再感覺後腳
3. 在彎形當中轉換

## 129 前刃減速過多會導致彎形不對稱

### 問題描述 (What)
前刃的壓力會給在彎的最下面位置，在這個位置會有很大的噴雪，這種做法會很影響彎與彎之間連接的節奏，是必經之路但需要改善。

### 改善目標 (Why)
讓後腿在上半段介入，雪的噴濺在上半弧跟滾落線完成往外噴，不會在最下面往外噴，幫助前刃到後刃的連接緊密，不會產生卡頓。

### 改善方法 (How)
**問題分析**：
- 壓力給在彎的最下面位置
- 在這個位置有很大噴雪
- 影響彎與彎之間連接的節奏

**解決方法**：
1. **從後刃換到前刃的時候**
2. **在上半段的時候**
3. **讓後腳、後腿盡量介入得早一點**

**如何介入**：
- 使用 Cowboy Knee（牛仔膝蓋）
- 讓膝蓋稍微朝著板尾的方向
- 或固定器的方向
- 稍微下降一點
- 讓後腿在這個位置產生抓地力

**效果**：
- 板尾那一部分不會在最下面介入
- 而是在上半段介入
- 雪的噴濺在上半弧跟滾落線完成往外噴
- 不會在最下面往外噴
- 幫助前刃到後刃的連接緊密
- 不會在最下面產生卡頓

## 130 前刃降重心

### 問題描述 (What)
後刃可以下蹲給板施壓或立刃，但前刃只能到某個位置，很多人為了降低重心而彎腰，但彎腰只是腦袋離雪近了，屁股還是那麼高，沒區別，無法真正增加壓力。

### 改善目標 (Why)
在前刃上真正降低重心給板增加壓力，多推雪，保持背是直的，壓力非常大，讓板子去轉彎旋轉。

### 改善方法 (How)
**錯誤做法**：
- 彎腰降低重心
- 只是腦袋離雪近了
- 屁股還是那麼高
- 沒區別

**正確做法**：
1. **壓在前刃之後已經是極限**
2. **讓兩個膝蓋往兩邊打開一點點**
3. **如果打開還不夠**
4. **通過把腳踝勾起來一點點再次往下壓**

**效果**：
- 既能保持背是直的
- 還能把這一部分往下降增加壓力
- 站一小會兒腿已經酸得不行
- 壓力非常大
- 比正常下蹲還酸

**對比**：
- 彎腰：腦袋下降但壓力不增加，還會往前趴
- 膝蓋打開 + 腳踝勾：真正下降增加壓力，背直不往前趴

**練習建議**：
- 先正常壓前刃
- 再使勁往下壓一下
- 會看到稍微多下去一點點
- 壓力迅速讓板子去轉彎旋轉

**核心**：
膝蓋往兩邊打開 + 腳踝勾起 = 真正降重心 + 增加壓力 + 背直
