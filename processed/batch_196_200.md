# 196. 通過"軸轉"幫你陡坡和蘑菇控速

## 問題描述 (What)

我在陡坡和蘑菇裡控速困難，彎轉得不夠圓，搓雪轉彎時抓地力不足。我不知道如何使用軸轉（pivoting）來幫助控速和讓彎形更圓潤。

## 改善目標 (Why)

我需要學習Counter Rotation（反向旋轉）的軸轉方式。做法是：後腿往後拉、往山下方向push推，前腳往山上方向走。當推後腳時，前腳本身就會往山上方向走，板子會出現軸轉。這是靠前腿與後腿反擰造成的。

先在彎結束的地方（過了滾落線之後、走下半圓時）開始嘗試。這樣彎會更加圓潤，速度控制更好，身體會進行反頂的軸轉。這是一個很緩慢的動作，不要做太快。之後可以把這個過程延展開來，一直拉到彎的滾落線甚至滾落線之前。這樣搓雪時彎形更圓潤、抓地力更強、速度控制非常好。

## 改善方法 (How)

### 步驟拆解

1. **理解Counter Rotation**：
   - CASI的兩種軸轉：Rotational和Counter Rotational
   - 今天講Counter Rotation

2. **理解動作模式**：
   - 後腿往後拉、往山下方向push推
   - 前腳往山上方向走
   - 推後腳時前腳本身就會往山上走
   - 板子會出現軸轉

3. **前後刃都一樣**：
   - 不管後刃還是前刃
   - 動作模式都一樣
   - 後腳稍稍往下推一點
   - 把前腳往回拉

4. **階段一：彎結束位置練習**：
   - 在過了滾落線之後
   - 開始走下半圓的時候
   - 在這個位置先嘗試

5. **感受效果**：
   - 彎會更加圓潤
   - 速度控制更好
   - 身體進行反頂的軸轉

6. **注意動作速度**：
   - 這是很緩慢的動作
   - 不要做太快

7. **階段二：延展到上半弧**：
   - 把這個過程延展開來、拉伸開來
   - 一直拉到彎的滾落線
   - 甚至滾落線之前去做

8. **最終效果**：
   - 搓雪時彎形更加圓潤
   - 抓地力會更強
   - 速度控制非常好

### 教練提示

- Counter Rotation是反向旋轉
- 後腳往下推、前腳往回拉
- 動作要緩慢，不要太快
- 先在彎結束位置練習
- 熟練後延展到滾落線之前
- 對搓雪、陡坡、蘑菇都有幫助

### 常見錯誤

- 動作做太快
- 只用一條腿，沒有前後腿反擰
- 不知道從哪個位置開始
- 沒有延展到上半弧
- 不理解Counter Rotation的原理

### 修正方法

- 放慢動作速度
- 確保前後腿反擰
- 先在彎結束位置練習
- 熟練後逐漸延展到滾落線之前
- 感受彎形圓潤和抓地力增強

# 197. 通過"肩膀"的起伏幫助你增加滑行穩定性

## 問題描述 (What)

我不知道上半身尤其是肩膀應該處於什麼位置，或者該怎麼動來幫助下半身的滑行。我可能只關注開肩問題，但不理解肩膀的上下起伏（往下壓或往上提）如何幫助增加對雪板的壓力、抓地力和穩定性。

## 改善目標 (Why)

我需要學習肩膀的上下起伏動作。肩膀會往下壓或往上提，出現一個倒向的動作。這可以幫助對雪板增加壓力、增加更多抓地力和穩定性。

具體做法：
- **前刃時**：略微把後肩抬起一點點，讓前肩下壓
- **後刃時**：後肩往下沉，讓前肩稍微往上抬一點點

這個動作可以在小彎、蘑菇、刻滑裡面應用。

## 改善方法 (How)

### 步驟拆解

1. **理解肩膀的動作**：
   - 肩膀會往下壓或往上提
   - 出現一個倒向的動作
   - 不是開肩的問題（以後會單獨講）

2. **理解好處**：
   - 對雪板增加壓力
   - 增加更多抓地力
   - 增加穩定性

3. **前刃時的動作**：
   - 轉到前刃時
   - 略微把後肩抬起一點點
   - 讓前肩下壓

4. **後刃時的動作**：
   - 在後刃時
   - 後肩往下沉
   - 讓前肩稍微往上抬一點點

5. **應用場景**：
   - 小彎
   - 蘑菇
   - 刻滑

6. **觀看示範**：
   - 在不同場景中觀察示範
   - 感受肩膀起伏的效果

### 教練提示

- 肩膀的上下起伏不同於開肩
- 前刃：後肩抬、前肩壓
- 後刃：後肩沉、前肩抬
- 可以增加壓力、抓地力、穩定性
- 在小彎、蘑菇、刻滑中都適用

### 常見錯誤

- 只關注開肩，忽略上下起伏
- 前後刃的肩膀動作搞反
- 動作幅度太大或太小
- 不知道如何應用
- 沒有感受到壓力和抓地力的增加

### 修正方法

- 理解肩膀上下起伏的作用
- 記住前刃後肩抬、後刃後肩沉
- 控制動作幅度，略微即可
- 在小彎、蘑菇、刻滑中練習
- 感受壓力和抓地力的變化

# 198. 道內應該"下帶上"

## 問題描述 (What)

我在滑行時可能存在上半身帶下半身轉彎的問題，或者自己沒有感覺到這個問題。我不理解為什麼不應該上帶下，也不知道如何檢查和改善。

## 改善目標 (Why)

我需要理解底層邏輯：**不推薦上半身帶下半身**。原因是如果上半身帶下半身，等傳遞到板子會有延遲，反應不會很快。應該以髖開始，然後膝蓋腳踝跟隨。最快的是腳踝，但力量比較小，所以以髖為主、膝蓋與腳踝跟隨。

檢查和改善方法：把手背在後面，以髖去主導下半身帶動轉彎。手背後面時上半身間接打開朝前，方便下半身主動發力，不會讓上半身擋住。如果手背後面滑得特別別扭，說明時不時會上帶下。如果手背後面滑得跟平常一樣，說明下帶上的感覺非常好。如果胳膊粗背不到後面，可以把兩手插兜，但背在後面效果最好。

## 改善方法 (How)

### 步驟拆解

1. **理解底層邏輯**：
   - 不推薦上半身帶下半身
   - 上帶下會有延遲，反應不快
   - 應該以髖為主

2. **理解正確方式**：
   - 以髖開始
   - 膝蓋與腳踝跟隨
   - 腳踝最快但力量小
   - 所以以髖為主、膝蓋腳踝跟隨

3. **檢查方法**：
   - 把手背在後面
   - 以髖主導下半身帶動轉彎
   - 手背後面時上半身間接打開朝前

4. **判斷標準**：
   - 手背後面滑得特別別扭：說明會上帶下
   - 手背後面滑得跟平常一樣：說明下帶上很好

5. **替代方法**：
   - 如果胳膊粗背不到後面
   - 可以把兩手插兜
   - 但背在後面效果最好

6. **理解好處**：
   - 方便下半身主動發力
   - 不會讓上半身擋住

### 教練提示

- 道內應該下帶上，不是上帶下
- 上帶下會有延遲
- 以髖為主、膝蓋腳踝跟隨
- 手背後面是最好的檢查方法
- 如果別扭說明有上帶下問題

### 常見錯誤

- 上半身帶下半身轉彎
- 不知道自己有這個問題
- 不理解為什麼不應該上帶下
- 沒有檢查方法
- 不知道如何改善

### 修正方法

- 理解上帶下會有延遲
- 練習以髖為主的轉彎
- 用手背後面的方法檢查
- 如果別扭就多練習手背後面
- 目標是手背後面也能流暢滑行

# 199. 蘑菇裡：前刃換後刃更穩定

## 問題描述 (What)

我在蘑菇裡因為地形不平整，從前刃換後刃或後刃換前刃時會被送起來。落地時需要找平衡，或者自己跳起來換刃時落地不穩定。我不知道如何讓落地更穩定。

## 改善目標 (Why)

我需要學會讓落地更穩定的方法。今天先講前換後：落地時不是刃落地，而是**板底落地**。所有關節的彎曲都是為了讓板底落地。

具體做法：前換後刃時，當板底落地時，盡量蹲低，人朝山上方向下沉。當朝山上方向下沉時，兩個膝蓋就朝前刃發力，小腿會頂在鞋舌上面，這樣會把原本後刃拉起來的角度直接壓回來。當壓回來時兩個板同時落、板底同時落地，就很穩定。

## 改善方法 (How)

### 步驟拆解

1. **理解問題**：
   - 地形不平整會被送起來
   - 落地時需要找平衡
   - 自己跳起來換刃落地不穩

2. **核心原則**：
   - 落地時不是刃落地
   - 而是板底落地
   - 所有關節彎曲都為了板底落地

3. **前換後的做法**：
   - 當板底落地時盡量蹲低
   - 人朝山上方向下沉
   - 這是關鍵動作

4. **理解力學**：
   - 朝山上方向下沉時
   - 兩個膝蓋朝前刃發力
   - 小腿頂在鞋舌上面

5. **效果**：
   - 把原本後刃拉起來的角度壓回來
   - 兩個板同時落
   - 板底同時落地
   - 很穩定

6. **練習示範**：
   - 觀看示範理解動作
   - 感受板底落地的穩定性

### 教練提示

- 落地時板底落地，不是刃落地
- 人朝山上方向下沉是關鍵
- 膝蓋朝前刃發力、小腿頂鞋舌
- 這樣可以壓回後刃角度
- 兩個板同時落地最穩定

### 常見錯誤

- 落地時刃先落地
- 沒有朝山上方向下沉
- 膝蓋沒有朝前刃發力
- 小腿沒有頂鞋舌
- 兩個板不是同時落地

### 修正方法

- 刻意讓板底落地
- 練習朝山上方向下沉
- 感受膝蓋朝前刃發力
- 感受小腿頂鞋舌的感覺
- 確保兩個板同時落地

# 200. 蘑菇裡你過度重心了麼？

## 問題描述 (What)

我在蘑菇裡滑行時，後刃可以減速，但前刃沒有辦法減速。原因是重心沒有換到前刃。我不知道如何在包裡正確換重心。

## 改善目標 (Why)

我需要學會在包裡換重心的兩種方法（都推薦團身換）：

**關鍵**：換重心的位置都是在**上包的時候去換**，而不是上到包頂再換。上到包頂時人應該站得比較直，重心應該已經過度了。一旦過了包頂就面臨往下走，要面臨施壓，所以當上包頂時重心應該得到轉換了。

**方法一：站起來換重心**（不是團身）
- 在上包過程中站起來換重心

**方法二：團身換重心**
- 上包時想著包會推你
- 主動彎曲膝蓋與腳踝
- 腿就會被推起來
- 上半身保持不變
- 一旦過了包頂再把腿伸展出去減速

兩種方法都要記住：把重心從後刃換到前刃是在上包過程中完成的。

## 改善方法 (How)

### 步驟拆解

1. **識別問題**：
   - 後刃可以減速
   - 前刃沒辦法減速
   - 原因是重心沒換到前刃

2. **理解換重心時機**：
   - 在上包的時候去換
   - 不是上到包頂再換
   - 上到包頂時人應該站比較直
   - 重心應該已經過度了

3. **理解為什麼**：
   - 一旦過了包頂就往下走
   - 要面臨施壓
   - 所以上包頂時重心應該已轉換

4. **方法一：站起來換重心**：
   - 不是團身
   - 在上包過程中站起來
   - 換重心

5. **方法二：團身換重心**：
   - 上包時想著包會推你
   - 主動彎曲膝蓋與腳踝
   - 腿被推起來
   - 上半身保持不變
   - 過了包頂再把腿伸展出去減速

6. **重點強調**：
   - 把重心從後刃換到前刃
   - 在上包過程中完成

7. **場地選擇**：
   - 先在稍微綠道一點有包的地方嘗試
   - 之後再慢慢進入藍道

### 教練提示

- 換重心在上包時，不是包頂
- 包頂時重心應該已經過度了
- 兩種方法都可以：站起來或團身
- 團身時主動彎曲膝蓋腳踝
- 過包頂後伸展腿減速

### 常見錯誤

- 上到包頂才換重心
- 重心沒有過度就下包
- 前刃無法減速
- 不知道換重心時機
- 只會一種方法

### 修正方法

- 刻意在上包時換重心
- 確保包頂時重心已過度
- 兩種方法都練習
- 先在綠道有包處練習
- 熟練後再到藍道
