## 116 前刃中後段：立住了

### 問題描述 (What)
前刃彎從上面往下轉到下半段時，人會越往下轉越往彎裡面去趴、往前倒，這樣會越來越喪失壓力，無法再多給板子壓力。

### 改善目標 (Why)
在彎的後半段（滾落線之後下半段）能再增加壓力讓人不倒，保持背是直的，多給壓力的同時人不會往離心方向倒。

### 改善方法 (How)
**問題分析**：
- 前刃很好感受刃壓在雪裡，壓力很從容能給上
- 但轉到下半段時，給舌頭壓力往下壓，人會情不自禁往前趴往前倒
- 往前倒時絕對不可能再多給板子壓力
- 越往前倒越喪失壓力

**解決方法**（在滾落線位置和彎的後半段）：
1. **兩個膝蓋稍微往固定器方向打開一點**
2. **腳踝上勾一點**
3. **效果**：
   - 可以多下降一點
   - 同時保持背是直的
   - 不會往前趴

**錯誤做法**：
- 往前趴
- 背彎曲
- 人往離心方向倒

**正確做法**：
- 膝蓋往固定器方向打開
- 腳踝上勾
- 背保持直
- 人再下降一點
- 多給壓力但不倒

**關鍵時機**：
- 彎的中後段
- 滾落線位置
- 滾落線之後下半段

**核心原則**：
膝蓋打開 + 腳踝上勾 = 多下降 + 背直 + 不往前倒 + 增加壓力

## 117 前換後：保持腳踝關閉

### 問題描述 (What)
前刃轉後刃時，人往後躺，腳踝會打開（open），這樣很容易直接翻過去摔倒。

### 改善目標 (Why)
防止人往後倒，轉過去的時候人還是居中在板子上面。

### 改善方法 (How)
**問題分析**：
- 前轉後時，如果人往後躺
- 腳踝會 open（打開）往後躺
- 很容易直接翻過去摔倒

**解決方法**：
1. **當往後躺的時候**
2. **讓腳踝不要打開**
3. **讓膝蓋還是挺向前面方向**
4. **只不過人往後躺一點**
5. **腳踝往上勾一點**

**效果**：
- 轉過去的時候人還是居中在板子上面
- 不會翻過去摔倒

**重點觀察**：
- 注意前腿
- 尤其是前腿
- 轉的時候盡量保持腳踝不會打開

**核心動作**：
- 腳踝關閉（不打開）
- 膝蓋挺向前
- 腳踝往上勾
- 人往後躺但居中

**關鍵**：
腳踝關閉 + 膝蓋向前 + 腳踝上勾 = 人居中不翻倒

## 118 前換後：膝蓋帶動腳踝去控制立刃

### 問題描述 (What)
前刃換後刃入刃時，刃容易擦出去或掃出去，無法直接立住，立刃角度不精準。

### 改善目標 (Why)
幫助控制立刃和入刃，一入刃刃就不會擦出去或掃出去，會直接立住，把刃壓在雪裡面更加精準。

### 改善方法 (How)
**核心技巧**：
1. **膝蓋朝板頭方向**
2. **通過髖軸轉向板頭方向**
3. **前腿腳踝會自動很自然地往上勾**

**好處**：
- 一入刃，刃不會擦出去
- 不會掃出去
- 會直接立住

**側面觀察**：
- 當膝蓋往板頭方向一頂的時候
- 腳踝很容易去控制立刃角度

**對比說明**：
- **錯誤**：膝蓋不往板頭頂，單純去立刃 → 很容易倒在地上
- **正確**：膝蓋通過髖的轉動往板頭頂 → 重心可以保持在板子上面

**動作順序**：
1. 髖軸轉向板頭方向
2. 膝蓋朝板頭方向
3. 前腿腳踝自動往上勾
4. 刃直接立住

**效果**：
- 重心保持在板子上
- 立刃更精準
- 刃壓在雪裡更準確

**注意**：
- 這是前刃換後刃的技巧
- 後刃換前刃會在其他視頻講解

**核心原則**：
髖軸轉 → 膝蓋朝板頭 → 腳踝自動上勾 → 刃直接立住

## 119 前刃換後刃基礎：如何不後「躺」

### 問題描述 (What)
前刃換後刃時，身體會像自由落體一樣往下躺，往後躺著摔過去，這很恐怖。

### 改善目標 (Why)
確保人不會往後躺，而是朝行進方向行進，不會摔跤。

### 改善方法 (How)
**適用場景**：
- 比較基礎的前刃換後刃
- 在稍微藍道或者黑道

**核心原理**：
- 跟後換前是一樣的
- 當平板的時候，人是往板頭方向去壓、去頂的

**小技巧**：
1. **當平板的時候**
2. **讓膝蓋朝著固定器的角度去頂過去**
3. **換句話說：把膝蓋朝前面方向壓過去**

**效果**：
- 確保人不會往後躺
- 不會躺著摔過去

**正確信號**：
- 當把膝蓋朝著固定器角度壓過去的時候
- 會感覺前腳舌頭外側這一側受到了壓力
- 這是一個很好的信號
- 證明人沒有往後躺
- 是往板頭方向去發力的

**結果**：
- 朝行進方向行進
- 而不是往後躺
- 就不會摔跤

**關鍵動作**：
平板時 → 膝蓋朝固定器角度頂過去 → 往板頭方向壓 → 不往後躺

**檢查方法**：
感覺前腳舌頭外側受壓 = 做對了

## 120 前刃基礎站姿：道內可以更直

### 問題描述 (What)
後刃換前刃時人不站起來，導致髖頂到某個位置沒辦法再往前，長此以往腰會疼，尤其是下腰位置，因為一直壓著。

### 改善目標 (Why)
讓前刃站姿更直，沉在上面保持放鬆，前刃會給足夠的支撐，對以後前刃站姿和發力準確度有極大幫助。

### 改善方法 (How)
**問題分析**：
- 後刃換前刃時人不站起來
- 髖頂到某個位置沒辦法再往前
- 長期會導致腰疼，尤其下腰位置

**錯誤站姿**：
- 彎著的
- 髖頂住
- 無法再往前

**正確站姿**：
- 沉在上面
- 保持放鬆
- 更直一點

**進階練習**（容錯率很低）：
1. **靜態練習**：
   - 先在靜態下試試正確站姿
   - 讓正確位置壓住
2. **動態練習**：
   - 後刃換前刃時
   - 在綠道練習
   - 嘗試跳後刃跳到前刃這個位置
3. **落地要求**：
   - 落在包封的上端
   - 整個前腳掌上
   - 會感覺到很穩
   - 前刃會給足夠的支撐

**練習特點**：
- 容錯率很低
- 一旦做不到基本上有可能就會摔跤
- 但一旦做到了，對以後前刃站姿有極大幫助
- 發力的準確度會有很大幫助

**效果**：
- 身體在前刃上面
- 前刃給足夠支撐
- 感覺很穩
- 站姿準確性提高
- 發力準確度提高

**注意事項**：
- 這是比較難的練習
- 需要在綠道練習
- 觀察仔細的話能看出差異

**核心**：
沉在上面 + 保持放鬆 + 更直 = 前刃給足夠支撐
