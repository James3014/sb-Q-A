# 164. 撬板頭時候上半身跟上

## 問題描述 (What)

我在粉雪曬了兩三天後（溫度高、不下雪）的軟雪裡滑行，尤其是在道外，當我用力時板子刃會砍到雪裡。我需要在每次換刃時借助反彈力把板頭從雪裡拔出來，但我可能在送板子時重心沒有跟上，導致人往後倒、壓著板尾，然後摔倒。

## 改善目標 (Why)

我需要學會在送板子（把板子往前送、拉起板頭）時，上半身要跟上。當我把板子往前送的時候，人要隨著板子往前探，上半身（頭、肩膀）要跟著往前來。這樣當板子送起來、換到下一個刃時，我的重心還是居中處於板子上面，就可以順著滾落線衝下去。如果上半身沒跟上，人會往後倒壓著板尾而摔倒。

## 改善方法 (How)

### 步驟拆解

1. **識別雪況**：
   - 粉雪曬了兩三天
   - 溫度高、不下雪
   - 雪比較軟，刃容易砍進去
   - 尤其在道外

2. **理解送板子的方法**：
   - 方法一：兩個腳一起送
   - 方法二：重心往後壓，板頭被拉起來
   - 這兩種方法都可以把板頭從雪裡拔出來

3. **關鍵：上半身跟上**：
   - 當板子往前送的時候
   - 人要隨著板子往前探
   - 頭、肩膀、上半身要跟著往前來
   - 這是同時進行的動作

4. **保持居中**：
   - 板子送起來後
   - 換到下一個刃時
   - 重心還是居中在板子上面
   - 可以順著滾落線衝下去

5. **避免錯誤**：
   - 如果只送板子，上半身沒跟上
   - 人會往後倒
   - 壓著板尾
   - 導致摔倒

### 教練提示

- 送板子和上半身前探是同時進行的
- 上半身包括頭、肩膀、整個上半身
- 目標是換刃後重心還是居中
- 這個技巧在慢雪、不好換刃的情況特別有用
- 道外軟雪是典型應用場景

### 常見錯誤

- 只送板子，上半身沒跟上
- 送板子後人往後倒
- 壓著板尾導致摔倒
- 換刃後重心不居中
- 不知道要同時進行兩個動作

### 修正方法

- 刻意練習送板子時上半身前探
- 感受頭、肩膀往前的動作
- 確保換刃後重心居中
- 先在較緩的道外練習
- 逐漸應用到更軟更深的雪況
