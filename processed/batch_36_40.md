# 36 又爛又粘的雪：speed check plus

## 📝 教學內容

### 問題描述 (What)
在崎嶇不平、已經成褶的雪裡面，而且比較粘（扔地上都粘住的那種）。這種雪剎車也難受，不剎呢速度還快。

### 改善目標 (Why)
- 解決怎麼去剎車的問題
- 在比較密一點的樹林或者比較粘一些、比較 heavy 一點的雪裡面減速
- 控制速度

### 改善方法 (How)

**Speed Check Plus 技巧**：

1. **基本動作**：
   - 做 Speed Check
   - 把重心壓在板尾

2. **動作效果**：
   - 當去一壓的時候，板子不會往下走
   - 板頭會往這邊（往左邊去撇）
   - 當它撇的時候，板頭由於壓的力量特別大，板頭會彈起來
   - 當彈起來之後，速度就在開始減少了

3. **釋放壓力**：
   - 等把這個壓力減少放開的時候
   - 板頭會迅速的指向山下
   - 然後速度就會瞬間加速

**關鍵提示**：
- 這就是一種在這種比較密一點的樹林或者這種比較粘一些、比較 heavy 一點的雪裡面減速的方法

# 37 上半弧（閉合彎）第一集：結束時擰板

## 📝 教學內容

### 問題描述 (What)
彎形是開放的，不夠圓潤，沒有上半弧。

### 改善目標 (Why)
- 讓彎形由開放變成圓潤
- 形成上半弧
- 幫助減速（擁有上半弧彎形圓潤是可以幫助減速的最有效的工具）
- 在遇到一些需要考試或者比較陡一點的地方讓速度減緩

### 改善方法 (How)

**擰板形成上半弧技巧**：

1. **什麼是上半弧**：
   - 從滾落線之前到滾落線這個位置，就是所謂的上半弧
   - 當有這一部分的時候，可以有效的幫助減速，因為行程加多了
   - 如果行程不加入這個，直接從滾落線開始結束，會讓速度加快，因為很多時間都處於滾落線上面

2. **形成上半弧的第一步**：
   - 如果板子不往這邊走，也就沒有辦法出現上半弧
   - 當板子從上面從滾落線轉到這兒的時候，換刃之後，才能形成上半弧
   - 如果板子在滾落線到滾落線在這兒就換刃了，不往這邊拐在這兒就換刃了，那直接進入下一個彎的時候也就變成無滾落線了

3. **後腳發力擰板**：
   - 怎麼讓板子從這個位置讓它往這邊走呢，就需要借助到後腳
   - 當在滾落線的時候，需要讓後腳多發點力
   - 往哪發力呢？往山的下側或者兩側去發力
   - 這樣板子會產生一個形變，才會讓板子開始往地面走

4. **Torsional Flex（扭轉形變）**：
   - 在彎結束的時候，要進入下一個彎了
   - 讓後腳往下壓一點點
   - 當往下壓，前腳保持放鬆的話，板子會出現一個叫做 Torsional Flex
   - 就是這個往下壓，這個它保持不動，相當於它就有點往上走了
   - 會出現一個擰板的感覺

5. **前刃和後刃都適用**：
   - 後刃換前刃也是同理
   - 當後刃換前刃的時候，後腳往下踩，會往這邊下
   - 會出現這種一個頂的感覺
   - 當在彎結束的時候，這個就代表了板頭已經開始往山上走了
   - 當它一旦開始往山上走的時候，下半弧的開始也就會出現了

# 38 上半弧第二集：後腿搓（碾雪）和踢（無抓地力）

## 📝 教學內容

### 問題描述 (What)
在彎形的過程當中，如果重心還沒有過渡到後腳的時候就去推後腳的話，板尾就會掃出去，會以前腿為軸去掃。如果在過渡的過程中去推後腳，也有可能會造成這種情況。

### 改善目標 (Why)
- 讓板尾不會掃出去
- 讓彎直接閉合
- 讓彎不會往下走，而是往這邊走

### 改善方法 (How)

**後腳推的時機技巧**：

1. **重心過渡**：
   - 在彎形的過程當中，重心其實是會從前腳慢慢過渡到後腳

2. **錯誤時機**：
   - 如果重心還沒有過渡到後腳的時候去推後腳的話，板尾就會掃出去
   - 會以前腿為軸去掃
   - 如果在過渡的過程中去推後腳，也有可能會造成這種情況

3. **正確時機**：
   - 最好的練習方式是：當重心從前腳已經過渡到板尾這個位置的時候，再去推後腳
   - 板尾絕對不會掃出去
   - 會讓彎直接閉合
   - 彎就不會往下走了，而是往這邊走

# 39 不做這個，折疊無效

## 📝 教學內容

### 問題描述 (What)
換刃的時候，從刃直接趴過去換到刃，沒有機會去折疊，或者說會很別扭。

### 改善目標 (Why)
- 能夠流暢地加入折疊
- 控制好速度
- 讓板子刃不會帶著走
- 可以靠去推雪去搓雪，幫助控制想行進的方向

### 改善方法 (How)

**平板過渡折疊技巧**：

1. **核心概念**：
   - 當換刃的時候，不要著急從刃換到刃
   - 中間加一個平板的過程
   - 把重心從這移過去再折疊

2. **錯誤方式**：
   - 如果從這個地方直接趴過去的話，是沒有機會去折疊的

3. **正確方式**：
   - 當一旦平板，把重心換過去的時候
   - 很流暢的加入折疊

**效果**：
- 同時也可以幫助控制好速度
- 這樣板子刃就不會帶著走了
- 而是可以靠去推雪去搓雪，幫助控制想行進的方向
- 同時速度也會控制的比較好

# 40 不做這個折疊無效（高級道和蘑菇）

## 📝 教學內容

### 問題描述 (What)
在高級道和蘑菇裡面，換刃時重心換過來之後會造成一點點的傾倒，往前趴或者往後躺。在中級道上面是 OK 的，因為容錯率比較高，但是在高級道上面由於坡度比較陡、雪況比較複雜，這樣會不穩定容易摔跤。

### 改善目標 (Why)
- 在高級道上面或者比較陡一點的藍道黑道或者包裡面，重心穩定
- 不會摔跤

### 改善方法 (How)

**高級道折疊範圍控制技巧**：

1. **中級道 vs 高級道的差異**：
   - 在中級道裡面，簡單一點的綠道裡面，重心換過來之後同時會造成一點點的傾倒，往前趴或者往後躺，在中級道上面是可以、是 OK 的，因為容錯率比較高，允許出現失誤
   - 但是如果在高級道上面，由於坡度比較陡、雪況比較複雜，需要更精準的控制

2. **前刃折疊**：
   - 當折疊的時候，盡量讓在前刃的裡面
   - 不要太出去前刃

3. **後刃折疊**：
   - 當折疊的時候，盡量維持髖裡面去折疊
   - 而不是往外折疊

**效果**：
- 這樣在高級道上面或者比較陡一點的藍道黑道或者包裡面，重心就是穩定的，就不會摔跤了
