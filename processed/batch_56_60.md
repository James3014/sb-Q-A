# 56 後刃彎結束時候腿如何保持彎曲

## 📝 教學內容

### 問題描述 (What)
在滑雪的時候不管是搓雪啊或者刻滑啊，當滑的時候後刃的時候有時候，當一去把膝蓋彎曲會變成膝蓋鎖死的樣子。一旦當彎曲腳踝的時候，會把膝蓋被鎖死。在滑雪的過程中，當後刃去推雪或者搓雪任何事情也好的時候，會出現一個在彎結束的時候膝蓋會直起來，就感覺腿整個是蹬直了。

### 改善目標 (Why)
- 讓膝蓋跟腳踝都保持微曲狀態
- 不會出現鎖死的狀態
- 讓發力變大
- 更加穩定

### 改善方法 (How)

**膝蓋與腳踝同時彎曲技巧**：

1. **問題原因**：
   - 膝蓋與腳踝，它們兩個是一起起作用的
   - 只要彎膝蓋了，腳踝同時去彎
   - 沒有先後，只有同時

2. **正確動作**：
   - 一旦要彎膝蓋與腳踝的時候，會這樣去彎
   - 屁股會稍微往板子中央多來了一點點
   - 這樣的話會讓穩定
   - 同時當帶燈的時候，發力也會變大

3. **效果**：
   - 每次當去登的時候，發力這麼多就夠了
   - 彎結尾的時候，就能保持一個膝蓋跟腳踝都是微曲狀態
   - 不會出現這麼一個鎖死的狀態

# 57 後膝蓋的彎曲方向

## 📝 教學內容

### 問題描述 (What)
看到別人膝蓋成這個樣子或者那個樣子的時候，不知道為什麼，也不知道如何做到。

### 改善目標 (Why)
- 增加這一區域板子的抓地力
- 讓彎形迅速的閉合
- 當髖關節往這邊移動的時候，後膝蓋可以有更多的彎曲的程度
- 壓雪的時候力量就會更大
- 速度從而也就能控制的更好

### 改善方法 (How)

**髖關節配合膝蓋方向技巧**：

1. **膝蓋往外打開**：
   - 如果想讓膝蓋往外面多打開一點的時候，或者朝著這個方向多打開一點的時候
   - 需要讓髖關節到這個位置
   - 當彎的時候，膝蓋自然而然就會往這邊多打開了
   - 它的好處是什麼呢？它可以增加這一區域板子的抓地力

2. **膝蓋內扣**：
   - 當在公園裡面上道具的時候，會看到很多人就會去有點內扣膝蓋來增加他們這個 style
   - 膝蓋不只是單純的自己這樣的內扣，它也是需要髖關節的配合
   - 當髖關節旋轉到一定地步的時候，膝蓋會很自然的往裡內扣
   - 但是如果髖關節不旋轉，去內扣膝蓋，它就會傷害到膝蓋，從而不會增加 style

3. **滑行應用**：
   - 當讓膝蓋、髖關節往這邊膝蓋往板尾方向稍稍打開一點點的時候
   - 它這個增加板子這個區域的抓地力，可以對滑行起到一個很好的作用

4. **效果**：
   - 一旦把髖關節往後一點點，當膝蓋會往這邊略微打開的時候
   - 它可以幫助把彎形迅速的閉合
   - 第二點呢當髖關節往這邊移動的時候，後膝蓋它可以有更多的彎曲的程度
   - 它去壓雪的時候它的力量就會更大，速度從而也就能控制的更好

# 58 在"包"裡通過把彎轉"完"控速

## 📝 教學內容

### 問題描述 (What)
在包裡面不知道如何通過把彎轉完來控制速度。

### 改善目標 (Why)
- 通過把彎轉完來幫助控制速度
- 同時也有彎形了
- 彎形也就圓了
- 速度也減下來了

### 改善方法 (How)

**彎轉完控速技巧**：

1. **動作原理**：
   - 當在彎結束的時候，板子會繼續朝行進方向過渡
   - 板頭甚至有點可能會往山上要拐的意思
   - 但是人呢，人不會跟著板子往山上去拐
   - 一往山上拐，人就會往後躺，這是不希望發生的

2. **人與板的配合**：
   - 人在板子往上走的時候，人會繼續往下走
   - 所以說當人往下走之後，板子會被人拽回來來進行下一個彎

3. **Counter Rotation**：
   - 板子它在往上去繞的時候，人會開始往下走
   - 它會出現這麼一個夾角
   - 然後再通過這個夾角，產生了這個 Counter Rotation
   - 把人帶回來

4. **效果**：
   - 這樣的話彎形也就圓了
   - 速度也減下來了
   - 因為當板子往上走的時候，它本身就會減速

**應用情境**：
- 通過這個方式來幫助在包裡面，尤其是大包裡面來控制速度

# 59 "科班"後遺症：前手的位置

## 📝 教學內容

### 問題描述 (What)
滑的其實挺好的，小彎可能已經接近於三級了，但是當前換後的時候，尤其是在滾落線的時候，把前手放在身體的前面。當在滾落線還放在身體前面的時候，在彎結束的這個地方，會很容易出現一個髖會這種憋住，沒有辦法得到一個充分的轉動這種感覺。尤其是當板子給回彈的時候，由於手在這，身體、胸也會開始逐漸寒胸。

### 改善目標 (Why)
- 讓髖能夠充分轉動
- 讓胸腔打開
- 讓轉動是非常自然的
- 讓後腿有足夠的抓地力
- 能夠完全打開之後進行一個更有效的起伏

### 改善方法 (How)

**前手背後面練習**：

1. **問題分析**：
   - 如果手放在前面的話，胸腔會寒胸
   - 在彎快結束的時候，有時候後腿可能甚至都有可能有點抓地力不足
   - 是因為沒有辦法完全打開之後進行一個更有效的、充分的起伏

2. **練習方法**：
   - 把前手背後面
   - 會感覺這個位置胸腔是打開的
   - 就像小時候穿的那個貝貝夾一樣

3. **效果**：
   - 如果手放在後面的話，在大腿後側這個地方的話
   - 胸腔也打開了
   - 轉動是非常自然的

**關鍵提示**：
- 這個練習可能會讓感覺到有點不舒服，那就對了
- 因為有可能從來沒感受過在滾落線之後上半身還是比較直的、胸腔是打開的這種感覺

# 60 不要"Park and Ride"

## 📝 教學內容

### 問題描述 (What)
滑小彎的時候，在進入滾落線之前會保持一個不動的姿勢，然後之後結束彎的時候過了滾落線之後就起來，然後再進入一個這種姿勢，然後再起來。就感覺這個人一卡一卡的，比較僵硬、比較 static。

### 改善目標 (Why)
- 讓滑得更加的 Lively（鮮活）
- 更加的靈動一些
- 更加 Dynamic 一些
- 更加 Smooth 一些
- 能夠隨意地控制彎形

### 改善方法 (How)

**延長加速段技巧**：

1. **問題分析**：
   - 會有一個彎曲，在這裡開始蹲，一直蹲到這裡起來
   - 在這裡起來的時候加速了
   - 當一蹲下來的時候會加速
   - 這個壞處就是沒有辦法隨意地控制彎形
   - 因為身體處於一個僵的狀態，只能說在維持它去減速
   - 因為其實是在側向推坡而在減速

2. **改善方法**：
   - 需要在這裡開始蹲，把加速從這裡延長
   - 這裡加速，這裡減速
   - 彎就可以看起來更加的奶奶美、更加的靈動一些

3. **減速的位置**：
   - 減速的位置就在這個圓的上面這一丟丟
   - 下半段由於開始蹬，其實是開始加速的
   - 而且這個加速是通過 deflection 反彈力產生的

4. **練習提示**：
   - 這種感覺是對的
   - 當試的時候，起初會感覺到一些不穩
   - 甚至有可能把自己人彈射出去
   - 但是這種感覺是對的
   - 強加練習，就會越來越磨合得越來越好
