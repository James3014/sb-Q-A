# 26 冰蘑菇基礎：下個包頂"撞"

## 📝 教學內容

### 問題描述 (What)
在比較冰的蘑菇包裡滑行，不知道如何安全地轉彎和減速。

### 改善目標 (Why)
- 在冰包裡面稍微安全一點去滑
- 能夠轉彎和減速
- 利用包頂作為緩衝區

### 改善方法 (How)

**撞包頂減速技巧**：

1. **基本方法**：
   - 推下來
   - 頂到下一個包的包頂停住

2. **有積雪的情況**：
   - 比較冰的包，一般在包下來之後在下一個包的包頂有一定的積雪
   - 可以利用那個位置當一個緩衝區去幫助減速

3. **沒有積雪的情況**：
   - 從冰上推下來的時候，推的勁兒抓地力要想辦法讓它大一點
   - 這樣當頂到下一個包的包頂時，不會因為力量太大震得膝蓋疼或者翻過去

4. **練習建議**：
   - 把動作慢慢熟練之後，會有一些 Rhythm（節奏）

**關鍵提示**：
- 這是一個比較安全但也比較是屬於下策的方式，不屬於最好的方式
- 個人不太建議去滑冰包，因為沒有什麼樂趣
- 但如果雪場冰包是唯一有的，那沒辦法，可以這麼去練一練

# 27 後刃開髖：看看缺不缺這個

## 📝 教學內容

### 問題描述 (What)
後刃下去時，只是轉髖，當轉得越來越多時會自然而然把膝蓋拉直。在雪道上會有一種卡住的感覺，下降到某個位置會感覺卡住。

### 改善目標 (Why)
- 讓髖的轉動更順暢
- 避免膝蓋拉直
- 提升轉彎效率和彎的圓潤度
- 讓下降的弧度過程非常 smooth（順滑）
- 在蘑菇包裡面非常有用

### 改善方法 (How)

**髖膝配合技巧**：

1. **髖打開的感覺**：
   - 髖得保持打開
   - 以中間為軸這個地方在轉
   - 腰腹這個部位有點像擰毛巾一樣把水從毛巾裡面擰出來
   - 保持這麼一個姿勢，這種擰的姿勢是持續的

2. **膝蓋的作用**：
   - 不只是髖，膝蓋也在這個轉動的過程中起到很重要的作用
   - 有的人只是轉髖，當轉得越來越多時會自然而然把膝蓋拉直

3. **髖膝配合**：
   - 當轉髖的同時，盡量讓兩個膝蓋保持住一個彎曲的狀態
   - 甚至可以往前轉，往外擴一點
   - 這樣當下降的時候，整個髖的轉動是一個很順暢的樣子
   - 就不會在雪道上有一種卡住的感覺

**效果**：
- 當能把髖跟膝蓋連接到一起的時候，轉彎效率、彎的圓潤、下降的弧度過程是非常 smooth 順滑的

# 28 後刃換前刃擰不過來？試試這個

## 📝 教學內容

### 問題描述 (What)
後刃換前刃時很著急，會用力去轉，但擰不過來。

### 改善目標 (Why)
- 讓後刃換前刃更容易
- 避免著急用力轉
- 在陡坡上更容易把刃換過來
- 讓身體有往山下突下去的感覺

### 改善方法 (How)

**換到板底技巧**：

1. **核心概念**：
   - 不是非得把刃換到刃
   - 而是從後刃換到板底
   - 換到板底之後，就已經處於在前刃上面了
   - 因為坡是有坡度的

2. **坡度的作用**：
   - 站在前刃上面，但沒有想站在前刃上
   - 只想居中的站，就已經造成立刃了
   - 當換到板底的時候，在越陡的坡上面，越容易把刃換過來

3. **換刃方向**：
   - 當換到板底的時候，朝行進方向走
   - 也就是左邊或者右邊
   - 不要朝山下走

4. **下壓控速**：
   - 換到板底之後，下壓來幫助控制速度就夠了
   - 或者換前刃
   - 想讓身體有往山下突下去那種感覺

# 29 如何背直：雙手永遠別出現

## 📝 教學內容

### 問題描述 (What)
背部不夠挺直，駝背。

### 改善目標 (Why)
- 讓背挺直
- 讓人更直立於板子上面
- 更加穩定

### 改善方法 (How)

**手不出現在視野技巧**：

1. **核心技巧**：
   - 永遠不要讓兩個手出現在視野之中

2. **錯誤示範**：
   - 如果兩個手在視野裡面
   - 肩胛骨會往前走
   - 會讓背比較彎

3. **正確示範**：
   - 稍微往後放一點
   - 就算往前看，手也不會出現在視野裡面
   - 當這樣做的時候，背已經被拉直了

4. **滑行應用**：
   - 滑的時候，就想著這一個事情
   - 手不要出現在視野裡面
   - 基本上背就是直的了
   - 人更直立於板子上面，會更加穩定

**關鍵提示**：
- 這個跟肩和手有很多關係
- 這是最簡單的技巧

# 30 如何滑下午的冰面

## 📝 教學內容

### 問題描述 (What)
到下午時，上面的好雪已經被刮掉了，面條雪已經被刮掉了，下面剩比較硬或者比較冰的一層雪。不知道如何控制。

### 改善目標 (Why)
- 在冰面上保持穩定
- 增加板底接觸面積
- 更平衡在冰面上
- 能夠上下壓縮給壓力減速

### 改善方法 (How)

**冰面站姿技巧**：

1. **後刃站姿**：
   - 人站直時，板子跟坡度會產生一個角度
   - 往後面拐，角度會拉大
   - 為了多讓板底去接觸雪，增加接觸面積
   - 讓兩個膝蓋往前面方向跪下去，往下壓
   - 當往下跪時讓膝蓋往前壓，會感覺到小腿貼到了鞋舌

2. **前刃站姿**：
   - 同樣一個道理，本身就會產生一個角度
   - 一旦坡變陡或者冰面變冰，本能反應是要往前趴
   - 不應該往後仰（如果想到了後刃，可能會想是不是應該往後仰）
   - 只是更加讓重心壓在足弓這個位置
   - 如果壓不到足弓，也可以壓掌中寶

3. **壓力控制**：
   - 在這個站姿下，上下去壓縮去給壓力減速
   - 就可以保持一個穩定的狀態
   - 不管是冰面還是好雪都是一樣的道理

**關鍵提示**：
- 反復強調這個問題很多次
- 尤其是在冰面上
