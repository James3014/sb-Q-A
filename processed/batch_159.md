# 159. 滑雪"下蹲"這個量如何把控

## 問題描述 (What)

我在滑雪時不知道下蹲應該蹲多少才夠，有時候朋友說我蹲得不夠，但我不清楚這個"量"該如何把控。我可能蹲得太少導致無法控制速度，或者蹲得太多（超過90度）變成無效蹲，腿已經不再朝雪發力。我也不理解在包裡超過90度和主動蹲超過90度的區別。

## 改善目標 (Why)

我需要學會把控下蹲的量。正確的下蹲量是：大腿小腿不要超過90度（六七十度左右），在這個角度下蹲30秒到1分鐘會感覺大腿酸，這就是正確的量。超過90度就是無效蹲，因為已經靠關節支撐而不是肌肉發力。在包裡被雪推導致超過90度是正常的，但主動蹲不應該超過90度。我要通過靜態下蹲找到腿酸的感覺，然後在雪道上轉彎時複製這個感覺，如果能感覺到相同的酸，說明蹲的量就夠了。

## 改善方法 (How)

### 步驟拆解

1. **理解正確的蹲量**：
   - 大腿小腿角度約六七十度
   - 不要超過90度
   - 超過90度是無效蹲（靠關節不靠肌肉）

2. **靜態練習找感覺**：
   - 腳放在固定器裡
   - 下蹲到六七十度
   - Hold住30秒到1分鐘
   - 感覺大腿酸，可能需要腳踝往上勾讓小腿幫助支撐
   - 記住這種酸的感覺

3. **雪道上應用**：
   - 記住靜態蹲的酸感後立即去滑
   - 在陡一點的道或雪比較爛的道轉彎
   - 如果能感覺到腿有相同的酸，說明蹲的量夠了
   - 如果完全感覺不到酸且失去速度控制，需要再多蹲一點

4. **調整蹲量**：
   - 腳踝勾起來一點
   - 膝蓋彎曲一點
   - 稍微找到一點酸的感覺
   - 逐漸接近目標

5. **理解包裡的特殊情況**：
   - 在包裡被雪推可能會超過90度
   - 這是雪推造成的，不是主動蹲造成的
   - 主動蹲不應該超過90度

### 教練提示

- 正確的蹲量是六七十度，不要超過90度
- 通過靜態蹲找到腿酸的感覺，然後在雪道上複製
- 超過90度是無效蹲，靠關節不靠肌肉
- 包裡被雪推超過90度是正常的，但主動蹲不應該
- 如果感覺不到酸且失控，說明蹲得不夠

### 常見錯誤

- 蹲得太少，無法控制速度
- 蹲超過90度變成無效蹲
- 不知道如何判斷蹲的量是否正確
- 沒有通過靜態練習找到參考感覺
- 混淆主動蹲和被雪推的區別

### 修正方法

- 先做靜態蹲30秒到1分鐘，記住腿酸的感覺
- 立即去雪道上滑，嘗試複製這個酸感
- 如果感覺不到酸，逐步增加蹲的量
- 用腳踝和膝蓋的彎曲度來調整
- 在陡坡或爛雪上練習，更容易感受到
