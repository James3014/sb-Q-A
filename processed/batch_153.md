# 153. 樹林裡比較密：上下半身反擰

## 問題描述 (What)

我在比較茂密的樹林或包圍較窄的雪道上滑行時，轉彎空間有限，難以快速完成轉彎。雖然雪道本身不難（可能是藍道），但因為落差大、樹林密集、每個包圍都很窄，我需要更有效的技巧來快速把板子拉回來完成轉彎。

## 改善目標 (Why)

我需要學會利用上下半身的反擰（像擰毛巾一樣）來幫助把板子拖拽回來。通過保持上半身一直朝下（朝向滾落線），用下半身的力量去轉動，上下半身之間會產生反擰的張力。利用這個反擰的力量可以更快速地把板子拉回來，讓我在狹窄空間中也能流暢轉彎。我會感覺到腰部位置一直在一松一緊的狀態。

## 改善方法 (How)

### 步驟拆解

1. **建立上半身朝向**：保持上半身一直朝向滾落線（朝下）
2. **下半身轉動**：用下半身的力量去轉動板子
3. **產生反擰**：因為上半身保持不動，下半身轉動會產生反擰
4. **利用反擰**：利用這個反擰的張力把板子拉回來
5. **感受節奏**：感覺腰部位置一松一緊、一松一緊的節奏
6. **應用於密林**：在茂密樹林或窄包圍中運用這個技巧快速轉彎

### 教練提示

- 上半身要一直保持朝向滾落線，不要跟著下半身轉
- 反擰的感覺像擰毛巾，會在腰部產生張力
- 利用反擰的彈性把板子拉回來，不是硬拉
- 這個技巧在落差大、空間窄的地形特別有用

### 常見錯誤

- 上半身跟著下半身一起轉，沒有產生反擰
- 只做外形動作，沒有真正利用反擰的力量
- 腰部僵硬，無法產生一松一緊的節奏
- 在不需要的地形也過度使用這個技巧

### 修正方法

- 刻意保持上半身朝下，用視線固定方向
- 先在寬闊雪道練習，感受腰部的反擰張力
- 放鬆腰部，讓反擰自然產生和釋放
- 根據地形選擇性使用，不要過度依賴
