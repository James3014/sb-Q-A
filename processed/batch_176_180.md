# 176. 看看你的後刃是否：轉完

## 問題描述 (What)

我的小彎滑得還可以，重心、施壓、彎形對稱、控速都做得不錯，但一旦到陡坡或雪況差一點時，可能會控制不了速度，有可能會直接剎停（brake），或者hold不住。我的彎比較開放，而且沒有辦法利用在彎裡面儲存的能量來幫助把彎轉完。

## 改善目標 (Why)

我需要學會把彎轉完，利用蓄力傳導（Loading & Deflection）。在施壓儲存壓力的位置，可以稍稍多蹲那麼一會兒，等板子進行形變之後再慢慢起來，彎就會轉完，蓄力傳導的力也就出來了。關鍵是要有patience（耐心），把動作從0.5秒延長到0.8秒，多一會兒，等腿感覺到酸的時候再釋放壓力，就能感覺到板子給我產生的回彈，彎就轉完了。

## 改善方法 (How)

### 步驟拆解

1. **識別問題**：
   - 彎比較開放
   - 無法利用彎裡儲存的能量
   - 陡坡或差雪況時控制不了速度

2. **理解蓄力傳導**：
   - Loading：板子形變（壓彎）
   - Deflection：板子回彈（傳導力量）
   - 這是把彎轉完的關鍵

3. **在施壓位置多停留**：
   - 在儲存壓力的位置稍稍多蹲一會兒
   - 不要很快就起來
   - 等板子進行形變

4. **感受板子形變**：
   - 腿會感覺到酸
   - 這是板子正在形變的信號
   - 等這個感覺出現

5. **釋放壓力**：
   - 感覺到酸後再慢慢起來
   - 板子會產生回彈
   - 彎就轉完了

6. **延長時間**：
   - 原本動作0.5秒
   - 延長到0.8秒
   - 多一點patience

7. **場地選擇**：
   - 平的地方也能感受到
   - 陡的地方更容易感受到

### 教練提示

- 關鍵是patience，多停留一會兒
- 等腿感覺到酸再釋放
- 板子形變需要時間
- 在平地也能練習這個
- 陡坡上效果更明顯

### 常見錯誤

- 施壓後很快就起來，沒有等板子形變
- 沒有patience，動作太快
- 不會感受腿酸的信號
- 彎比較開放，沒有轉完
- 無法利用蓄力傳導

### 修正方法

- 刻意在施壓位置多停留
- 等腿感覺到酸再起來
- 把動作時間延長（0.5秒→0.8秒）
- 先在平地練習感受
- 再到陡坡應用

# 177. 看看你的後刃是否：駝背

## 問題描述 (What)

我的後刃站姿自己覺得還不錯，抓地力也有，但習慣了這種姿勢後會非常駝背。當我換後刃時，會不自覺地彎膝蓋彎腳踝，換完後會出現屁股出去、頭往前探的駝背姿勢。雖然這種姿勢抓地力也有、膝蓋腳踝也緊，但彎得比較多導致屁股出去，為了平衡把頭往前探。

## 改善目標 (Why)

我需要改善後刃的站姿，讓身體更直。當我們滑的時候速度可以幫助我們，所以應該是：肩膀在髖稍微外面一點，髖在腳的稍微外面一點（相對後刃來講）。可以想像肩膀上有兩杯水，當換到後刃時保持這兩杯水在上面而不是水灑了。這樣可以讓我的人在板子上更加直，發力時人是朝著前刃後刃發力，而不是駝背發力。

## 改善方法 (How)

### 步驟拆解

1. **識別駝背問題**：
   - 換後刃時彎膝蓋彎腳踝
   - 換完後屁股出去
   - 為了平衡頭往前探
   - 形成駝背姿勢

2. **理解正確站姿**：
   - 肩膀在髖稍微外面一點
   - 髖在腳的稍微外面一點
   - 相對後刃來講
   - 速度可以幫助我們

3. **使用兩杯水練習**：
   - 想像肩膀上有兩杯水
   - 換到後刃時保持兩杯水在上面
   - 不要讓水灑了
   - 這個練習很多人都聽過

4. **保持身體更直**：
   - 不要過度彎曲
   - 讓人在板子上更加直
   - 發力時朝著前刃後刃發力
   - 而不是駝背發力

5. **對比練習**：
   - 先做錯誤示範（駝背）
   - 再做正確示範（身體直）
   - 感受差異

### 教練提示

- 肩膀上兩杯水是很好的意象練習
- 就這麼一點點的改變會讓人更直
- 發力方向很重要：朝前刃後刃，不是駝背
- 速度可以幫助我們保持正確姿勢
- 雖然駝背也有抓地力，但不是最佳姿勢

### 常見錯誤

- 換後刃時過度彎曲膝蓋腳踝
- 屁股出去、頭前探形成駝背
- 不知道正確的身體排列
- 沒有用兩杯水的意象
- 發力方向錯誤

### 修正方法

- 用兩杯水意象提醒自己
- 換後刃時注意肩髖腳的排列
- 讓身體更直，不要過度彎曲
- 對比練習感受差異
- 注意發力方向

# 178. 看看你的滑行裡是否有"蹲"無"起"

## 問題描述 (What)

我的滑行已經從初級到中級再往上走，膝蓋與腳踝彎的程度不錯，但有一個常見問題：我的flexion（下降折疊）很好，但沒有extension（站起來），一直保持在蹲的位置。當到一定陡度時，速度還是得不到控制，因為我沒有利用好上下的空間（range）來釋放壓力減速。雖然一直蹲著也在釋放壓力，但力量沒有那麼大。

## 改善目標 (Why)

我需要學會釋放壓力站起來（extension）。在比較平的道（綠道藍道）時沒有起來還好，但一旦到陡一點的地方，彎會變大、速度控制不了，需要通過彎形把彎結束來控速。我要學會在彎的不同位置站起來釋放壓力：最簡單的是下半段，最好的是越早越好，但不要追求一步到位。先在下半段嘗試，然後慢慢往上移，最後到上半段。通過站起來的速度和頻率來控制彎形大小。

## 改善方法 (How)

### 步驟拆解

1. **識別問題**：
   - Flexion很好但沒有extension
   - 一直保持蹲的位置
   - 沒有利用上下空間釋放壓力
   - 陡坡時速度控制不了

2. **理解上下空間的重要性**：
   - 上下的range可以幫助釋放壓力減速
   - 一直蹲著力量不夠大
   - 需要站起來釋放

3. **階段一：下半段站起來**：
   - 最簡單的位置
   - 在彎的下半段釋放壓力站起來
   - 先練習這個

4. **階段二：中段站起來**：
   - 熟練後慢慢往上移
   - 在彎的中段站起來
   - 彎會開始變小

5. **階段三：上半段站起來**：
   - 最好的位置是越早越好
   - 在彎的上半段站起來
   - 彎會變得更小

6. **控制彎形大小**：
   - 如果想彎大一點：在早一點的地方釋放，但慢慢站起來
   - 如果想彎小一點：在早一點的地方釋放，快速站起來
   - 通過站起來的速度和頻率控制

7. **觀察效果**：
   - 站起來越往上提，彎越小
   - 可以通過視頻觀察自己的進步

### 教練提示

- 不要追求一步到位，循序漸進
- 先在下半段練，再慢慢往上移
- 站起來的速度和頻率可以控制彎形
- 越早站起來，彎越小
- 慢慢站起來，彎可以大一點

### 常見錯誤

- 只有flexion沒有extension
- 一直蹲著不站起來
- 不會利用上下空間
- 想一步到位在上半段站起來
- 不會通過站起來的速度控制彎形

### 修正方法

- 先練習在下半段站起來
- 熟練後逐漸往上移
- 感受站起來對速度控制的幫助
- 實驗不同的站起來速度
- 通過視頻檢查自己的進步

# 179. 第一出道外：試試練習這三個階段

## 問題描述 (What)

我在道內滑了很久，想要從壓雪道滑到道外，但不知道如何循序漸進地過渡。我可能會直接嘗試滑很大的包，步子邁得太大，導致失敗或受挫。

## 改善目標 (Why)

我需要通過三個階段來練習出道外：

**階段一**：找沒有包、雪況比較白的道外（最簡單），在藍道或綠道上，身體不太需要使勁推雪，先能轉過來，靠蹲去減速。

**階段二**：讓蹲發力越來越早，彎開始變形變小。雪會給很大反饋把人往上彈，如果承受不了彈力，用腿去吸收，或者推的時候不要靠爆發力。

**階段三**：用開放式彎形往兩邊推（例如往樹上噴濺雪的練習），當能控制好往兩邊推、駕馭一定速度後，再去嘗試真正有包的地形，需要精準控制彎形、往兩側推，滑得更流暢（flow）。

## 改善方法 (How)

### 步驟拆解

1. **階段一：基礎道外（無包、白雪）**：
   - 選擇沒有包的道外
   - 雪況比較白
   - 藍道或綠道
   - 身體不太需要使勁推雪
   - 先能轉過來
   - 靠蹲去減速
   - 這是最基本的

2. **階段二：早發力、控制彈力**：
   - 讓蹲發力越來越早
   - 彎開始變形變小
   - 雪會給很大反饋把人往上彈
   - 如果承受不了彈力：
     * 用腿去吸收一點
     * 或者推的時候不要靠爆發力
   - 讓推雪變得更加精準

3. **階段三：開放式彎形、往兩邊推**：
   - 用開放式彎形
   - 往兩邊推（例如往樹上噴濺雪）
   - 當能控制好往兩邊推時
   - 能駕馭一定速度時
   - 再去嘗試真正有包的地形

4. **有包地形的要求**：
   - 精準控制彎形
   - 往兩側推
   - 滑得更流暢（flow）

5. **示範對比**：
   - 一開始不壓，只是簡單轉
   - 然後開始慢慢給壓力，身體往下，主動蹬雪，彎變小
   - 最後有控制地蹬，彎控制在很小範圍內

### 教練提示

- 不要一開始就嘗試大包，步子太大
- 循序漸進通過三個階段
- 階段一最重要：先能轉過來
- 階段二學會控制雪的反饋
- 階段三才去有包的地形
- 往樹上噴濺雪是很好的練習

### 常見錯誤

- 直接嘗試滑大包，步子太大
- 跳過基礎階段
- 不會控制雪的彈力反饋
- 沒有練習往兩邊推
- 太早去有包地形

### 修正方法

- 嚴格按照三個階段練習
- 在每個階段都練熟再進入下一階段
- 選擇合適的地形（無包白雪→有包）
- 練習往樹上噴濺雪
- 感受雪的反饋並學會控制

# 180. 第一次滑深粉：站姿細節

## 問題描述 (What)

我第一次滑粉雪，知道重心需要往板尾移動，但不清楚應該移動多少。如果移多了很容易衝著衝著直接撅下來，沒有辦法拐彎，因為重心太在板尾了。我不知道正確的重心範圍和腳部感受的細節。

## 改善目標 (Why)

我需要學會正確的粉雪站姿細節。重心往板尾移動時，要感覺到後腳的外側（後面的外側），先感覺到外側，內側沒有感覺。然後稍微彎膝蓋，把內側也感覺到，也就是通過下降擠壓壓到的感覺。前腳要感覺腳的內側壓住。這個範圍就是很好的範圍，不會太往後或太往板頭。前腳的外側會感覺到嗎？會，但是當轉彎時由於腳會往裡一點，轉彎時會被拉扯的感覺到外側，而不是滑的時候壓住的外側。

## 改善方法 (How)

### 步驟拆解

1. **理解重心移動的問題**：
   - 重心需要往板尾移動（這個大家都知道）
   - 但移多了會撅下來無法拐彎
   - 移動多少很關鍵

2. **後腳感受（重點）**：
   - 先感覺到後腳的外側（後面的外側）
   - 內側現在沒有感覺
   - 稍微彎膝蓋
   - 把內側也感覺到
   - 通過下降擠壓壓到的感覺
   - 後腳兩邊都有感覺

3. **前腳感受**：
   - 感覺腳的內側壓住
   - 這個範圍就是很好的範圍

4. **前腳外側的特殊情況**：
   - 前腳外側會感覺到嗎？會
   - 但不是滑的時候壓住的
   - 是轉彎時由於腳往裡一點
   - 被拉扯的感覺到外側

5. **確認正確範圍**：
   - 後腳：外側+內側（通過彎膝蓋擠壓）
   - 前腳：內側壓住
   - 這樣不會太往後或太往板頭

6. **練習場地**：
   - 可以在斯拉什的雪（白夕陽雪）上面先嘗試
   - 感覺跟粉雪裡面差不多
   - 然後直接去粉雪裡面嘗試
   - 粉雪裡摔了需要爬出來，斯拉什不需要

### 教練提示

- 重心移動多少很關鍵
- 後腳先感覺外側，再通過彎膝蓋感覺內側
- 前腳感覺內側壓住
- 前腳外側是轉彎時被拉扯的感覺，不是壓住的
- 先在斯拉什雪上練習再去粉雪

### 常見錯誤

- 重心移太多導致撅下來
- 不知道應該感受腳的哪個部位
- 後腳只感覺外側，沒有通過彎膝蓋感覺內側
- 前腳感覺錯誤
- 直接去粉雪沒有先在斯拉什雪練習

### 修正方法

- 先在斯拉什雪上練習找感覺
- 刻意感受後腳外側，再彎膝蓋感受內側
- 確認前腳內側壓住
- 理解前腳外側是轉彎時被拉扯的
- 確認感受正確再去粉雪
