# 157. 滑蘑菇的一個底層邏輯

## 問題描述 (What)

我在滑蘑菇時不理解底層邏輯，可能會站得比較直，導致板子和包的側壁（側臂）形成很大的角度。這個角度就像在平道上刻滑一樣，刃會入得很深，板子會卡在雪裡轉不動，無法完成轉彎和減速。

## 改善目標 (Why)

我需要理解滑蘑菇的底層邏輯：當上包的側壁時，要通過壓縮身體來降低板刃角度，而不是把刃立起來。板刃角度小了，板子就可以轉動來減速。如果板刃角度太大（身體站太直），板子會卡在雪裡轉不了。這個原理適用於所有大包小包的側壁。

## 改善方法 (How)

### 步驟拆解

1. **理解側壁結構**：
   - 每個包都有側面（側壁/側臂）
   - 側壁是斜著的角度
   - 不同包的側壁斜度可能不同

2. **理解角度問題**：
   - 如果身體站得比較直，板子和側壁會形成很大角度
   - 這個角度相當於在平道上刻滑，刃會入很深
   - 板子會卡在雪裡轉不動

3. **壓縮身體降低角度**：
   - 上側壁時壓縮身體（下蹲）
   - 把板刃角度變小
   - 板子就可以轉動了

4. **轉彎減速**：
   - 板刃角度小的狀態下轉動板子
   - 實現減速和控制
   - 避免板子卡在雪裡

5. **在綠道練習**：先在綠道練習壓縮身體降低板刃角度的感覺

### 教練提示

- 滑蘑菇的底層邏輯是控制板刃角度
- 側壁是斜的，如果站太直板子會卡住
- 壓縮身體是為了降低板刃角度，讓板子能轉
- 這個原理適用於所有大小包

### 常見錯誤

- 身體站太直，板刃角度太大
- 把刃立起來，導致板子卡在雪裡
- 不理解側壁的斜度特性
- 沒有通過壓縮身體來調整板刃角度

### 修正方法

- 上側壁時刻意壓縮身體
- 感受板刃角度的變化
- 先在綠道練習，再到蘑菇道應用
- 如果板子轉不動，檢查是否身體太直
