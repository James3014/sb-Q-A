# 52 平地小練習：感受重心居中

## 📝 教學內容

### 問題描述 (What)
在推推滑的時候，大部分人可能是在後刃這個地方去推，很少有人會在前刃。如果推後面的時候這樣，推前面的時候那樣的話，肯定摔跤或者不穩。

### 改善目標 (Why)
- 保證前腳是居中的
- 幫助自然而然的降一些重心
- 當做的時候就是比較穩的
- 確保壓力點處於前腳的正中心足弓的位置

### 改善方法 (How)

**平地推推滑練習**：

1. **動作要領**：
   - 後面推一下，前面推一下，後面推一下
   - 需要把前腳重心保持在居中一點

2. **練習方法**：
   - 在平地走的時候，可以去嘗試一下
   - 左邊一下，右邊一下，左邊一下，右邊一下
   - 確保壓力點處於前腳的正中心足弓的位置

3. **效果**：
   - 可以通過這個練習去推來保證前腳是居中的
   - 靠這個幫助自然而然的降一些重心
   - 這樣的話當做的時候就是比較穩的

# 53 企鵝步

## 📝 教學內容

### 問題描述 (What)
在雪的雪道上平地的時候，不願意拖板，比較簡短一點的路程想要走兩步，但不知道如何正確走企鵝步。

### 改善目標 (Why)
- 能夠穩定地走企鵝步
- 往前走或倒著走都能保持穩定
- 不會趴地上或往後躺

### 改善方法 (How)

**企鵝步技巧**：

1. **基本動作**：
   - 把這個腳往前放，這個腳跟著
   - 需要讓重心有點往行進的方向先扔過去
   - 還有上半身往前扔著，這樣就走回來了

2. **往前走**：
   - 當走企鵝步的時候，要往前走
   - 盡量把重心放在後腳跟
   - 如果要是往前走的話，想象重心在後腳跟
   - 這樣的話往前走就是比較穩的
   - 如果要是放在腳尖了，再往前走，可能走著走著就趴地上了

3. **倒著走（倒企鵝）**：
   - 要是想倒著走，倒企鵝
   - 會往後躺一點
   - 但是呢，會讓重心盡量放在腳尖這個位置
   - 然後往前走

# 54 企鵝步進階：盪板（TicTac）

## 📝 教學內容

### 問題描述 (What)
想要在平地行走比較進階的方式，不只是簡單的企鵝步。

### 改善目標 (Why)
- 在原地下啟動往前走
- 更有效率地在平地移動
- 甚至可以上坡

### 改善方法 (How)

**盪板（Tick Tack）技巧**：

1. **基本原理**：
   - 跟企鵝步很像
   - 當要往這邊走，身體朝這邊打開一點
   - 想往前走，會靠往兩邊的這個力量去幫助往前走

2. **往後蹬**：
   - 要是往這邊蹬的時候，往這邊
   - 往後蹬的時候，後面蹬的時候
   - 會把壓力放在後腳跟多一點
   - 為什麼呀？一旦放在前腳掌，人當一蹬就會往這邊
   - 它的力量阻力，它會讓失去抓地力，反而不會讓蹬上力量

3. **往這邊推**：
   - 如果要是往這邊登呢，往這邊推呢
   - 會讓前腳掌頭去推
   - 這樣的話往這邊推的力量也就更大了
   - 這樣的話就能簡單的走起來

**進階練習**：
- 如果覺得在平地簡單，可以嘗試上坡，這個就比較有難度了

# 55 先學會在蘑菇裡減速

## 📝 教學內容

### 問題描述 (What)
室內的蘑菇是比較大的，而且它是拉鍊型的。不知道怎麼去安全的去滑這個蘑菇。

### 改善目標 (Why)
- 安全的去滑這個蘑菇
- 能夠控制速度
- 找到節奏
- 不管這個包多深或者多陡或者多冰，都能應對

### 改善方法 (How)

**蘑菇減速邏輯**：

1. **核心邏輯**：
   - 每轉一個包，需要去停下
   - 就是真正的刹住了，或者說把速度減到絕對能控制的地步
   - 然後再進入下一個彎，或者說換刃進入下一個彎

2. **停下的位置**：
   - 只有在那個位置停，才能安全的進入下一個彎
   - 從前刃到這兒的時候，進入下一個彎
   - 需要在手指這個方向這個位置刹停
   - 會盡量刹停，帶著速度刹停

3. **效果**：
   - 當進入下一個包的時候，會看著包穩定
   - 因為這個包比較深，所以說開始刹停的話，進入包下一刻就穩定

4. **練習進階**：
   - 當盡量可以控制住刹停的時候，找到節奏的時候
   - 就可以每一個包稍微把速度降下來，然後再去轉下一個彎
   - 這樣的話會愈加熟練
