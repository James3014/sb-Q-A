# 🏂 單板教學 App - 手機 UI 設計

## 使用情境
- 雪場纜車上 / 休息區快速查找
- 戴手套操作（大按鈕、少打字）
- 強光下閱讀（高對比）
- 找到問題 → 看解法 → 馬上練

### 設計重點
- **問題優先**：卡片最上方顯示「問題描述」，讓用戶快速對號入座
- **大觸控區**：整張卡片可點擊，戴手套也好按
- **標籤藥丸**：程度/雪道/技能 一目了然
- **高對比**：深色背景 + 亮色文字

## 🎨 配色方案（雪場高對比）

| 元素 | 顏色 | 用途 |
|------|------|------|
| 背景 | `#0f172a` (深藍黑) | 減少反光 |
| 卡片 | `#1e293b` (深灰藍) | 區隔層次 |
| 主文字 | `#f1f5f9` (亮白) | 高可讀性 |
| 問題標題 | `#fbbf24` (琥珀黃) | 醒目 |
| 標籤-程度 | `#22c55e` (綠) | |
| 標籤-雪道 | `#3b82f6` (藍) | |
| 標籤-技能 | `#a855f7` (紫) | |
| 做對 | `#22c55e` (綠) | 正向回饋 |
| 做錯 | `#ef4444` (紅) | 警示 |
