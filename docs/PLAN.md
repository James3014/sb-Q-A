## UX 優化計畫（2025-11-28）✅ 已完成

> 基於 UX 審查報告，按 Linus 原則：低成本高回報優先

### 已完成項目（~13hr → 實際 ~4hr）

```
Phase 1 ✅
├── 74. 字級 18px + 行高 1.8
├── 75. 錯誤頁加返回按鈕 + spinner
├── 76. 震動回饋（Android 支援）
└── 77. Skeleton 載入狀態

Phase 2 ✅
├── 78. 底部固定操作欄（60px）
└── 79. 觸控目標 ≥48px
```

### 暫緩項目（觀察需求後再做）

| 項目 | 暫緩原因 |
|------|---------|
| Snow Mode | 剛完成紅黑主題，避免連續大改 |
| 語音搜尋 | 瀏覽器支援度差（iOS 不支援）|
| Service Worker | PWA 配置複雜 |
| 麵包屑導航 | 有幫助但非必要 |

### 驗收結果 ✅
- [x] 字級達 WCAG AA（正文 18px）
- [x] 錯誤頁有明確返回路徑
- [x] 主按鈕有觸覺回饋（Android）
- [x] 載入時顯示骨架屏
- [x] 底部操作欄固定可見
- [x] build 通過
