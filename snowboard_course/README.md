# 單板滑雪教學資產包

歡迎使用本教學資產包。這是一套為單板滑雪教練與學員設計的系統化教學資源，旨在將碎片化的知識重構為一套層級分明、可評量、可追溯的完整體系。

所有內容均基於提供的逐字稿資料生成，任何資訊缺口皆以 `【資訊缺口：...】` 標示。

---

## 2. 如何使用本資產包

### A. 教練情境 (Instructor Use Case)

1.  **課前準備**：
    *   參考 `01_curriculum_map.md` 了解學員當前水平與下階段目標。
    *   使用 `02_module_blueprints.md` 準備特定模組的教學重點。
    *   從 `06_sessions_playbook.md` 挑選合適的課程腳本，並準備所需教具。

2.  **課堂執行**：
    *   從 `03_drill_cards.md` 中選取合適的練習（Drill）來教授微技能。
    *   當學員遇到困難時，利用 `04_error_diagnosis_matrix.md` 快速診斷問題並指派修正練習。

3.  **課後評量**：
    *   根據 `05_assessment_and_rubrics.md` 中的規準評估學員表現，並提供具體回饋。
    *   利用 `08_glossary_and_traceability.md` 統一溝通語言，確保術語清晰。

### B. 學員情境 (Learner Use Case)

1.  **課前預習**：
    *   閱讀 `07_learner_handbook.md` 了解即將學習的動作要點與視覺化步驟。
    *   查閱 `01_curriculum_map.md` 查看自己的學習進度與未來路徑。

2.  **自主練習**：
    *   使用 `07_learner_handbook.md` 中的「要點核對」與「自我修正」技巧進行練習。
    *   填寫「練習日記模板」，記錄進度、困難與心得。

3.  **自我評量**：
    *   參考 `05_assessment_and_rubrics.md` 的自評表，檢視自己是否達到通關標準。
    *   若對術語不解，可查詢 `08_glossary_and_traceability.md`。

## 3. 資料來源與追溯方法

本資產包的所有教學內容皆提煉自 `sources/` 目錄下的逐字稿。`08_glossary_and_traceability.md` 文件中包含了詳細的追溯索引，將每個知識點（如一個練習、一個概念）連結回其原始出處（檔案、講者、時間碼），確保了教學內容的準確性與可驗證性。

`sources/transcripts_index.csv` 則提供了所有已解析逐字稿的元數據索引。
