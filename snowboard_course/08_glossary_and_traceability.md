# 詞彙庫與追溯索引 (Glossary & Traceability)

本文件統一定義了關鍵術語，並將所有教學內容追溯回原始資料，確保一致性與準確性。

## 1. 詞彙庫 (Glossary)

| 術語 (繁體中文) | 英文對照 (English) | 同義詞/相關詞 | 定義 |
|---|---|---|---|
| 站姿 | Stance | 身體姿勢 | 滑雪時身體的基本準備姿勢，通常指髖、膝、踝關節彎曲，重心放低且穩定的狀態。 |
| 鋼邊 / 邊刃 | Edge | 刃 | 雪板底部的金屬邊條，用於切入雪中以提供抓地力和控制。分為腳跟側與腳尖側。 |
| 腳跟側 | Heelside | 後刃 (Backside) | 靠近腳跟一側的鋼邊。 |
| 腳尖側 | Toeside | 前刃 (Frontside) | 靠近腳尖一側的鋼邊。 |
| 橫移 | Traverse | 斜切坡 | 雪板橫過山坡的滑行路徑，不直接指向山下。 |
| 落葉飄 | Falling Leaf | 推坡 | 在單一鋼邊上，透過前後腳的壓力變化，左右來回移動的控制練習，軌跡如飄落的葉子。 |
| 搓雪 | Skidding | | 雪板在滑行時，板底與行進方向存在一定夾角，導致雪板在雪上產生橫向滑動摩擦，用以降速或控制。 |
| 刻滑 / 走刃 | Carving | | 雪板完全用鋼邊切開雪面滑行，板底不發生橫向滑動，在雪上留下一道乾淨利落的細線。 |
| 軸轉 | Axial Rotation | 全身一起轉 | 以人體從頭到腳的中軸線為中心，頭、肩、髖、膝、腳等所有關節同步旋轉，以引導雪板。 |
| 反擰 | Counter-rotation | | 上半身與下半身朝相反方向旋轉，產生一個扭力來快速改變雪板方向或減速，是應對複雜地形的關鍵技巧。 |
| 傾倒 | Inclination / Leaning | | 身體整體向彎心傾斜，以對抗離心力並增加立刃角度，是刻滑的關鍵動作之一。 |
| 關節折疊 | Angulation / Flexion | | 彎曲腳踝、膝蓋、髖關節等，以在身體傾倒之外獨立調整立刃角度和吸收地形。 |
| 開放式站姿 | Open Stance | | 一種站姿，指髖關節和上半身朝滑行方向適度打開，以獲得更大的運動空間和更快的換刃效率。 |
| 閉合式站姿 | Closed Stance | | 一種站姿，指髖關節和肩膀基本與雪板保持平行，有利於穩定控速。 |
| 提早施壓 | Early Pressure | | 在換刃後、到達滾落線之前就開始下降身體並對刃施加壓力，以實現勻速過彎的進階控速技巧。 |
| 被動傾倒 | Passive Inclination | | 指身體的傾倒應是關節折疊和壓力施加後的自然結果，而非主動向彎內“躺”或“趴”。 |
| 追身 | Chasing the Body | | 饅頭滑行技巧，指在過包頂後，身體主動向前移動，去“追趕”已經開始下落的雪板，以保持平衡。 |

## 3. 附錄：驗證與反思

本節旨在紀錄開發此教學資產包時的自我審查過程。

### 三條關鍵自我查核問題

1.  **對齊 (Alignment)**: 所有評量標準是否都能在課程地圖中找到對應的學習成果？
    *   **回答**: 是。`05`中的所有評量維度均與`01`中的「可觀察學習成果」進行了匹配。

2.  **無循環 (Acyclicity)**: 課程地圖中的技能依賴關係是否清晰且無死循環？
    *   **回答**: 是。技能樹嚴格遵循從 Level 0 到 Level 4 的遞進結構，確保了學習路徑的合理性。

3.  **安全覆蓋 (Safety Coverage)**: `02`、`03`、`06` 文件是否都包含了必要的安全提示與風險控制措施？
    *   **回答**: 是。`02` 的模組大綱有「風險點與安全控制」章節。`03` 的技能卡有「安全注意」欄位。`06` 的課程腳本也包含風險控制清單。
