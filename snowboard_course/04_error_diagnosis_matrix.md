# 錯誤診斷矩陣 (Error Diagnosis Matrix)

此矩陣用於快速診斷學員在滑行中表現出的問題，並找到可能的原因與對應的修正練習。

| 問題表現 (Problem Observation) | 可能成因 (Potential Cause) | 觀察線索 (Observable Clues) | 立即介入練習 (Intervention Drill) |
|---|---|---|---|
| **前刃彎無法結束，板子一直往山下衝** | **旋轉 (Rotation):** 前刃開肩，上半身沒有隨雪板轉動，前肩一直指向山下。 | 前刃轉彎的後半段，學員上半身與雪板朝向不一致，看起來像在與雪板對抗。 | **DR-L2-004** 紳士滑行<br>或 **抓腋下練習**：前手抓住後手腋下，物理上阻止開肩。 |
| **後刃滑行時，板尾左右亂甩，俗稱「掃雪」** | **姿勢 (Stance):** 頂胯，重心錯誤地壓在後腳。 | 身體呈反弓形，屁股刻意頂向板頭，導致板尾失壓。 | **DR-L1-002** 重心跳躍檢查，在橫切坡時跳躍，檢查落地是否平穩。 |
| **前刃站不穩，小腿巨酸，或向前撲倒** | **姿勢 (Stance):** 前刃哈腰，用彎腰代替屈膝和屈髖，重心過高且不穩。 | 身體像鞠躬，而非坐下。頭部過低，屁股翹起。腳踝緊張，用腳尖踮著。 | **前刃站姿練習**：靠牆練習屈膝，或在平地練習「偷了東西往屁股後藏」的動作，找到髖關節前移、膝蓋下跪的感覺。 |
| **換刃時，刃在雪中「卡」住，突然摔倒** | **路線/邊刃 (Line/Edging):** 換刃時機錯誤，在板子還未開始轉向時就過早、過快地放平雪板。 | 轉彎軌跡呈Z字型，有明顯的卡頓點，而非平滑的S型。 | **DR-L2-001** 斜行啟動，強調換刃前先讓板子斜向滑行，順勢「滾」入下一個彎。 |
| **速度稍快或坡度稍陡，後刃就抓不住雪，產生劇烈抖動（突突突）** | **壓力/姿勢 (Pressure/Stance):** 彎結束時後腿完全蹬直，失去了緩衝和持續施壓的能力。 | 換刃後，板頭直接指向滾落線，滑行者表情驚恐，身體僵硬。後腿在彎中會突然伸直。 | **DR-L3-006** 摸鞋下蹲，找到正確的下蹲深度。<br>或練習在一次轉彎中，連續蹬腿推雪2-3次，確保腿部總有餘力。 |
| **轉彎時感覺笨重，轉不過去** | **旋轉 (Rotation):** 未能以髖關節為核心帶動全身轉動，只動了頭或肩膀。 | 動作看起來不協調，上半身和下半身像在打架。 | **DR-L2-002** 水壺人轉體，用倒茶的意象練習，體會以身體中心發起旋轉的感覺。 |
| **在不平整雪或饅頭中，身體僵硬，被頂飛** | **姿勢 (Stance):** 關節鎖死，沒有主動彎曲膝蓋和腳踝來吸收地形。 | 身體直上直下，遇到雪包時整個人被向上彈起。 | **摸固定器練習**：在橫切坡時，練習彎腰用雙手觸摸固定器，強迫身體降低重心並彎曲關節。 |
| **後刃刻滑時腳跟卡雪** | **壓力/邊刃 (Pressure/Edging):** 過度追求立刃角度（過度傾倒），而沒有施加足夠的向下的壓力來讓板子形變。 | 為了立刃而身體過度傾倒，但板子並未有效吃進雪裡，導致腳跟或固定器先碰到雪。 | **壓力優先練習**：降低立刃角度，專注於感受用體重將後刃“踩”進雪裡的感覺，而不是單純“躺”下去。 |
| **Ollie跳不起來，或只能雙腳跳(Pop)** | **壓力 (Pressure):** 沒有做出從板頭到板尾的壓力轉移來蓄力，未能利用板子彈性。 | 起跳時沒有明顯的板尾下壓和回彈動作，跳躍高度極低。 | **Ollie三步分解**：在平地練習“1.點板頭 -> 2.坐板尾 -> 3.蹬起”的連續動作，體會板子彈性。 |
| **陡坡後刃彎腰撅屁股** | **姿勢 (Stance):** 髖關節閉合，導致下蹲空間不足，只能靠彎腰來降重心。 | 在後刃下降時，上半身前傾，屁股明顯向後突出，而非垂直下降。 | **打開髖關節下蹲**：讓髖關節稍微朝向板頭方向打開，為身體垂直下降創造空間。`Sam-71` |
| **小迴轉時刃立太高，被板子帶著走** | **邊刃 (Edging):** 過度依賴立刃，而沒有結合搓雪來主動控制彎形大小。 | 彎的上半段板子軌跡很直，抓地過強，到下半段才突然轉向，無法縮小彎的半徑。 | **增加搓雪練習**：在彎的上半段就有意識地推板，用板底去“搓”雪，而不是只用刃“切”。`Sam-23` |
| **後刃換前刃時，身體往前趴** | **重心轉移 (Weight Transfer):** 換刃過急，重心從腳跟暴力轉移到腳尖，缺乏平順過渡。 | 換刃瞬間，上半身猛然前傾，甚至需要用手去撐地。 | **平板過渡練習**：在換刃時，有意識地加入一個短暫的“平板”狀態，感受重心從腳跟滾動到腳掌，再到腳尖的過程。`Sam-11` |
| **滑行節奏死板，像機器人 (Park and Ride)** | **壓力 (Pressure):** 壓力施放的時機太晚，只在滾落線之後才開始減速。 | 彎的上半段身體姿態不變，直直滑下，過了滾落線才突然下蹲剎車，一卡一頓。 | **提早下降練習**：強制在換刃後、進入滾落線之前就開始下降身體，將減速區提前。`Sam-60` |
| **換刃後，板頭立刻往下掉，上半弧消失** | **邊刃/壓力 (Edging/Pressure):** 換刃後前腳沒有立即建立有效的抓地力。 | 換刃後，板子不聽使喚，直接指向山下，無法做出圓滑的弧線上半段。 | **DR-L3-004** 只壓前腳推坡，建立前腳在換刃瞬間的肌肉記憶和信心。 |
| **後刃彎不完整，無法順利收彎** | **旋轉 (Rotation):** 上半身（尤其是前肩）過度閉合，阻礙了髖關節的完整轉動。 | 後刃彎的後半段，板子無法繼續轉向橫切坡，導致下一個彎的起始位置不佳。 | **前手引導練習**：在整個後刃彎中，有意識地將前手保持在後刃鋼邊的上方，為髖關節創造轉動空間。`Sam-63` |
| **重心過度壓前腳，導致前腿膝蓋酸痛** | **姿勢 (Stance):** 習慣性地將身體重量前傾，尤其是在彎的末尾。 | 滑行後前腿大腿或膝蓋有明顯的酸痛感，而後腿感覺不累。 | **後腳感知練習**：練習在滑行中有意識地去感受「後腳外側」的壓力，以此來校準重心，將其向中間或後方轉移。`Sam-前腿膝盖疼...` |
