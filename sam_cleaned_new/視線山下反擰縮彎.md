# 視線山下反擰縮彎

## 📝 教學內容

### 問題描述 (What)
學生視線一直盯著兩側，導致無法從中回轉變成小回轉，持續橫貫雪道。

### 改善目標 (Why)
- 減少橫貫雪道
- 將彎形從中回轉縮小成小回轉
- 加快前刃換後刃的速度

### 改善方法 (How)

**新手 vs 進階視線管理**:
- **新手**: 視線看向要去的方向，起引導作用，幫助橫貫雪道
- **進階**: 視線保持看向山下，減少橫貫

**進階技巧 - 視線看山下**:
- **前刃動作**: 保持視線看向山下（不看兩側）
- **結果**: 在彎結束時上下半身出現反擰
- **反擰好處1**: 幫助從前刃換後刃，加速變快
- **反擰好處2**: 逼著自己轉小彎而不是大彎

**錯誤模式**:
- 視線看向兩側 → 左右帶動 → 慢慢橫貫過去 → 彎形變大

- **做對訊號**: 視線穩定看山下、彎結束時上下身反擰、彎形縮小、減少橫貫
- **做錯訊號**: 視線跟著看兩側、無法縮小彎形、持續橫貫雪道
