# 轉彎先降後轉建立抓地

## 📝 教學內容

### 問題描述 (What)
學生轉彎時順序錯誤，先轉胯/身體後才下蹲，導致抓地力不足，速度控制不穩定（坡度越陡速度越快，無法主動控制）。

### 改善目標 (Why)
- 建立轉彎時的抓地力，讓板子能抓住雪面
- 不管坡度陡緩都能主動控制速度
- 讓速度全程穩定，不會忽快忽慢

### 改善方法 (How)
- **正確順序**: 換重心 → 先下降 → 再轉胯（下半身關節）
- **換重心後**: 在弧線上半段不需要倫倒，直接下降（此時坡度已經幫你立刃）
- **下降後**: 當速度控制好後，再轉動下半身關節帶著板子下來
- **練習進程**: 先在綠道練習 → 藍道熟練 → 再到黑道
- **做對訊號**: 轉彎過程中有抓地力，速度可控，全程穩定
- **做錯訊號**: 上半弧失速，下半弧速度過快，整體速度忽快忽慢
